## Game Modes 🏅

### Daily Quest
- **Description**: One new math problem each day
- **Rewards**: Currency + Experience Points
- **Objective**: Maintain your daily streak

### Level Mode
- **Description**: From Level 1 (super easy) to Level 100 (impossible)
- **Rewards**: Badges + Currency + Collectibles
- **Objective**: Complete all levels

### Mini Games
- **Speed Math**: Answer as fast as possible
- **Match Operations**: Find calculations with the same result
- **Equation Builder**: Arrange symbols to create correct equations
- **Rewards**: Currency + Hints

### Battle Mode
- **1v1 Live Duel**: Battle another player
- **Round Format**: 3 problems in 3 minutes
- **Winner**: Whoever answers correctly and fastest
- **Rewards**: Currency, Ranking Points, Badges
