# G01_F01_SF05: Create Initial Profile

## 📋 Description

After email verification, create user profile with username, avatar, country, and Free subscription role. This completes the onboarding process.

---

## 🎯 Detailed Requirements

### Input Parameters

**Request Body (JSON)**
```json
{
  "username": "player_123",
  "full_name": "John Doe",
  "country": "US",
  "avatar_preference": "default"
}
```

**Validation Rules**
```python
- username: 3-30 chars, alphanumeric + underscore only, unique
- full_name: Optional, max 100 chars
- country: Optional, ISO 3166-1 alpha-2 code (2 chars)
- avatar_preference: Optional, enum: [default, generated, custom]
```

## 🗏️ Business Logic (10 Steps)

1. **Authenticate User**
   - Verify JWT token is valid
   - Extract user_id from token claims
   - Return 401 if token invalid

2. **Fetch User Record**
   - Query: `SELECT * FROM users WHERE id = :user_id`
   - Return 404 if user not found

3. **Check Profile Not Already Created**
   - Query: `SELECT profile_completed FROM users WHERE id = :user_id`
   - If profile_completed = true: return 400 (cannot re-create profile)
   - Message: "Profile already created for this account"

4. **Validate Username Format**
   - Check length: 3 â‰¤ len(username) â‰¤ 30
   - Check characters: ^[a-zA-Z0-9_]+$
   - Reject special chars, spaces, hyphens
   - Return 422 if validation fails

5. **Check Username Uniqueness**
   - Query: `SELECT COUNT(*) FROM user_profiles WHERE LOWER(username) = LOWER(:username)`
   - If count > 0: return 409 Conflict
   - Message: "Username already taken"

6. **Validate Country Code (if provided)**
   - Check against ISO 3166-1 alpha-2 list
   - If invalid: return 422
   - Allow NULL for optional field

7. **Generate Avatar URL**
   - If avatar_preference = "default": use `avatars/default-{random(1-10)}.png`
   - If avatar_preference = "generated": call avatar generator API
   - If avatar_preference = "custom": use placeholder
   - Store full URL: `https://avatars.example.com/{path}`

8. **Create User Profile**
   - Insert into user_profiles table:
     - user_id
     - username
     - country (if provided)
     - avatar_url
     - profile_completed = true
     - created_at = NOW()

9. **Update User Table**
   - Update users.profile_completed = true
   - Update users.avatar_url = avatar_url
   - Add default role = "Free" (subscription)
   - Set updated_at = NOW()

10. **Return Success Response**
    - HTTP 200 OK
    - Include user_id, username, avatar_url, role
    - Include profile_completed = true flag

# app/schemas/profile.py
from pydantic import BaseModel, Field, validator
from typing import Optional
from datetime import datetime
import re

class CreateProfileSchema(BaseModel):
    """Create user profile request schema"""
    username: str = Field(..., min_length=3, max_length=30)
    full_name: Optional[str] = Field(None, max_length=100)
    country: Optional[str] = Field(None, regex="^[A-Z]{2}$", description="ISO 3166-1 alpha-2")
    avatar_preference: Optional[str] = Field("default", regex="^(default|generated|custom)$")

    @validator('username')
    def validate_username(cls, v: str) -> str:
        """Validate username format"""
        if not re.match(r"^[a-zA-Z0-9_]+$", v):
            raise ValueError("Username can only contain letters, numbers, and underscore")
        return v

    class Config:
        json_schema_extra = {
            "example": {
                "username": "player_123",
                "full_name": "John Doe",
                "country": "US",
                "avatar_preference": "default"
            }
        }

class ProfileResponseSchema(BaseModel):
    """User profile response schema"""
    success: bool
    data: dict

    class Config:
        json_schema_extra = {
            "example": {
                "success": True,
                "data": {
                    "user_id": 1,
                    "username": "player_123",
                    "avatar_url": "https://avatars.example.com/default-1.png",
                    "role": "Free",
                    "level": 1,

                }
            }
        }

### Service Layer

```python
# app/services/profile_service.py
from sqlalchemy import func
from sqlalchemy.orm import Session
from app.models import User, UserProfile
from app.schemas.profile import CreateProfileSchema
from fastapi import HTTPException, status
import logging
import random

logger = logging.getLogger(__name__)
ISO_COUNTRIES = {"US", "GB", "CA", "AU", "VN", "JP", "CN", ...}  # Full list

class ProfileService:
    
    @staticmethod
    async def create_profile(
        user_id: int,
        profile_data: CreateProfileSchema,
        db: Session
    ) -> dict:
        """
        Create user profile after registration.
        
        Args:
            user_id: Authenticated user ID
            profile_data: Profile creation payload
            db: Database session
            
        Returns:
            Dictionary with profile data
            
        Raises:
            HTTPException: For validation/business logic errors
        """
        try:
            # Fetch user
            user = db.query(User).filter(User.id == user_id).first()
            if not user:
                raise HTTPException(
                    status_code=status.HTTP_404_NOT_FOUND,
                    detail="User not found"
                )
            
            # Check profile not already created
            if user.profile_completed:
                raise HTTPException(
                    status_code=status.HTTP_400_BAD_REQUEST,
                    detail="Profile already created for this account"
                )
            
            # Check username uniqueness
            existing = db.query(UserProfile).filter(
                func.lower(UserProfile.username) == func.lower(profile_data.username)
            ).first()
            if existing:
                logger.warning(f"Username taken: {profile_data.username}")
                raise HTTPException(
                    status_code=status.HTTP_409_CONFLICT,
                    detail="Username already taken"
                )
            
            # Validate country code if provided
            if profile_data.country and profile_data.country not in ISO_COUNTRIES:
                raise HTTPException(
                    status_code=status.HTTP_422_UNPROCESSABLE_ENTITY,
                    detail="Invalid country code"
                )
            
            # Generate avatar URL
            if profile_data.avatar_preference == "default":
                avatar_num = random.randint(1, 10)
                avatar_url = f"https://avatars.example.com/default-{avatar_num}.png"
            elif profile_data.avatar_preference == "generated":
                avatar_url = f"https://avatars.example.com/generated/{user_id}.png"
            else:
                avatar_url = "https://avatars.example.com/placeholder.png"
            
            # Create user profile
            user_profile = UserProfile(
                user_id=user_id,
                username=profile_data.username.lower(),
                country=profile_data.country,
                avatar_url=avatar_url
            )
            db.add(user_profile)
            
            # Update user record
            user.profile_completed = True
            user.avatar_url = avatar_url
            user.subscription_role = "Free"
            user.full_name = profile_data.full_name or user.full_name
            
            db.commit()
            
            logger.info(f"Profile created for user: {user_id}, username: {profile_data.username}")
            
            return {
                "user_id": user_id,
                "username": profile_data.username,
                "full_name": user.full_name,
                "country": profile_data.country,
                "avatar_url": avatar_url,
                "role": "Free",
                "level": user.level,
                "profile_completed": True,
                "created_at": user_profile.created_at.isoformat()
            }
        
        except HTTPException:
            raise
        except Exception as e:
            logger.error(f"Profile creation error: {str(e)}", exc_info=True)
            raise HTTPException(
                status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
                detail="Failed to create profile"
            )
```

### API Handler

```python
# app/api/v1/profiles.py
from fastapi import APIRouter, Depends, HTTPException, status
from sqlalchemy.orm import Session
from app.schemas.profile import CreateProfileSchema, ProfileResponseSchema
from app.services.profile_service import ProfileService
from app.database import get_db
from app.security import get_current_user
import logging

router = APIRouter(prefix="/profiles", tags=["profiles"])
logger = logging.getLogger(__name__)

@router.post("", response_model=ProfileResponseSchema)
async def create_profile(
    profile_data: CreateProfileSchema,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> ProfileResponseSchema:
    """
    Create user profile after registration.
    
    Request Body:
    - username: 3-30 chars (alphanumeric + underscore, unique)
    - full_name: Optional full name
    - country: Optional ISO 3166-1 alpha-2 code
    - avatar_preference: Optional (default, generated, custom)
    
    Returns:
    - 200 OK: Profile created
    - 400 Bad Request: Profile already exists
    - 409 Conflict: Username taken
    - 422 Unprocessable Entity: Validation error
    - 500 Internal Server Error: Server error
    """
    try:
        logger.info(f"Profile creation for user: {user.id}, username: {profile_data.username}")
        
        result = await ProfileService.create_profile(
            user_id=user.id,
            profile_data=profile_data,
            db=db
        )
        
        return ProfileResponseSchema(
            success=True,
            data=result
        )
    
    except HTTPException:
        raise
    except Exception as e:
        logger.error(f"Error creating profile: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to create profile"
        )
```

## 📊’ Security Considerations

1. **Authentication Required**
   - Verify JWT token is valid and not expired
   - Extract user_id from token claims
   - Only authenticated users can create profile

2. **Username Validation**
   - Strict regex pattern: `^[a-zA-Z0-9_]+$`
   - Length limits: 3-30 chars (prevent abuse)
   - Case-insensitive uniqueness check
   - Prevent reserved keywords/admin usernames

3. **Rate Limiting**
   - One profile per user (prevent duplicate creation attempts)
   - Profile creation locked after 3 failed attempts
   - Use Redis for rate limit tracking

4. **Data Validation**
   - Validate country code against ISO 3166-1 list
   - Validate avatar preference enum
   - Sanitize all text inputs (remove XSS vectors)

5. **Avatar URL Generation**
   - Use deterministic (consistent) URLs for avatars
   - Validate avatar URL is from allowed domain
   - Store as full URL with protocol for security
