# G01_F02_SF01: Refresh Token

✅ MVP | **Token Management** | **Authentication**

## 📋 Description

Use refresh token to obtain new access token without re-authentication. Access tokens expire after 1 hour; refresh tokens valid for 7 days.

## 🎯 API Specification

**Endpoint**: `POST /api/v1/auth/refresh-token`

**Input**
```json
{
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

**Output (200 OK)**
```json
{
  "success": true,
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "token_type": "Bearer",
    "expires_in": 3600
  }
}
```

**Errors**: 401 (invalid token), 429 (rate limit), 500 (error)

## 🗏️ Business Logic

1. **Validate Token Provided**
   - Check refresh_token in request body
   - Return 400 if missing

2. **Check Token Not Blacklisted**
   - Query: `SELECT * FROM token_blacklist WHERE token = :token`
   - Return 401 if revoked

3. **Decode Token**
   - Decode JWT without verification first

4. **Verify Token Signature**
   - Use SECRET_KEY to verify
   - Return 401 if invalid signature

5. **Check Token Expiry**
   - Verify exp > NOW()
   - Return 401 if expired

6. **Extract User ID**
   - Get user_id from token claims
   - Query user from database
   - Return 404 if user not found

7. **Verify User Active**
   - Check user.is_active = true
   - Return 403 if inactive

8. **Generate New Access Token**
   - Create JWT: user_id, email, role, level, exp=now+1h

9. **Apply Rate Limit**
   - Track per user: `token_refresh:{user_id}`
   - Limit: 10 refreshes per minute
   - Return 429 if exceeded

10. **Return New Access Token**
    - HTTP 200 OK
    - Include new access_token, token_type, expires_in

## 🗄️ Database Schema

Existing: `users`, `tokens`

## 💻 Backend Implementation

```python
# app/api/v1/auth.py

@router.post("/refresh-token")
async def refresh_token(
    refresh_data: dict,
    db: Session = Depends(get_db)
):
    """Get new access token using refresh token"""
    try:
        result = await AuthService.refresh_access_token(
            refresh_data.get("refresh_token"),
            db
        )
        return {"success": True, "data": result}
    except HTTPException:
        raise
    except Exception as e:
        logger.error(f"Token refresh error: {str(e)}")
        raise HTTPException(status_code=500, detail="Failed to refresh token")

# Service
async def refresh_access_token(refresh_token: str, db: Session) -> dict:
    """Generate new access token"""
    try:
        if not refresh_token:
            raise HTTPException(status_code=400, detail="Refresh token required")
        
        # Check blacklist
        blacklisted = db.query(TokenBlacklist).filter_by(token=refresh_token).first()
        if blacklisted:
            raise HTTPException(status_code=401, detail="Token revoked")
        
        # Decode and verify
        payload = decode_jwt(refresh_token)
        user_id = payload.get("user_id")
        
        # Rate limit
        rate_key = f"token_refresh:{user_id}"
        refresh_count = await redis_client.get(rate_key)
        if refresh_count and int(refresh_count) >= 10:
            raise HTTPException(status_code=429, detail="Too many refresh attempts")
        
        # Get user
        user = db.query(User).filter(User.id == user_id).first()
        if not user:
            raise HTTPException(status_code=404, detail="User not found")
        
        if not user.is_active:
            raise HTTPException(status_code=403, detail="User inactive")
        
        # Generate new access token
        new_access_token = generate_jwt_token(
            data={"user_id": user.id, "email": user.email, "role": "Free", "level": user.level},
            expires_delta=timedelta(hours=1)
        )
        
        await redis_client.incr(rate_key)
        await redis_client.expire(rate_key, 60)
        
        logger.info(f"Token refreshed for user: {user.id}")
        
        return {
            "access_token": new_access_token,
            "token_type": "Bearer",
            "expires_in": 3600
        }
    except HTTPException:
        raise
    except Exception as e:
        logger.error(f"Token refresh error: {str(e)}")
        raise HTTPException(status_code=500, detail="Token refresh failed")
```

## 📊’ Security

1. **Token Verification**: Verify signature and expiry
2. **User Status**: Only active users can refresh
3. **Rate Limiting**: Prevent token refresh bombing
4. **Blacklist Check**: Revoked tokens rejected
5. **No Refresh Token Rotation**: Don't issue new refresh token (keep same)

@pytest.mark.asyncio
async def test_refresh_token_success(async_client, refresh_token):
    """Test successful token refresh"""
    response = await async_client.post(
        "/api/v1/auth/refresh-token",
        json={"refresh_token": refresh_token}
    )
    assert response.status_code == 200
    assert "access_token" in response.json()["data"]

async def test_refresh_token_rate_limit(async_client, refresh_token):
    """Test rate limiting"""
    for i in range(10):
        response = await async_client.post(
            "/api/v1/auth/refresh-token",
            json={"refresh_token": refresh_token}
        )
        assert response.status_code == 200
    
    response = await async_client.post(
        "/api/v1/auth/refresh-token",
        json={"refresh_token": refresh_token}
    )
    assert response.status_code == 429
