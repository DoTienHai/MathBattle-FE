# G01_F02_SF02: Logout User

✅ MVP | **Token Management** | **Session**

## 📋 Description

Logout user by revoking access and refresh tokens. Add tokens to blacklist and end session.

## 🎯 API Specification

**Endpoint**: `POST /api/v1/auth/logout`

**Headers**
```
Authorization: Bearer {access_token}
```

**Input** (optional)
```json
{
  "logout_all_devices": false
}
```

**Output (200 OK)**
```json
{
  "success": true,
  "data": {
    "message": "Logged out successfully"
  }
}
```

## 🗏️ Business Logic

1. **Verify Authentication**
   - Extract user_id from access_token

2. **Get Current Tokens**
   - Extract access_token from Authorization header
   - Extract refresh_token from database (if remember_me used)

3. **Blacklist Tokens**
   - Add access_token to token_blacklist table
   - Add refresh_token to blacklist if exists
   - Set expiry = token expiration time

4. **Optional: Logout All Devices**
   - If logout_all_devices = true:
     - Query all tokens for user from tokens table
     - Add all to blacklist

5. **End Session**
   - Query login_sessions WHERE user_id = :user_id
   - Update last_activity = NOW()
   - Option to delete session or mark as ended

6. **Return Success**
   - HTTP 200 OK
   - Message: "Logged out successfully"

## 🗄️ Database Schema

```sql
CREATE TABLE token_blacklist (
  id SERIAL PRIMARY KEY,
  token VARCHAR(500) NOT NULL,
  expires_at TIMESTAMP NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 💻 Backend Implementation

```python
@router.post("/logout")
async def logout(
    logout_all: dict,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db),
    request: Request = None
):
    """Logout user and revoke tokens"""
    try:
        token = get_token_from_header(request)
        result = await AuthService.logout_user(user.id, token, logout_all.get("logout_all_devices", False), db)
        return {"success": True, "data": result}
    except Exception as e:
        logger.error(f"Logout error: {str(e)}")
        raise HTTPException(status_code=500, detail="Logout failed")

async def logout_user(user_id: int, access_token: str, logout_all: bool, db: Session) -> dict:
    """Logout user and revoke tokens"""
    try:
        # Add current token to blacklist
        payload = decode_jwt(access_token)
        exp = datetime.fromtimestamp(payload.get("exp"))
        
        blacklisted_token = TokenBlacklist(
            token=access_token,
            expires_at=exp
        )
        db.add(blacklisted_token)
        
        if logout_all:
            # Get all user tokens
            tokens = db.query(Token).filter_by(user_id=user_id).all()
            for token_record in tokens:
                blacklist_entry = TokenBlacklist(
                    token=token_record.refresh_token,
                    expires_at=token_record.expires_at
                )
                db.add(blacklist_entry)
                db.delete(token_record)
        
        db.commit()
        logger.info(f"User logged out: {user_id}")
        
        return {"message": "Logged out successfully"}
    except Exception as e:
        logger.error(f"Logout error: {str(e)}")
        raise HTTPException(status_code=500, detail="Logout failed")
```

## 📊’ Security

1. **Token Blacklisting**: Immediately revoke access/refresh tokens
2. **Expiry Management**: Cleanup blacklisted tokens when expired
3. **Logout All**: Allow user to revoke all sessions for security

@pytest.mark.asyncio
async def test_logout_success(async_client, user_headers):
    """Test successful logout"""
    response = await async_client.post("/api/v1/auth/logout", headers=user_headers)
    assert response.status_code == 200

@pytest.mark.asyncio
async def test_logout_token_revoked(async_client, user_headers):
    """Test that token is revoked after logout"""
    await async_client.post("/api/v1/auth/logout", headers=user_headers)
    response = await async_client.get("/api/v1/users/profile", headers=user_headers)
    assert response.status_code == 401

@pytest.mark.asyncio
async def test_logout_all_devices(async_client, user):
    """Test logout from all devices"""
    response = await async_client.post(
        "/api/v1/auth/logout",
        headers=get_auth_headers(user),
        json={"logout_all_devices": True}
    )
    assert response.status_code == 200

@pytest.mark.asyncio
async def test_logout_unauthorized(async_client):
