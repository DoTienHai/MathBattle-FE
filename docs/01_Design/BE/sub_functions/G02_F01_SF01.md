## 📋 Description

List all available quizzes with pagination, filters (difficulty, category), and sorting options. Authenticated users can browse quizzes they're eligible to attempt.

---

## 🗏️ Business Logic (11 Steps)

1. **Validate User Authentication**
   - Verify JWT token is valid
   - Extract user_id from token claims
   - Return 401 if token invalid or expired

2. **Validate Query Parameters**
   - Check page â‰¥ 1
   - Check page_size âˆˆ [1, 100]
   - Validate difficulty âˆˆ [easy, medium, hard] if provided
   - Validate sort_by âˆˆ [created_at, title, difficulty]
   - Return 400 Bad Request if validation fails

3. **Build Base Query**
   - SELECT quizzes WHERE is_deleted=false AND is_published=true
   - Include quiz metadata (title, description, difficulty, category)

4. **Apply Difficulty Filter**
   - If difficulty parameter provided: WHERE difficulty = :difficulty
   - Default: no filter (show all difficulties)

5. **Apply Category Filter**
   - If category provided: WHERE category = :category
   - Ensure category exists in quiz_categories table
   - Return 404 if invalid category

6. **Apply Search Filter**
   - If search provided: WHERE title ILIKE :search OR description ILIKE :search
   - Use case-insensitive search

7. **Apply Status Filter**
   - If user status == "completed": WHERE quiz_id IN (SELECT quiz_id FROM quiz_attempts WHERE user_id=:user_id AND is_passed=true)
   - If user status == "available": Exclude completed quizzes from results
   - If user status == "locked": Check prerequisites (level requirements, parent quiz completion)

8. **Apply Sorting**
   - If sort_by="created_at": ORDER BY quizzes.created_at [ASC|DESC]
   - If sort_by="title": ORDER BY quizzes.title [ASC|DESC]
   - If sort_by="difficulty": ORDER BY FIELD(difficulty, 'easy', 'medium', 'hard')

9. **Fetch Metadata for Each Quiz**
   - Count questions: SELECT COUNT(*) FROM quiz_questions WHERE quiz_id=:quiz_id
   - Get user best score: SELECT MAX(score) FROM quiz_attempts WHERE user_id=:user_id AND quiz_id=:quiz_id
   - Get attempt count: SELECT COUNT(*) FROM quiz_attempts WHERE user_id=:user_id AND quiz_id=:quiz_id
   - Determine user_status (available | locked | completed)

10. **Apply Pagination**
    - Calculate offset = (page - 1) * page_size
    - LIMIT page_size OFFSET offset
    - Get total_items: SELECT COUNT(*) without pagination

11. **Return Response**
    - Map quizzes to response schema
    - Include pagination metadata
    - Cache for 5 minutes (Redis key: quizzes:user:{user_id}:filters:{hash})

### Service Layer

```python
# app/services/quiz_service.py
from sqlalchemy import and_, or_, func
from sqlalchemy.orm import Session
from typing import Optional, Tuple, Dict, List
import math

class QuizService:
    
    @staticmethod
    async def get_available_quizzes(
        user_id: int,
        page: int = 1,
        page_size: int = 10,
        difficulty: Optional[str] = None,
        category: Optional[str] = None,
        search: Optional[str] = None,
        sort_by: str = "created_at",
        sort_order: str = "desc",
        status: Optional[str] = None,
        db: Session = None
    ) -> Tuple[List[Dict], int]:
        """
        Fetch paginated list of available quizzes with filters.
        
        Args:
            user_id: ID of authenticated user
            page: Page number (1-indexed)
            page_size: Items per page
            difficulty: Filter by difficulty level
            category: Filter by category
            search: Search in title/description
            sort_by: Sort column (created_at, title, difficulty)
            sort_order: ASC or DESC
            status: User status (available, locked, completed)
            db: Database session
            
        Returns:
            Tuple of (quizzes_list, total_count)
        """
        # Build base query
        query = db.query(Quiz).filter(
            and_(
                Quiz.is_deleted == False,
                Quiz.is_published == True
            )
        )
        
        # Apply filters
        if difficulty:
            query = query.filter(Quiz.difficulty == difficulty)
        
        if category:
            # Verify category exists
            cat = db.query(QuizCategory).filter(
                QuizCategory.name == category
            ).first()
            if not cat:
                raise ValueError(f"Category '{category}' not found")
            query = query.filter(Quiz.category == category)
        
        if search:
            search_pattern = f"%{search}%"
            query = query.filter(
                or_(
                    Quiz.title.ilike(search_pattern),
                    Quiz.description.ilike(search_pattern)
                )
            )
        
        # Get total count before pagination
        total_count = query.count()
        
        # Apply sorting
        if sort_by == "title":
            order_col = Quiz.title
        elif sort_by == "difficulty":
            # Custom sorting: easy â†’ medium â†’ hard
            order_col = func.field(
                Quiz.difficulty,
                "easy", "medium", "hard"
            )
        else:  # created_at
            order_col = Quiz.created_at
        
        if sort_order.lower() == "asc":
            query = query.order_by(order_col.asc())
        else:
            query = query.order_by(order_col.desc())
        
        # Apply pagination
        offset = (page - 1) * page_size
        quizzes = query.offset(offset).limit(page_size).all()
        
        # Enrich with user-specific data
        result = []
        for quiz in quizzes:
            # Get question count
            question_count = db.query(func.count(QuizQuestion.id)).filter(
                QuizQuestion.quiz_id == quiz.id
            ).scalar()
            
            # Get user attempts
            attempt_count = db.query(func.count(QuizAttempt.id)).filter(
                and_(
                    QuizAttempt.user_id == user_id,
                    QuizAttempt.quiz_id == quiz.id
                )
            ).scalar()
            
            # Get best score
            best_score = db.query(func.max(QuizAttempt.score)).filter(
                and_(
                    QuizAttempt.user_id == user_id,
                    QuizAttempt.quiz_id == quiz.id
                )
            ).scalar()
            
            # Determine user status
            is_completed = db.query(QuizAttempt).filter(
                and_(
                    QuizAttempt.user_id == user_id,
                    QuizAttempt.quiz_id == quiz.id,
                    QuizAttempt.is_passed == True
                )
            ).first() is not None
            
            user_status = "completed" if is_completed else "available"
            if status == "locked":
                user_status = "locked"  # Add prerequisite logic if needed
            
            result.append({
                "quiz_id": quiz.id,
                "title": quiz.title,
                "description": quiz.description,
                "category": quiz.category,
                "difficulty": quiz.difficulty,
                "question_count": question_count or 0,
                "time_limit_minutes": quiz.time_limit_minutes,
                "passing_score": quiz.passing_score,
                "points_reward": quiz.points_reward,
                "user_status": user_status,
                "user_attempts": attempt_count or 0,
                "best_score": best_score,
                "created_at": quiz.created_at,
                "thumbnail_url": quiz.thumbnail_url
            })
        
        return result, total_count
```

```python
# app/api/v1/quizzes.py
from fastapi import APIRouter, Depends, Query, HTTPException, status
from typing import Optional, List
import math

router = APIRouter(prefix="/quizzes", tags=["quizzes"])

@router.get("", response_model=QuizListResponse)
async def get_available_quizzes(
    page: int = Query(1, ge=1),
    page_size: int = Query(10, ge=1, le=100),
    difficulty: Optional[str] = Query(None, regex="^(easy|medium|hard)$"),
    category: Optional[str] = Query(None),
    search: Optional[str] = Query(None, max_length=100),
    sort_by: str = Query("created_at", regex="^(created_at|title|difficulty)$"),
    sort_order: str = Query("desc", regex="^(asc|desc)$"),
    status: Optional[str] = Query(None, regex="^(available|locked|completed)$"),
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> QuizListResponse:
    """
    Get paginated list of available quizzes.
    
    Query Parameters:
    - page: Page number (default: 1)
    - page_size: Items per page (default: 10, max: 100)
    - difficulty: Filter by difficulty (easy, medium, hard)
    - category: Filter by category name
    - search: Search in title/description
    - sort_by: Sort by (created_at, title, difficulty)
    - sort_order: Sort order (asc, desc)
    - status: Filter by user status (available, locked, completed)
    
    Returns:
    - Paginated quiz list with user-specific metadata
    """
    try:
        logger.info(f"User {user.id} fetching quizzes: page={page}, size={page_size}")
        
        quizzes, total_count = await QuizService.get_available_quizzes(
            user_id=user.id,
            page=page,
            page_size=page_size,
            difficulty=difficulty,
            category=category,
            search=search,
            sort_by=sort_by,
            sort_order=sort_order,
            status=status,
            db=db
        )
        
        total_pages = math.ceil(total_count / page_size)
        
        return QuizListResponse(
            success=True,
            data={
                "quizzes": quizzes,
                "pagination": {
                    "page": page,
                    "page_size": page_size,
                    "total_items": total_count,
                    "total_pages": total_pages,
                    "has_next": page < total_pages,
                    "has_prev": page > 1
                }
            }
        )
    
    except ValueError as e:
        logger.warning(f"Validation error: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to fetch quizzes: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to fetch quizzes"
        )
```

## 📊’ Security Considerations

1. **Authentication Required**
   - All requests must include valid JWT token
   - Return 401 Unauthorized if token invalid/expired
   - Extract user_id from token claims

2. **Authorization & Data Access**
   - User can only see published quizzes
   - User metadata (attempts, best_score) calculated only for authenticated user
   - Prevent access to unpublished quizzes (only admins)

3. **Input Validation**
   - Validate all query parameters with strict regex patterns
   - page_size limited to max 100 to prevent resource exhaustion
   - Search string max 100 chars to prevent long queries
   - Return 400 Bad Request for invalid inputs

4. **Rate Limiting**
   - Implement rate limit: 100 requests per minute per user
   - Return 429 Too Many Requests if exceeded
   - Cache results for 5 minutes (Redis) to reduce database queries

5. **SQL Injection Prevention**
   - Use parameterized queries (SQLAlchemy handles this)
   - Never concatenate user input directly into SQL
   - Use ORM models for all queries
