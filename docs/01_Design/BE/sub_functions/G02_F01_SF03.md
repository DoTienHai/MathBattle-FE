## 📋 Description

Fetch all questions for a specific quiz with their multiple choice options. Returns questions in randomized order with options but WITHOUT correct answers (answers revealed only after user submission).

## 🗏️ Business Logic (10 Steps)

1. **Validate User Authentication**
   - Extract JWT token from Authorization header
   - Verify token validity and extract user_id
   - Return 401 Unauthorized if invalid

2. **Validate Quiz ID Parameter**
   - Check quiz_id is positive integer
   - Return 400 Bad Request if invalid format

3. **Fetch Quiz & Verify Access**
   - Query: SELECT * FROM quizzes WHERE id = :quiz_id AND is_deleted = false
   - Return 404 if quiz not found
   - For unpublished quizzes: check if user is creator or admin
   - Return 403 if unauthorized

4. **Check Quiz Eligibility**
   - Verify user meets prerequisites
   - Verify user level â‰¥ quiz minimum_level_requirement
   - Return 403 if user not eligible
   - Log eligibility checks for security audit

5. **Verify User Enrollment**
   - Check quiz_access_control table
   - Verify user has permission to attempt quiz
   - Return 403 if user not enrolled/permitted

6. **Fetch All Quiz Questions**
   - Query: SELECT * FROM quiz_questions WHERE quiz_id = :quiz_id ORDER BY order
   - Join with questions table to get question details
   - Ensure questions are not deleted

7. **Fetch Options for Each Question**
   - For each question: SELECT * FROM quiz_options WHERE question_id = :question_id
   - Order options by order field
   - Set is_correct = null (security: don't reveal correct answer)
   - Include option_id, option_number (A, B, C, D), option_text

8. **Randomize Question Order (if requested)**
   - If randomize=true: shuffle questions array
   - Keep option order consistent (A, B, C, D) but shuffle within each question
   - Seed randomization with: hash(user_id + quiz_id + date) for reproducibility during quiz

9. **Include Metadata**
   - Add time_limit_minutes to response
   - Add total_questions count
   - Add explanation field (null until after submission)

10. **Return Response & Log Access**
    - Map questions to response schema
    - Log quiz access attempt (timestamp, user_id, quiz_id)
    - Cache for 1 minute (to prevent repeated loads during quiz)

### Service Layer

```python
# app/services/question_service.py
from sqlalchemy import and_
from sqlalchemy.orm import Session
from typing import List, Dict, Tuple
import random
import hashlib
from datetime import datetime

class QuestionService:
    
    @staticmethod
    async def get_quiz_questions(
        quiz_id: int,
        user_id: int,
        randomize: bool = True,
        db: Session = None
    ) -> Dict:
        """
        Fetch all questions for a quiz with options (without correct answers).
        
        Args:
            quiz_id: ID of quiz
            user_id: ID of authenticated user
            randomize: Whether to randomize question order
            db: Database session
            
        Returns:
            Dictionary with quiz and all questions/options
            
        Raises:
            ValueError: If quiz not found
            PermissionError: If user unauthorized or ineligible
        """
        # Fetch quiz
        quiz = db.query(Quiz).filter(
            and_(
                Quiz.id == quiz_id,
                Quiz.is_deleted == False
            )
        ).first()
        
        if not quiz:
            raise ValueError(f"Quiz {quiz_id} not found")
        
        # Check access
        if not quiz.is_published:
            user = db.query(User).filter(User.id == user_id).first()
            if user_id != quiz.created_by_id and user.role != "admin":
                raise PermissionError("Not authorized to view this quiz")
        
        # Check eligibility
        user = db.query(User).filter(User.id == user_id).first()
        if user.current_level < quiz.minimum_level_requirement:
            raise PermissionError("User level too low for this quiz")
        
        # Check prerequisites
        prerequisites = db.query(QuizPrerequisite).filter(
            QuizPrerequisite.quiz_id == quiz_id
        ).all()
        
        for prereq in prerequisites:
            is_passed = db.query(QuizAttempt).filter(
                and_(
                    QuizAttempt.user_id == user_id,
                    QuizAttempt.quiz_id == prereq.prerequisite_quiz_id,
                    QuizAttempt.is_passed == True
                )
            ).first() is not None
            
            if not is_passed:
                raise PermissionError("Prerequisites not completed")
        
        # Check access control
        access = db.query(QuizAccessControl).filter(
            and_(
                QuizAccessControl.user_id == user_id,
                QuizAccessControl.quiz_id == quiz_id
            )
        ).first()
        
        if access is None and not user.is_admin:
            raise PermissionError("User not enrolled in this quiz")
        
        # Log access
        logger.info(f"User {user_id} loading quiz {quiz_id} questions")
        
        # Fetch questions
        quiz_questions = db.query(QuizQuestion).filter(
            QuizQuestion.quiz_id == quiz_id
        ).order_by(QuizQuestion.order).all()
        
        if not quiz_questions:
            raise ValueError(f"No questions found for quiz {quiz_id}")
        
        questions_list = []
        
        for idx, qz_q in enumerate(quiz_questions):
            question = db.query(Question).filter(
                and_(
                    Question.id == qz_q.question_id,
                    Question.is_deleted == False
                )
            ).first()
            
            if not question:
                continue
            
            # Fetch options
            options = db.query(QuizOption).filter(
                QuizOption.question_id == question.id
            ).order_by(QuizOption.order).all()
            
            options_list = [
                QuizOption(
                    option_id=opt.id,
                    option_number=opt.option_number,
                    option_text=opt.option_text,
                    is_correct=None  # SECURITY: Don't expose correct answer
                )
                for opt in options
            ]
            
            # Shuffle options (but keep same questions for same user)
            if randomize:
                seed = int(hashlib.md5(
                    f"{user_id}{quiz_id}{question.id}".encode()
                ).hexdigest(), 16)
                random.seed(seed)
                random.shuffle(options_list)
            
            questions_list.append(
                QuestionDetail(
                    question_id=question.id,
                    question_number=idx + 1,
                    question_text=question.text,
                    question_type=question.question_type or "multiple_choice",
                    difficulty=question.difficulty or "medium",
                    points=qz_q.points,
                    options=options_list,
                    explanation=None
                )
            )
        
        # Randomize question order if requested
        if randomize:
            seed = int(hashlib.md5(
                f"{user_id}{quiz_id}".encode()
            ).hexdigest(), 16)
            random.seed(seed)
            random.shuffle(questions_list)
        
        return {
            "quiz_id": quiz.id,
            "quiz_title": quiz.title,
            "total_questions": len(questions_list),
            "time_limit_minutes": quiz.time_limit_minutes,
            "questions": questions_list
        }
```

### API Handler

```python
# app/api/v1/questions.py
from fastapi import APIRouter, Depends, HTTPException, status, Path, Query
import logging

router = APIRouter(prefix="/quizzes", tags=["questions"])
logger = logging.getLogger(__name__)

@router.get("/{quiz_id}/questions", response_model=dict)
async def get_quiz_questions(
    quiz_id: int = Path(..., ge=1),
    randomize: bool = Query(True),
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Get all questions and options for a quiz.
    
    Path Parameters:
    - quiz_id: ID of quiz (required, > 0)
    
    Query Parameters:
    - randomize: Randomize question/option order (default: true)
    
    Returns:
    - All questions with multiple choice options
    - Correct answers hidden until submission
    
    Responses:
    - 200: Questions retrieved successfully
    - 400: User not eligible (level too low, etc.)
    - 401: Unauthorized
    - 403: Access denied or prerequisites not met
    - 404: Quiz not found
    """
    try:
        logger.info(f"User {user.id} fetching questions for quiz {quiz_id}")
        
        questions_data = await QuestionService.get_quiz_questions(
            quiz_id=quiz_id,
            user_id=user.id,
            randomize=randomize,
            db=db
        )
        
        return {
            "success": True,
            "data": questions_data
        }
    
    except ValueError as e:
        logger.warning(f"Quiz not found: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=str(e)
        )
    
    except PermissionError as e:
        logger.warning(f"Permission denied: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to fetch questions: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to fetch questions"
        )
```

## 📊’ Security Considerations

1. **Never Expose Correct Answers**
   - Set is_correct = null in response
   - Only reveal after user submits answer and quiz ends
   - Validate answers against database, never client-side

2. **Access Control Validation**
   - User must be enrolled in quiz (quiz_access_control)
   - User must meet level requirements
   - User must have completed prerequisites
   - Admin bypasses some restrictions

3. **Randomization Strategy**
   - Use seed = hash(user_id + quiz_id) for consistency
   - Same user gets same order per quiz per day
   - Different users get different random orders
   - Prevents cheating by shared question order patterns

4. **Rate Limiting**
   - 100 requests/minute per user
   - Cache for 1 minute (prevents multiple loads)
   - Log all access attempts for audit trail

5. **Input Validation**
   - quiz_id must be positive integer
   - randomize must be boolean
   - Validate path and query parameters
   - Return 422 for validation errors
