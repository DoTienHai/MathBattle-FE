## 📋 Description

Admin or content creator creates a new quiz with metadata, configuration, and associates it with questions. Creates quiz record, initializes settings, and optionally publishes for users.

## 🗏️ Business Logic (12 Steps)

1. **Validate User Authentication & Authorization**
   - Extract JWT token from Authorization header
   - Verify token validity and extract user_id
   - Check user.role == "admin" OR user.can_create_quizzes = true
   - Return 401 if unauthorized
   - Return 403 if user lacks admin/creator permissions
   - Log authorization check for audit trail

2. **Validate Required Input Fields**
   - title: 1-255 chars, not null, unique check (case-insensitive)
   - category: must exist in quiz_categories table
   - difficulty: must be in [easy, medium, hard]
   - Return 400 Bad Request for any validation failure
   - Return 409 Conflict if title already exists

3. **Validate Optional Fields**
   - time_limit_minutes: 5-120 range
   - passing_score: 0-100 range
   - points_reward: 0-1000 range
   - minimum_level_requirement: 1-100 range
   - Return 400 for out-of-range values

4. **Validate Question IDs (if provided)**
   - For each question_id in question_ids:
     - Query: SELECT * FROM questions WHERE id = :question_id
     - Verify question exists and not deleted
     - Return 404 if question not found
   - Count total valid questions

5. **Check for Duplicate Quiz**
   - Query: SELECT * FROM quizzes WHERE title ILIKE :title
   - Case-insensitive check
   - Return 409 Conflict if duplicate exists

6. **Create Quiz Record**
   - INSERT INTO quizzes (title, description, category, difficulty, ...)
   - Set created_by_id = user_id
   - Set created_at = CURRENT_TIMESTAMP
   - Set is_published = is_published parameter (default: false)
   - Return quiz_id

7. **Add Questions to Quiz**
   - If question_ids provided:
     - For each question_id with order:
       - INSERT INTO quiz_questions (quiz_id, question_id, order, points)
       - Assign 10 points per question (default)
   - If no questions provided: create empty quiz

8. **Update Question Count**
   - Query: COUNT(*) FROM quiz_questions WHERE quiz_id = :quiz_id
   - UPDATE quizzes SET question_count = :count WHERE id = :quiz_id

9. **Store Tags (if provided)**
   - If tags array provided:
     - Convert to comma-separated string
     - UPDATE quizzes SET tags = :tags WHERE id = :quiz_id

10. **Create Access Control Rules**
    - If is_published = true:
      - Check is_public flag (true = all users can access)
      - If is_public = false: only enrolled users via quiz_access_control
    - INSERT audit log: quiz_created, quiz_id, created_by_id, timestamp

11. **Handle Published Status**
    - If is_published = true:
      - Verify quiz has â‰¥ 3 questions
      - Return 400 if insufficient questions
      - Log publication event
    - If is_published = false:
      - Mark as draft, only visible to creator/admin

12. **Return Success Response**
    - Return 201 Created with quiz metadata
    - Include quiz_id for future references
    - Log quiz creation event with full details

### Service Layer

```python
# app/services/quiz_service.py
from sqlalchemy import and_
from sqlalchemy.orm import Session
from typing import List, Optional, Dict
import logging

logger = logging.getLogger(__name__)

class QuizService:
    
    @staticmethod
    async def create_quiz(
        quiz_data: QuizCreate,
        user_id: int,
        db: Session
    ) -> Dict:
        """
        Create new quiz with questions.
        
        Args:
            quiz_data: Quiz creation request data
            user_id: ID of admin creating quiz
            db: Database session
            
        Returns:
            Dictionary with created quiz details
            
        Raises:
            PermissionError: If user not authorized
            ValueError: If validation fails
        """
        # Check authorization
        user = db.query(User).filter(User.id == user_id).first()
        if not user or (user.role != "admin" and not user.can_create_quizzes):
            raise PermissionError("Only admins can create quizzes")
        
        # Check category exists
        category = db.query(QuizCategory).filter(
            QuizCategory.name == quiz_data.category
        ).first()
        if not category:
            raise ValueError(f"Category '{quiz_data.category}' not found")
        
        # Check duplicate title
        existing = db.query(Quiz).filter(
            func.lower(Quiz.title) == func.lower(quiz_data.title)
        ).first()
        if existing:
            raise ValueError("Quiz title already exists")
        
        # Validate questions if provided
        questions_to_add = []
        if quiz_data.question_ids:
            for q_id in quiz_data.question_ids:
                question = db.query(Question).filter(
                    and_(
                        Question.id == q_id,
                        Question.is_deleted == False
                    )
                ).first()
                if not question:
                    raise ValueError(f"Question {q_id} not found")
                questions_to_add.append(question)
        
        # If publishing, require minimum questions
        if quiz_data.is_published and len(questions_to_add) < 3:
            raise ValueError("Published quiz must have at least 3 questions")
        
        # Create quiz
        quiz = Quiz(
            title=quiz_data.title,
            description=quiz_data.description,
            category=quiz_data.category,
            difficulty=quiz_data.difficulty,
            time_limit_minutes=quiz_data.time_limit_minutes,
            passing_score=quiz_data.passing_score,
            points_reward=quiz_data.points_reward,
            is_published=quiz_data.is_published,
            created_by_id=user_id,
            minimum_level_requirement=quiz_data.minimum_level_requirement,
            difficulty_level_range=quiz_data.difficulty_level_range,
            thumbnail_url=quiz_data.thumbnail_url,
            question_count=len(questions_to_add),
            tags=",".join(quiz_data.tags) if quiz_data.tags else None
        )
        
        db.add(quiz)
        db.flush()  # Get quiz_id
        
        # Add questions
        for idx, question in enumerate(questions_to_add):
            quiz_question = QuizQuestion(
                quiz_id=quiz.id,
                question_id=question.id,
                order=idx + 1,
                points=10
            )
            db.add(quiz_question)
        
        # Log audit
        audit_log = QuizAuditLog(
            quiz_id=quiz.id,
            action="quiz_created",
            performed_by_id=user_id,
            details=f"Created quiz: {quiz.title}, {len(questions_to_add)} questions"
        )
        db.add(audit_log)
        
        db.commit()
        logger.info(f"Admin {user_id} created quiz {quiz.id}: {quiz.title}")
        
        return {
            "quiz_id": quiz.id,
            "title": quiz.title,
            "description": quiz.description,
            "category": quiz.category,
            "difficulty": quiz.difficulty,
            "question_count": quiz.question_count,
            "time_limit_minutes": quiz.time_limit_minutes,
            "passing_score": quiz.passing_score,
            "points_reward": quiz.points_reward,
            "is_published": quiz.is_published,
            "created_by_id": quiz.created_by_id,
            "created_at": quiz.created_at,
            "minimum_level_requirement": quiz.minimum_level_requirement
        }

# app/api/v1/quizzes.py
from fastapi import APIRouter, Depends, HTTPException, status
import logging

router = APIRouter(prefix="/quizzes", tags=["quizzes"])
logger = logging.getLogger(__name__)

@router.post("", response_model=dict, status_code=201)
async def create_quiz(
    quiz_data: QuizCreate,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Create a new quiz (admin only).
    
    Request Body:
    - title: Quiz title (required, 1-255 chars)
    - category: Category name (required)
    - difficulty: easy|medium|hard (required)
    - description: Quiz description (optional)
    - time_limit_minutes: 5-120 (optional, default: 30)
    - passing_score: 0-100 (optional, default: 70)
    - points_reward: 0-1000 (optional, default: 50)
    - question_ids: Array of question IDs (optional)
    - is_published: Whether to publish (optional, default: false)
    
    Returns:
    - 201: Quiz created successfully with quiz_id
    - 400: Validation error (invalid inputs, missing questions)
    - 401: Unauthorized
    - 403: User not admin
    - 409: Duplicate quiz title
    """
    try:
        logger.info(f"Admin {user.id} creating quiz: {quiz_data.title}")
        
        quiz_result = await QuizService.create_quiz(
            quiz_data=quiz_data,
            user_id=user.id,
            db=db
        )
        
        return {
            "success": True,
            "data": {
                **quiz_result,
                "message": "Quiz created successfully"
            }
        }
    
    except PermissionError as e:
        logger.warning(f"Permission denied: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail=str(e)
        )
    
    except ValueError as e:
        logger.warning(f"Validation error: {str(e)}")
        if "already exists" in str(e):
            raise HTTPException(
                status_code=status.HTTP_409_CONFLICT,
                detail=str(e)
            )
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to create quiz: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to create quiz"
        )
```

## 📊’ Security Considerations

1. **Authorization Check**
   - Only admins (role="admin") or users with can_create_quizzes=true
   - Return 403 Forbidden if unauthorized
   - Log all quiz creation attempts

2. **Input Validation**
   - Strict validation on all input fields
   - SQL injection prevention via parameterized queries
   - File upload validation for thumbnail_url (if applicable)
   - Return 400 Bad Request for invalid inputs

3. **Business Rules Enforcement**
   - Published quizzes must have â‰¥ 3 questions
   - Return 400 if rule violated
   - Unique title constraint prevents duplicate quizzes
   - Return 409 Conflict if duplicate

4. **Audit Logging**
   - Log all quiz creation in quiz_audit_log table
   - Include timestamp, user_id, quiz_id, details
   - Track question assignments and modifications

5. **Rate Limiting**
   - Limit admins to 50 quiz creations per hour
   - Return 429 Too Many Requests if exceeded
   - Prevents spam/abuse
