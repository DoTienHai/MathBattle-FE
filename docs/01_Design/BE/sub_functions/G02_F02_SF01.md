## 📋 Description

User initiates a quiz attempt by creating an attempt record. Validates user eligibility, records start timestamp, initializes attempt state, and returns session token for tracking progress.

## 🗏️ Business Logic (11 Steps)

1. **Validate User Authentication**
   - Extract JWT token from Authorization header
   - Verify token validity and extract user_id
   - Return 401 Unauthorized if invalid

2. **Validate Quiz ID Parameter**
   - Check quiz_id is positive integer
   - Return 400 Bad Request if invalid format

3. **Fetch Quiz Details**
   - Query: SELECT * FROM quizzes WHERE id = :quiz_id AND is_deleted = false
   - Return 404 Not Found if quiz not found
   - Verify quiz.is_published = true
   - Return 403 Forbidden if quiz not published

4. **Check User Eligibility**
   - Verify user.current_level â‰¥ quiz.minimum_level_requirement
   - Return 403 Forbidden if level too low
   - Check user meets prerequisite quizzes (is_passed = true)
   - Return 403 Forbidden if prerequisites incomplete
   - Log eligibility check for audit

5. **Check User Enrollment**
   - Query quiz_access_control: user has access to this quiz
   - If quiz restricted to specific users: verify enrollment
   - Return 403 Forbidden if not enrolled
   - Public quizzes: no enrollment check needed

6. **Check for Existing Active Attempt**
   - Query: SELECT * FROM quiz_attempts WHERE user_id = :user_id AND quiz_id = :quiz_id AND completed_at IS NULL
   - If active attempt exists: return existing attempt_id (don't create new)
   - Allow multiple completions but only one active attempt per user

7. **Check Attempt Limits**
   - Query: COUNT(*) FROM quiz_attempts WHERE user_id = :user_id AND quiz_id = :quiz_id
   - Apply attempt limit rules (if configured)
   - Return 403 Forbidden if limit exceeded

8. **Create Attempt Record**
   - INSERT INTO quiz_attempts:
     - user_id, quiz_id
     - started_at = CURRENT_TIMESTAMP
     - completed_at = NULL
     - score = NULL, is_passed = NULL
     - time_spent_seconds = NULL
   - Return attempt_id

9. **Generate Session Token**
   - Create JWT token with claims:
     - user_id, quiz_id, attempt_id
     - exp = started_at + time_limit_minutes
     - iss = "quiz-attempt"
   - Session token for tracking answer submissions

10. **Record Audit Log**
    - INSERT INTO quiz_audit_log:
      - action = "attempt_started"
      - quiz_id, user_id
      - details include timestamp, user_level

11. **Return Response**
    - Return 201 Created with attempt_id and session_token
    - Include time_limit and expiration time
    - Log attempt start for analytics

### Service Layer

```python
# app/services/quiz_service.py
from sqlalchemy import and_
from sqlalchemy.orm import Session
from datetime import datetime, timedelta
import jwt
import logging
import os

logger = logging.getLogger(__name__)

class QuizAttemptService:
    
    @staticmethod
    async def start_attempt(
        quiz_id: int,
        user_id: int,
        db: Session
    ) -> dict:
        """
        Start a new quiz attempt for user.
        
        Args:
            quiz_id: ID of quiz to attempt
            user_id: ID of user starting attempt
            db: Database session
            
        Returns:
            Dictionary with attempt details and session token
            
        Raises:
            ValueError: If quiz not found/not eligible
            PermissionError: If user unauthorized
        """
        # Fetch quiz
        quiz = db.query(Quiz).filter(
            and_(
                Quiz.id == quiz_id,
                Quiz.is_deleted == False,
                Quiz.is_published == True
            )
        ).first()
        
        if not quiz:
            raise ValueError("Quiz not found or not published")
        
        # Fetch user and verify eligibility
        user = db.query(User).filter(User.id == user_id).first()
        if not user:
            raise PermissionError("User not found")
        
        if user.current_level < quiz.minimum_level_requirement:
            raise PermissionError("User level too low for this quiz")
        
        # Check prerequisites
        prerequisites = db.query(QuizPrerequisite).filter(
            QuizPrerequisite.quiz_id == quiz_id
        ).all()
        
        for prereq in prerequisites:
            is_completed = db.query(QuizAttempt).filter(
                and_(
                    QuizAttempt.user_id == user_id,
                    QuizAttempt.quiz_id == prereq.prerequisite_quiz_id,
                    QuizAttempt.is_passed == True
                )
            ).first() is not None
            
            if not is_completed:
                raise PermissionError("Prerequisites not completed")
        
        # Check access control
        access = db.query(QuizAccessControl).filter(
            and_(
                QuizAccessControl.user_id == user_id,
                QuizAccessControl.quiz_id == quiz_id
            )
        ).first()
        
        if access is None and not user.is_admin:
            raise PermissionError("User not enrolled in quiz")
        
        # Check for existing active attempt
        active_attempt = db.query(QuizAttempt).filter(
            and_(
                QuizAttempt.user_id == user_id,
                QuizAttempt.quiz_id == quiz_id,
                QuizAttempt.completed_at == None
            )
        ).first()
        
        if active_attempt:
            logger.info(f"Returning existing active attempt {active_attempt.id}")
            # Return existing attempt instead of creating new
            attempt = active_attempt
        else:
            # Create new attempt
            now = datetime.utcnow()
            attempt = QuizAttempt(
                user_id=user_id,
                quiz_id=quiz_id,
                started_at=now,
                completed_at=None,
                score=None,
                is_passed=None
            )
            db.add(attempt)
            db.flush()  # Get attempt.id
        
        # Generate session token
        expiry_time = attempt.started_at + timedelta(minutes=quiz.time_limit_minutes)
        session_token = jwt.encode(
            {
                "user_id": user_id,
                "quiz_id": quiz_id,
                "attempt_id": attempt.id,
                "exp": expiry_time,
                "iss": "quiz-attempt"
            },
            os.getenv("SECRET_KEY", "dev-secret"),
            algorithm="HS256"
        )
        
        # Store session
        session = QuizSession(
            attempt_id=attempt.id,
            session_token=session_token,
            expires_at=expiry_time,
            is_active=True
        )
        db.add(session)
        
        # Get question count
        question_count = db.query(func.count(QuizQuestion.id)).filter(
            QuizQuestion.quiz_id == quiz_id
        ).scalar() or 0
        
        # Audit log
        audit = QuizAuditLog(
            quiz_id=quiz_id,
            user_id=user_id,
            attempt_id=attempt.id,
            action="attempt_started",
            details=f"User level: {user.current_level}"
        )
        db.add(audit)
        db.commit()
        
        logger.info(f"User {user_id} started attempt {attempt.id} for quiz {quiz_id}")
        
        return {
            "attempt_id": attempt.id,
            "quiz_id": quiz.id,
            "user_id": user_id,
            "quiz_title": quiz.title,
            "total_questions": question_count,
            "time_limit_minutes": quiz.time_limit_minutes,
            "started_at": attempt.started_at,
            "expires_at": expiry_time,
            "session_token": session_token
        }
```

### API Handler

```python
# app/api/v1/quizzes.py
from fastapi import APIRouter, Depends, HTTPException, status, Path
import logging

router = APIRouter(prefix="/quizzes", tags=["quizzes"])
logger = logging.getLogger(__name__)

@router.post("/{quiz_id}/attempts", status_code=201)
async def start_quiz_attempt(
    quiz_id: int = Path(..., ge=1),
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Start a new quiz attempt.
    
    Path Parameters:
    - quiz_id: ID of quiz to start (required)
    
    Returns:
    - 201: Attempt started with session token
    - 400: Quiz not found
    - 401: Unauthorized
    - 403: User not eligible or not enrolled
    - 404: Quiz not found
    """
    try:
        logger.info(f"User {user.id} starting quiz {quiz_id} attempt")
        
        attempt_data = await QuizAttemptService.start_attempt(
            quiz_id=quiz_id,
            user_id=user.id,
            db=db
        )
        
        return {
            "success": True,
            "data": {
                **attempt_data,
                "message": "Quiz attempt started successfully"
            }
        }
    
    except ValueError as e:
        logger.warning(f"Quiz not found: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=str(e)
        )
    
    except PermissionError as e:
        logger.warning(f"Permission denied: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to start attempt: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to start quiz attempt"
        )
```

## 📊’ Security Considerations

1. **Authentication & Authorization**
   - Valid JWT token required
   - User must meet level/prerequisite requirements
   - User must be enrolled in quiz (if applicable)
   - Return 401/403 for access violations

2. **Session Token Security**
   - Generate unique session token per attempt
   - Include expiration time (matches quiz time limit)
   - Tokens must be validated before accepting answers
   - Invalidate token after quiz completion

3. **Attempt Tracking**
   - One active attempt per user per quiz at any time
   - Return existing attempt if user starts twice
   - Prevents duplicate score submissions
   - Timestamp all activity for audit trail

4. **Time Limit Enforcement**
   - Set expires_at = started_at + time_limit_minutes
   - Validate on answer submission
   - Auto-submit when time expires (if configured)

5. **Rate Limiting**
   - Limit to 5 new attempts per minute per user
   - Prevent quiz bombing/abuse
   - Return 429 Too Many Requests if exceeded
