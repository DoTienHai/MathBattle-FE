# G02_F02_SF02: Submit Answer

✅ MVP

## 📋 Description

User submits an answer to a single quiz question. Records the answer, timestamp, validates against database (not client), records time spent, and immediately scores the answer.

## 🗏️ Business Logic (12 Steps)

1. **Validate User Authentication**
   - Extract JWT token from Authorization header
   - Verify token validity and extract user_id
   - Return 401 Unauthorized if invalid

2. **Validate Session Token**
   - Decode attempt_session_token
   - Verify user_id, quiz_id, attempt_id match
   - Check token not expired (current_time < token.exp)
   - Return 403 Forbidden if token invalid/expired
   - Check attempt_id matches active attempt

3. **Validate Path Parameters**
   - quiz_id and question_id must be positive integers
   - Verify question belongs to quiz
   - Query: SELECT * FROM quiz_questions WHERE quiz_id=:quiz_id AND question_id=:question_id
   - Return 400 Bad Request if invalid

4. **Validate Option ID**
   - Query: SELECT * FROM quiz_options WHERE question_id=:question_id AND id=:option_id
   - Verify option exists and belongs to question
   - Return 400 Bad Request if option not found

5. **Check for Duplicate Submission**
   - Query: SELECT * FROM quiz_responses WHERE attempt_id=:attempt_id AND question_id=:question_id
   - If already answered: return error (can't re-answer)
   - Prevent answer changes after submission
   - Return 409 Conflict if duplicate

6. **Check Time Limit**
   - Verify: current_time <= quiz_attempt.started_at + quiz.time_limit_minutes
   - Log if close to timeout (< 30 seconds remaining)
   - Return 403 Forbidden if time expired

7. **Check Cheating Signals**
   - Compare time_spent_seconds with typical answer time
   - Flag if < 2 seconds (suspiciously fast)
   - Flag if > time_limit (impossible if timer enforced)
   - Log to cheating_detection table
   - Return answer but mark for review

8. **Determine Correctness**
   - Query: SELECT * FROM quiz_options WHERE id=:option_id AND is_correct=true
   - is_correct = option.is_correct (from database, never client)
   - Get correct_option for feedback

9. **Record Answer**
   - INSERT INTO quiz_responses:
     - attempt_id, question_id, option_id
     - is_correct (from database)
     - time_spent_seconds
     - submitted_at = CURRENT_TIMESTAMP
     - cheating_flag (if triggered)

10. **Calculate Points**
    - points_earned = 10 if correct, 0 if incorrect (default scoring)
    - Apply time bonuses if configured (bonus for fast correct answers)
    - UPDATE quiz_responses SET points_earned = :points
    - Store for later total calculation

11. **Update Attempt Progress**
    - Calculate current_score = SUM(points_earned) from all responses
    - Count answered_questions = COUNT(*) from quiz_responses
    - Calculate questions_remaining = total_questions - answered_questions
    - Store for response progress data

12. **Return Response**
    - Return 200 OK with answer result
    - Include is_correct, points_earned
    - Include feedback (explanation from question)
    - Include progress update
    - Log answer submission for analytics

## 🗄️ Database Schema

### Tables Required/Extended

**quiz_responses**
```sql
CREATE TABLE quiz_responses (
  id SERIAL PRIMARY KEY,
  attempt_id INT NOT NULL REFERENCES quiz_attempts(id),
  question_id INT NOT NULL REFERENCES questions(id),
  option_id INT NOT NULL REFERENCES quiz_options(id),
  is_correct BOOLEAN NOT NULL,
  points_earned INT DEFAULT 0,
  time_spent_seconds INT NOT NULL,
  submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  cheating_flag BOOLEAN DEFAULT false,
  UNIQUE(attempt_id, question_id)
);
```

```sql
CREATE TABLE cheating_detection (
  id SERIAL PRIMARY KEY,
  attempt_id INT NOT NULL REFERENCES quiz_attempts(id),
  question_id INT NOT NULL REFERENCES questions(id),
  flag_type VARCHAR(50),
  details TEXT,
  severity VARCHAR(20),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Service Layer

```python
# app/services/quiz_service.py
import jwt
from sqlalchemy import and_, func
from datetime import datetime
import logging

logger = logging.getLogger(__name__)

class AnswerService:
    
    @staticmethod
    async def submit_answer(
        quiz_id: int,
        question_id: int,
        submission: AnswerSubmission,
        user_id: int,
        db: Session
    ) -> dict:
        """
        Submit answer to quiz question.
        
        Args:
            quiz_id: Quiz ID
            question_id: Question ID
            submission: Answer submission data
            user_id: User ID
            db: Database session
            
        Returns:
            Dictionary with result and progress
            
        Raises:
            ValueError: If validation fails
            PermissionError: If unauthorized
        """
        # Validate session token
        try:
            decoded = jwt.decode(
                submission.attempt_session_token,
                os.getenv("SECRET_KEY", "dev-secret"),
                algorithms=["HS256"]
            )
            if decoded["user_id"] != user_id or decoded["quiz_id"] != quiz_id:
                raise PermissionError("Token mismatch")
        except jwt.ExpiredSignatureError:
            raise PermissionError("Session expired")
        except jwt.InvalidTokenError:
            raise PermissionError("Invalid session token")
        
        attempt_id = decoded["attempt_id"]
        
        # Verify attempt is active
        attempt = db.query(QuizAttempt).filter(
            and_(
                QuizAttempt.id == attempt_id,
                QuizAttempt.user_id == user_id,
                QuizAttempt.quiz_id == quiz_id,
                QuizAttempt.completed_at == None
            )
        ).first()
        
        if not attempt:
            raise PermissionError("Invalid or completed attempt")
        
        # Check time limit
        quiz = db.query(Quiz).filter(Quiz.id == quiz_id).first()
        elapsed = (datetime.utcnow() - attempt.started_at).total_seconds()
        time_limit_seconds = quiz.time_limit_minutes * 60
        
        if elapsed > time_limit_seconds:
            raise ValueError("Quiz time limit exceeded")
        
        # Verify question belongs to quiz
        qz_q = db.query(QuizQuestion).filter(
            and_(
                QuizQuestion.quiz_id == quiz_id,
                QuizQuestion.question_id == question_id
            )
        ).first()
        
        if not qz_q:
            raise ValueError("Question not found in this quiz")
        
        # Verify option belongs to question
        option = db.query(QuizOption).filter(
            and_(
                QuizOption.question_id == question_id,
                QuizOption.id == submission.option_id
            )
        ).first()
        
        if not option:
            raise ValueError("Invalid option for this question")
        
        # Check for duplicate
        existing = db.query(QuizResponse).filter(
            and_(
                QuizResponse.attempt_id == attempt_id,
                QuizResponse.question_id == question_id
            )
        ).first()
        
        if existing:
            raise ValueError("Question already answered")
        
        # Detect cheating
        cheating_flag = False
        if submission.time_spent_seconds < 2:
            logger.warning(f"Fast answer detected: {submission.time_spent_seconds}s")
            cheating_flag = True
            cheat_log = CheatingDetection(
                attempt_id=attempt_id,
                question_id=question_id,
                flag_type="suspiciously_fast",
                details=f"Answered in {submission.time_spent_seconds}s",
                severity="low"
            )
            db.add(cheat_log)
        
        # Determine correctness (from database, not client)
        is_correct = option.is_correct
        points_earned = 10 if is_correct else 0
        
        # Apply time bonus (optional)
        if is_correct and submission.time_spent_seconds < 20:
            points_earned += 5  # Bonus for fast correct answer
        
        # Record response
        response = QuizResponse(
            attempt_id=attempt_id,
            question_id=question_id,
            option_id=submission.option_id,
            is_correct=is_correct,
            points_earned=points_earned,
            time_spent_seconds=submission.time_spent_seconds,
            submitted_at=datetime.utcnow(),
            cheating_flag=cheating_flag
        )
        db.add(response)
        db.flush()
        
        # Calculate current progress
        current_score = db.query(func.sum(QuizResponse.points_earned)).filter(
            QuizResponse.attempt_id == attempt_id
        ).scalar() or 0
        
        answered_count = db.query(func.count(QuizResponse.id)).filter(
            QuizResponse.attempt_id == attempt_id
        ).scalar() or 0
        
        total_questions = db.query(func.count(QuizQuestion.id)).filter(
            QuizQuestion.quiz_id == quiz_id
        ).scalar() or 0
        
        # Get correct option for feedback
        correct_option = db.query(QuizOption).filter(
            and_(
                QuizOption.question_id == question_id,
                QuizOption.is_correct == True
            )
        ).first()
        
        # Get question explanation
        question = db.query(Question).filter(Question.id == question_id).first()
        explanation = question.explanation if question else None
        
        db.commit()
        logger.info(f"User {user_id} answered Q{question_id} in attempt {attempt_id}")
        
        return {
            "response_id": response.id,
            "quiz_id": quiz_id,
            "question_id": question_id,
            "option_id": submission.option_id,
            "is_correct": is_correct,
            "points_earned": points_earned,
            "time_spent_seconds": submission.time_spent_seconds,
            "submitted_at": response.submitted_at,
            "feedback": {
                "correct_option_id": correct_option.id if correct_option else None,
                "explanation": explanation
            },
            "progress": {
                "answered_count": answered_count,
                "total_questions": total_questions,
                "current_score": current_score,
                "questions_remaining": total_questions - answered_count
            }
        }
```

### API Handler

```python
# app/api/v1/questions.py
@router.post("/{quiz_id}/questions/{question_id}/answers")
async def submit_answer(
    quiz_id: int = Path(..., ge=1),
    question_id: int = Path(..., ge=1),
    submission: AnswerSubmission,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Submit answer to quiz question.
    
    Returns:
    - 200: Answer submitted and scored
    - 400: Validation error
    - 401: Unauthorized
    - 403: Session expired or access denied
    - 409: Question already answered
    """
    try:
        logger.info(f"User {user.id} submitting answer to Q{question_id}")
        
        result = await AnswerService.submit_answer(
            quiz_id=quiz_id,
            question_id=question_id,
            submission=submission,
            user_id=user.id,
            db=db
        )
        
        return {
            "success": True,
            "data": result
        }
    
    except PermissionError as e:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail=str(e)
        )
    
    except ValueError as e:
        if "already answered" in str(e):
            raise HTTPException(
                status_code=status.HTTP_409_CONFLICT,
                detail=str(e)
            )
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
```

## 📊’ Security Considerations

1. **Server-Side Validation**
   - NEVER trust client's is_correct value
   - Always validate option_id against database
   - Validate option belongs to question
   - Return correct answer only AFTER submission

2. **Session Security**
   - Verify session token with each submission
   - Check token expiration (matches quiz time limit)
   - Prevent token reuse across attempts
   - Invalidate token after quiz completion

3. **Cheating Detection**
   - Flag impossibly fast answers (< 2 sec)
   - Track suspicious timing patterns
   - Log all flagged attempts for review
   - Don't immediately penalize, just flag

4. **Answer Integrity**
   - Prevent re-answering same question
   - Return 409 Conflict if duplicate
   - Validate time_spent_seconds is reasonable
   - Audit all submissions

5. **Rate Limiting**
   - Limit to 60 answers per minute per user
   - Return 429 Too Many Requests if exceeded
   - Prevents automated answer submission
