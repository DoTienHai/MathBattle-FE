# G02_F02_SF03: Get Question Feedback

✅ MVP  
**Function**: Quiz Attempt & Scoring (G02_F02)  
**Status**: Ready for Implementation  
**Priority**: High (Phase 1)  
**Difficulty**: Medium

## 📋 Description

After user submits an answer, retrieve detailed feedback including correctness, explanation, correct answer, and next question info. Provides immediate educational feedback to help user learn from mistakes.

---

## 🗏️ Business Logic (9 Steps)

1. **Validate User Authentication**
   - Extract JWT token from Authorization header
   - Verify token validity and extract user_id
   - Return 401 Unauthorized if invalid

2. **Fetch Response Record**
   - Query: SELECT * FROM quiz_responses WHERE id = :response_id
   - Return 404 Not Found if response not found

3. **Verify Access**
   - Get quiz_attempt from response.attempt_id
   - Verify quiz_attempt.user_id == user_id
   - Verify quiz_attempt.quiz_id == quiz_id
   - Return 403 Forbidden if user not owner

4. **Fetch Question & Answer Details**
   - Get submitted option details
   - Get correct option details
   - Get question text and explanation

5. **Determine Feedback Type**
   - If response.is_correct == true:
     - message = "Excellent! That's correct."
     - Include educational tips
   - If response.is_correct == false:
     - message = "Not quite. That's incorrect."
     - Provide explanation of correct answer
     - Include learning resources

6. **Compile Performance Data**
   - points_earned = response.points_earned
   - bonus_points = calculated during submission (if any)
   - time_spent_seconds = response.time_spent_seconds

7. **Get Next Question**
   - Query: SELECT * FROM quiz_questions WHERE quiz_id = :quiz_id AND order > :current_order
   - Return next_question info
   - Set has_next = true/false

8. **Format Response**
   - Structure user_answer with selected option
   - Structure correct_answer with right option
   - Include all feedback details

9. **Return Response**
   - Return 200 OK with complete feedback
   - Include next question info for navigation

### Service Layer

```python
# app/services/feedback_service.py
class FeedbackService:
    
    @staticmethod
    async def get_answer_feedback(
        quiz_id: int,
        response_id: int,
        user_id: int,
        db: Session
    ) -> dict:
        """
        Get feedback for submitted answer.
        
        Args:
            quiz_id: Quiz ID
            response_id: Response ID
            user_id: User ID
            db: Database session
            
        Returns:
            Dictionary with feedback and next question
        """
        # Fetch response
        response = db.query(QuizResponse).filter(
            QuizResponse.id == response_id
        ).first()
        
        if not response:
            raise ValueError("Response not found")
        
        # Verify access
        attempt = db.query(QuizAttempt).filter(
            QuizAttempt.id == response.attempt_id
        ).first()
        
        if attempt.user_id != user_id or attempt.quiz_id != quiz_id:
            raise PermissionError("Access denied")
        
        # Get options
        user_option = db.query(QuizOption).filter(
            QuizOption.id == response.option_id
        ).first()
        
        correct_option = db.query(QuizOption).filter(
            and_(
                QuizOption.question_id == response.question_id,
                QuizOption.is_correct == True
            )
        ).first()
        
        # Get question
        question = db.query(Question).filter(
            Question.id == response.question_id
        ).first()
        
        # Build feedback
        if response.is_correct:
            message = "Excellent! That's correct."
            tips = [
                "Keep practicing to build speed",
                "Try to solve similar problems",
                "Review related concepts"
            ]
        else:
            message = "Not quite. That's incorrect."
            tips = [
                f"The correct answer is {correct_option.option_text}",
                "Review the explanation to understand why",
                "Try similar problems to practice"
            ]
        
        # Get next question
        current_qz_q = db.query(QuizQuestion).filter(
            and_(
                QuizQuestion.quiz_id == quiz_id,
                QuizQuestion.question_id == response.question_id
            )
        ).first()
        
        next_qz_q = db.query(QuizQuestion).filter(
            and_(
                QuizQuestion.quiz_id == quiz_id,
                QuizQuestion.order > current_qz_q.order if current_qz_q else True
            )
        ).order_by(QuizQuestion.order).first()
        
        next_question_id = next_qz_q.question_id if next_qz_q else None
        next_question_number = next_qz_q.order if next_qz_q else None
        has_next = next_qz_q is not None
        
        return {
            "response_id": response.id,
            "question_id": response.question_id,
            "user_answer": {
                "option_id": user_option.id,
                "option_text": user_option.option_text,
                "is_correct": response.is_correct
            },
            "correct_answer": {
                "option_id": correct_option.id,
                "option_text": correct_option.option_text
            },
            "feedback": {
                "is_correct": response.is_correct,
                "message": message,
                "explanation": question.explanation or "No explanation available",
                "educational_tips": tips
            },
            "performance": {
                "points_earned": response.points_earned,
                "bonus_points": 0,
                "time_spent_seconds": response.time_spent_seconds
            },
            "next_question": {
                "question_id": next_question_id,
                "question_number": next_question_number,
                "has_next": has_next
            }
        }
```

### API Handler

```python
# app/api/v1/feedback.py
@router.get("/{quiz_id}/responses/{response_id}/feedback")
async def get_answer_feedback(
    quiz_id: int = Path(..., ge=1),
    response_id: int = Path(..., ge=1),
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Get feedback for submitted answer.
    
    Returns:
    - 200: Feedback with correct answer and explanation
    - 403: Access denied (not owner of response)
    - 404: Response not found
    """
    try:
        feedback = await FeedbackService.get_answer_feedback(
            quiz_id=quiz_id,
            response_id=response_id,
            user_id=user.id,
            db=db
        )
        
        return {
            "success": True,
            "data": feedback
        }
    
    except ValueError as e:
        raise HTTPException(status_code=404, detail=str(e))
    except PermissionError as e:
        raise HTTPException(status_code=403, detail=str(e))
```

## 📊’ Security Considerations

1. **Access Control**
   - Only quiz taker can view their feedback
   - Verify response ownership via attempt.user_id
   - Return 403 if unauthorized

2. **Answer Revelation**
   - Only reveal correct answer AFTER submission
   - Include explanation for learning

3. **Rate Limiting**
   - 100 feedback requests/minute per user
   - Cache for 5 minutes
