# G02_F02_SF04: Auto-Score Question

✅ MVP  
**Function**: Quiz Attempt & Scoring (G02_F02)  
**Status**: Ready for Implementation  
**Priority**: High (Phase 1)  
**Difficulty**: High

## 📋 Description

Backend calculates score for each submitted answer based on correctness, time spent, and bonus rules. Applies consistent scoring logic, tracks points progression, and updates running total throughout quiz.

---

## 🎯 Detailed Requirements

### Input Parameters

```python
# Internal service call (triggered by submit_answer)
{
  "response_id": 5001,
  "is_correct": true,
  "time_spent_seconds": 45,
  "difficulty": "easy",
  "bonus_rules": "time_bonus|accuracy_bonus"
}
```

### Output Schemas

**Scoring Result**
```json
{
  "base_points": 10,
  "time_bonus": 5,
  "difficulty_multiplier": 1.0,
  "total_points": 15,
  "scoring_breakdown": {
    "correct_answer": true,
    "base_awarded": 10,
    "fast_answer_bonus": 5,
    "difficulty_factor": 1.0,
    "penalties": 0
  }
}
```

## 🗏️ Business Logic (10 Steps)

1. **Validate Response Data**
   - Fetch quiz_response record
   - Ensure response not already scored
   - Verify attempt is active

2. **Determine Base Points**
   - Default: 10 points per question
   - If correct: award base_points
   - If incorrect: award 0 points
   - Store in response.points_earned

3. **Apply Time Bonus**
   - If is_correct AND time_spent < 20 seconds:
     - bonus = 5 points
   - If is_correct AND time_spent < 10 seconds:
     - bonus = 10 points (premium speed bonus)
   - Log time for analytics

4. **Apply Difficulty Multiplier**
   - easy: 1.0x multiplier
   - medium: 1.5x multiplier
   - hard: 2.0x multiplier
   - Multiply total = (base + bonuses) * multiplier

5. **Check Penalty Rules**
   - If submitted too close to time limit:
     - -5 point penalty (rushed attempt)
   - If answer flagged for cheating:
     - Set to review (don't penalize immediately)
   - Log all penalties

6. **Calculate Attempt-Level Stats**
   - Current attempt score = SUM(points_earned) from all responses
   - Accuracy = COUNT(correct) / COUNT(total) * 100%
   - Average time per question

7. **Update Running Totals**
   - UPDATE quiz_attempts SET:
     - score = updated_total
     - time_spent_seconds = sum of all responses
   - Update progress cache

8. **Store Scoring Details**
   - INSERT INTO scoring_log:
     - response_id, base_points, bonuses, multiplier
     - timestamp, scoring_method

9. **Check Attempt Completion**
   - If all questions answered:
     - Calculate final score
     - Determine is_passed (score >= passing_score)
     - Set completed_at timestamp

10. **Return Scoring Result**
    - Return complete breakdown for analytics
    - Include final attempt score

## 🗄️ Database Schema

### Tables Required

**scoring_log**
```sql
CREATE TABLE scoring_log (
  id SERIAL PRIMARY KEY,
  response_id INT NOT NULL REFERENCES quiz_responses(id),
  base_points INT NOT NULL,
  time_bonus INT DEFAULT 0,
  penalty INT DEFAULT 0,
  difficulty_multiplier DECIMAL(3,2) DEFAULT 1.0,
  total_points INT NOT NULL,
  scoring_method VARCHAR(50) DEFAULT 'standard',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Indexes
```sql
CREATE INDEX idx_scoring_log_response_id ON scoring_log(response_id);
CREATE INDEX idx_scoring_log_created_at ON scoring_log(created_at);
```

## 💻 Backend Implementation

### Service Layer

```python
# app/services/scoring_service.py
from sqlalchemy import func
import logging

logger = logging.getLogger(__name__)

class ScoringService:
    
    DIFFICULTY_MULTIPLIER = {
        "easy": 1.0,
        "medium": 1.5,
        "hard": 2.0
    }
    
    BASE_POINTS = 10
    SPEED_BONUS_THRESHOLD_FAST = 10  # < 10 sec = 10 bonus
    SPEED_BONUS_THRESHOLD_NORMAL = 20  # < 20 sec = 5 bonus
    
    @staticmethod
    async def calculate_score(
        response_id: int,
        db: Session
    ) -> dict:
        """
        Calculate score for submitted answer.
        
        Args:
            response_id: ID of quiz response
            db: Database session
            
        Returns:
            Dictionary with scoring breakdown
        """
        # Fetch response
        response = db.query(QuizResponse).filter(
            QuizResponse.id == response_id
        ).first()
        
        if not response:
            raise ValueError("Response not found")
        
        # Fetch attempt and quiz
        attempt = db.query(QuizAttempt).filter(
            QuizAttempt.id == response.attempt_id
        ).first()
        
        quiz = db.query(Quiz).filter(
            Quiz.id == attempt.quiz_id
        ).first()
        
        # Fetch question for difficulty
        question = db.query(Question).filter(
            Question.id == response.question_id
        ).first()
        
        # Calculate base points
        base_points = ScoringService.BASE_POINTS if response.is_correct else 0
        
        # Calculate time bonus
        time_bonus = 0
        if response.is_correct:
            if response.time_spent_seconds < ScoringService.SPEED_BONUS_THRESHOLD_FAST:
                time_bonus = 10
            elif response.time_spent_seconds < ScoringService.SPEED_BONUS_THRESHOLD_NORMAL:
                time_bonus = 5
        
        # Get difficulty multiplier
        difficulty = question.difficulty or "medium"
        multiplier = ScoringService.DIFFICULTY_MULTIPLIER.get(difficulty, 1.0)
        
        # Apply multiplier to subtotal
        subtotal = base_points + time_bonus
        final_points = int(subtotal * multiplier)
        
        # Check for penalties
        penalty = 0
        time_limit_seconds = quiz.time_limit_minutes * 60
        elapsed = (datetime.utcnow() - attempt.started_at).total_seconds()
        time_remaining = time_limit_seconds - elapsed
        
        if time_remaining < 30 and response.is_correct:
            penalty = 5  # Rushed attempt penalty
        
        final_points = max(0, final_points - penalty)  # Ensure non-negative
        
        # Store response points
        response.points_earned = final_points
        db.add(response)
        db.flush()
        
        # Store scoring log
        scoring_log = ScoringLog(
            response_id=response.id,
            base_points=base_points,
            time_bonus=time_bonus,
            penalty=penalty,
            difficulty_multiplier=multiplier,
            total_points=final_points,
            scoring_method="standard"
        )
        db.add(scoring_log)
        db.flush()
        
        # Update attempt-level totals
        total_score = db.query(func.sum(QuizResponse.points_earned)).filter(
            QuizResponse.attempt_id == attempt.id
        ).scalar() or 0
        
        # Get all responses for attempt
        all_responses = db.query(QuizResponse).filter(
            QuizResponse.attempt_id == attempt.id
        ).all()
        
        # Calculate attempt stats
        total_time = sum([r.time_spent_seconds for r in all_responses])
        total_questions = db.query(func.count(QuizQuestion.id)).filter(
            QuizQuestion.quiz_id == attempt.quiz_id
        ).scalar() or 0
        
        answered_count = len(all_responses)
        correct_count = sum([1 for r in all_responses if r.is_correct])
        accuracy = (correct_count / answered_count * 100) if answered_count > 0 else 0
        
        # Update attempt record
        attempt.score = total_score
        attempt.time_spent_seconds = total_time
        
        # Check if attempt is complete
        if answered_count == total_questions:
            attempt.completed_at = datetime.utcnow()
            attempt.is_passed = total_score >= quiz.passing_score
            
            logger.info(f"Attempt {attempt.id} completed: score={total_score}, passed={attempt.is_passed}")
        
        db.commit()
        
        logger.info(f"Response {response_id} scored: {final_points} points")
        
        return {
            "base_points": base_points,
            "time_bonus": time_bonus,
            "difficulty_multiplier": multiplier,
            "total_points": final_points,
            "penalty": penalty,
            "scoring_breakdown": {
                "correct_answer": response.is_correct,
                "base_awarded": base_points,
                "fast_answer_bonus": time_bonus,
                "difficulty_factor": multiplier,
                "penalties": penalty,
                "final_score": final_points
            },
            "attempt_progress": {
                "current_attempt_score": total_score,
                "questions_answered": answered_count,
                "total_questions": total_questions,
                "accuracy": round(accuracy, 2),
                "is_complete": answered_count == total_questions
            }
        }
```

## 📊’ Security Considerations

1. **Server-Side Only**
   - All scoring calculations on backend
   - Never allow client to specify points
   - Validate all inputs before scoring

2. **Tamper Prevention**
   - Validate response exists and belongs to user
   - Check attempt is active
   - Prevent double-scoring

3. **Audit Trail**
   - Log all scoring calculations
   - Track scoring method used
   - Enable score review/appeal

4. **Consistency**
   - Use consistent scoring rules for all users
   - Document multipliers and bonuses
   - Make rules transparent
