# G02_F02_SF05: Save Answer Progress

✅ MVP  
**Function**: Quiz Attempt & Scoring (G02_F02)  
**Status**: Ready for Implementation  
**Priority**: High (Phase 1)  
**Difficulty**: Medium  

---

## 📋 Description

Auto-save quiz progress while user is taking quiz. Periodically saves current state, answers submitted, and running score to database. Enables resuming interrupted quizzes without losing work.

---

## 🗏️ Business Logic (9 Steps)

1. **Validate Session Token**
   - Decode attempt_session_token
   - Verify user_id, quiz_id, attempt_id match
   - Check token not expired
   - Return 403 if invalid

2. **Validate Attempt Status**
   - Fetch quiz_attempt record
   - Verify completed_at IS NULL (attempt in progress)
   - Return 400 if already completed

3. **Update Last Activity**
   - UPDATE quiz_sessions SET last_activity_at = CURRENT_TIMESTAMP
   - Extend session timeout if needed

4. **Validate Answer Data**
   - For each answer in current_progress:
     - Verify question_id belongs to quiz
     - Verify option_id valid (if provided)
     - Skip invalid entries

5. **Update Running Score**
   - Calculate current_score from all submitted responses
   - UPDATE quiz_attempts SET score = :current_score
   - Ensure score only increases (no decreases)

6. **Create Progress Checkpoint**
   - INSERT INTO quiz_progress_checkpoints:
     - attempt_id, quiz_id
     - answers_saved = COUNT(submitted answers)
     - current_score, time_elapsed_seconds
     - saved_at = CURRENT_TIMESTAMP

7. **Update Session Metadata**
   - Store checkpoint_id in session
   - Update progress indicators
   - Log save event

8. **Prevent Duplicate Saves**
   - If save happens within 5 seconds of last save:
     - Merge with last checkpoint (don't create new)
     - Update timestamp

9. **Return Progress Confirmation**
   - Return checkpoint_id for verification
   - Include saved metrics for display
   - Log save success

### Service Layer

```python
# app/services/progress_service.py
import logging
from datetime import datetime

logger = logging.getLogger(__name__)

class ProgressService:
    
    @staticmethod
    async def save_progress(
        user_id: int,
        progress_data: ProgressSaveRequest,
        db: Session
    ) -> dict:
        """
        Save quiz progress checkpoint.
        
        Args:
            user_id: User ID
            progress_data: Current progress data
            db: Database session
            
        Returns:
            Dictionary with save confirmation
        """
        attempt_id = progress_data.attempt_id
        quiz_id = progress_data.quiz_id
        
        # Fetch and validate attempt
        attempt = db.query(QuizAttempt).filter(
            and_(
                QuizAttempt.id == attempt_id,
                QuizAttempt.user_id == user_id,
                QuizAttempt.quiz_id == quiz_id,
                QuizAttempt.completed_at == None
            )
        ).first()
        
        if not attempt:
            raise PermissionError("Invalid or completed attempt")
        
        # Check for recent duplicate save
        recent_checkpoint = db.query(QuizProgressCheckpoint).filter(
            and_(
                QuizProgressCheckpoint.attempt_id == attempt_id,
                QuizProgressCheckpoint.saved_at >= datetime.utcnow() - timedelta(seconds=5)
            )
        ).order_by(QuizProgressCheckpoint.saved_at.desc()).first()
        
        # Calculate elapsed time
        elapsed = (datetime.utcnow() - attempt.started_at).total_seconds()
        
        # Validate and count submitted answers
        answers_saved = 0
        for answer in progress_data.current_progress.answered_questions:
            if answer.submitted:
                # Verify question belongs to quiz
                qz_q = db.query(QuizQuestion).filter(
                    and_(
                        QuizQuestion.quiz_id == quiz_id,
                        QuizQuestion.question_id == answer.question_id
                    )
                ).first()
                
                if qz_q:
                    answers_saved += 1
        
        # Calculate current score from database
        current_score = db.query(func.sum(QuizResponse.points_earned)).filter(
            QuizResponse.attempt_id == attempt_id
        ).scalar() or 0
        
        # Update attempt
        attempt.score = current_score
        attempt.time_spent_seconds = int(elapsed)
        
        # Create checkpoint (unless too recent)
        if not recent_checkpoint or (datetime.utcnow() - recent_checkpoint.saved_at).total_seconds() > 5:
            checkpoint = QuizProgressCheckpoint(
                attempt_id=attempt_id,
                quiz_id=quiz_id,
                answers_saved=answers_saved,
                current_score=current_score,
                time_elapsed_seconds=int(elapsed),
                saved_at=datetime.utcnow()
            )
            db.add(checkpoint)
            db.flush()
            checkpoint_id = checkpoint.id
        else:
            checkpoint_id = recent_checkpoint.id
        
        # Update session last activity
        session = db.query(QuizSession).filter(
            QuizSession.attempt_id == attempt_id
        ).first()
        
        if session:
            session.last_activity_at = datetime.utcnow()
            session.last_checkpoint_id = checkpoint_id
        
        db.commit()
        logger.info(f"Progress saved for attempt {attempt_id}: {answers_saved} answers, score={current_score}")
        
        return {
            "progress_saved": True,
            "checkpoint_id": checkpoint_id,
            "saved_at": datetime.utcnow(),
            "answers_saved": answers_saved,
            "current_score": current_score,
            "time_elapsed_seconds": int(elapsed)
        }
```

### API Handler

```python
# app/api/v1/progress.py
@router.post("/attempts/progress/save")
async def save_quiz_progress(
    progress_data: ProgressSaveRequest,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Save quiz progress checkpoint.
    
    Returns:
    - 200: Progress saved successfully
    - 400: Invalid attempt
    - 401: Unauthorized
    - 403: Session invalid or expired
    """
    try:
        logger.info(f"Saving progress for attempt {progress_data.attempt_id}")
        
        result = await ProgressService.save_progress(
            user_id=user.id,
            progress_data=progress_data,
            db=db
        )
        
        return {
            "success": True,
            "data": {
                **result,
                "message": "Progress saved successfully"
            }
        }
    
    except PermissionError as e:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to save progress: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to save progress"
        )
```

## 📊’ Security Considerations

1. **Session Validation**
   - Verify attempt is active
   - Validate session token
   - Return 403 if invalid

2. **User Ownership**
   - Verify user owns attempt
   - Check user_id matches
   - Prevent cross-user data access

3. **Data Integrity**
   - Validate all answer data before saving
   - Score only calculated server-side
   - Prevent score manipulation

4. **Rate Limiting**
   - Limit saves to 1 per 5 seconds (prevents spam)
   - Return 429 if exceeded
   - Merge rapid saves into single checkpoint
