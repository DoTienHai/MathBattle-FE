## 📋 Description

User submits completed quiz, finalize score, determine pass/fail status, award points/badges, and update leaderboard. Marks attempt as completed and triggers post-quiz workflows (notifications, achievements, etc.).

## 🗏️ Business Logic (14 Steps)

1. **Validate User Authentication**
   - Extract JWT token from Authorization header
   - Verify token validity and extract user_id
   - Return 401 Unauthorized if invalid

2. **Validate Session Token**
   - Decode attempt_session_token
   - Verify user_id, attempt_id match
   - Check token not expired
   - Return 403 if invalid

3. **Fetch Attempt Record**
   - Query: SELECT * FROM quiz_attempts WHERE id = :attempt_id
   - Verify completed_at IS NULL (attempt in progress)
   - Return 400 if already completed
   - Return 404 if not found

4. **Check Quiz Completion**
   - Count total questions in quiz
   - Count submitted responses
   - If submitted < total:
     - Return 400 Bad Request with remaining count
     - Don't allow partial submissions
   - Log incomplete submission attempt

5. **Calculate Final Score & Stats**
   - final_score = SUM(points_earned) from all responses
   - accuracy = COUNT(correct) / COUNT(total) * 100%
   - total_time = SUM(time_spent_seconds)
   - Pass/fail: is_passed = final_score >= quiz.passing_score

6. **Award Points to User**
   - points_awarded = quiz.points_reward if is_passed else 0
   - UPDATE users SET total_points = total_points + :points_awarded
   - Log point award in transaction

7. **Check for Achievements/Badges**
   - Speed achievement: time_spent < 5 minutes â†’ +10 points
   - Perfect score: accuracy == 100% â†’ +50 points
   - Streak: 3 consecutive passed quizzes â†’ badge
   - Level up: Check if user progressed to new level
   - INSERT into user_achievements for each unlocked

8. **Update Leaderboard**
   - Calculate current rank for this quiz
   - Query: SELECT COUNT(*) FROM quiz_attempts WHERE quiz_id=:quiz_id AND score>:final_score AND is_passed=true
   - leaderboard_rank = COUNT + 1
   - Upsert into leaderboard_entry

9. **Update User Progress**
   - UPDATE users SET:
     - quizzes_completed = quizzes_completed + 1
     - quizzes_passed = (if is_passed)
     - current_level = (recalculate based on total xp)

10. **Mark Attempt Complete**
    - UPDATE quiz_attempts SET:
      - completed_at = CURRENT_TIMESTAMP
      - is_passed = :is_passed
      - score = :final_score
      - time_spent_seconds = :total_time

11. **Get Recommendations**
    - If passed: recommend next difficulty quiz
    - If failed: recommend similar quiz to retry
    - Query based on user level and progress

12. **Invalidate Session**
    - UPDATE quiz_sessions SET is_active = false
    - Delete session token to prevent reuse
    - Prevent resuming completed quiz

13. **Send Notifications**
    - Queue notification for user (achievement unlock, passing)
    - Queue notification for leaderboard position
    - Update real-time dashboard if applicable

14. **Return Completion Response**
    - Return 200 OK with final results
    - Include achievements, leaderboard position
    - Include next quiz recommendation

### Service Layer

```python
# app/services/submission_service.py
import logging
from datetime import datetime

logger = logging.getLogger(__name__)

class SubmissionService:
    
    @staticmethod
    async def submit_quiz_attempt(
        attempt_id: int,
        user_id: int,
        db: Session
    ) -> dict:
        """
        Submit completed quiz attempt.
        
        Args:
            attempt_id: ID of attempt to submit
            user_id: User ID
            db: Database session
            
        Returns:
            Dictionary with final results
        """
        # Fetch attempt
        attempt = db.query(QuizAttempt).filter(
            and_(
                QuizAttempt.id == attempt_id,
                QuizAttempt.user_id == user_id,
                QuizAttempt.completed_at == None
            )
        ).first()
        
        if not attempt:
            raise ValueError("Attempt not found or already completed")
        
        # Fetch quiz
        quiz = db.query(Quiz).filter(Quiz.id == attempt.quiz_id).first()
        
        # Check completion
        total_questions = db.query(func.count(QuizQuestion.id)).filter(
            QuizQuestion.quiz_id == attempt.quiz_id
        ).scalar() or 0
        
        submitted_responses = db.query(func.count(QuizResponse.id)).filter(
            QuizResponse.attempt_id == attempt_id
        ).scalar() or 0
        
        if submitted_responses < total_questions:
            raise ValueError(f"Incomplete quiz ({total_questions - submitted_responses} questions remaining)")
        
        # Calculate stats
        all_responses = db.query(QuizResponse).filter(
            QuizResponse.attempt_id == attempt_id
        ).all()
        
        final_score = sum([r.points_earned for r in all_responses])
        correct_count = sum([1 for r in all_responses if r.is_correct])
        accuracy = (correct_count / submitted_responses * 100) if submitted_responses > 0 else 0
        total_time = sum([r.time_spent_seconds for r in all_responses])
        is_passed = final_score >= quiz.passing_score
        
        # Award points
        points_awarded = quiz.points_reward if is_passed else 0
        user = db.query(User).filter(User.id == user_id).first()
        user.total_points += points_awarded
        user.quizzes_completed += 1
        if is_passed:
            user.quizzes_passed += 1
        
        # Check achievements
        achievements_unlocked = []
        
        # Speed achievement
        if total_time < 300:  # < 5 minutes
            achievement = db.query(QuizAchievement).filter(
                QuizAchievement.id == "speed_demon"
            ).first()
            if achievement:
                ach = UserAchievement(
                    user_id=user_id,
                    achievement_id="speed_demon",
                    achievement_name=achievement.name,
                    points=achievement.points_reward
                )
                db.add(ach)
                user.total_points += achievement.points_reward
                achievements_unlocked.append({
                    "achievement_id": "speed_demon",
                    "achievement_name": achievement.name,
                    "points": achievement.points_reward
                })
        
        # Perfect score
        if accuracy == 100.0:
            achievement = db.query(QuizAchievement).filter(
                QuizAchievement.id == "perfect_score"
            ).first()
            if achievement:
                ach = UserAchievement(
                    user_id=user_id,
                    achievement_id="perfect_score",
                    achievement_name=achievement.name,
                    points=achievement.points_reward
                )
                db.add(ach)
                user.total_points += achievement.points_reward
                achievements_unlocked.append({
                    "achievement_id": "perfect_score",
                    "achievement_name": achievement.name,
                    "points": achievement.points_reward
                })
        
        # Complete attempt
        attempt.completed_at = datetime.utcnow()
        attempt.is_passed = is_passed
        attempt.score = final_score
        attempt.time_spent_seconds = total_time
        
        # Update leaderboard
        leaderboard_rank = db.query(func.count(QuizAttempt.id)).filter(
            and_(
                QuizAttempt.quiz_id == attempt.quiz_id,
                QuizAttempt.score > final_score,
                QuizAttempt.is_passed == True
            )
        ).scalar() or 0
        leaderboard_rank += 1
        
        leaderboard = LeaderboardEntry(
            quiz_id=attempt.quiz_id,
            user_id=user_id,
            rank=leaderboard_rank,
            score=final_score,
            time_spent_seconds=total_time
        )
        db.add(leaderboard)
        
        # Invalidate session
        session = db.query(QuizSession).filter(
            QuizSession.attempt_id == attempt_id
        ).first()
        if session:
            session.is_active = False
        
        # Get recommendation
        next_quiz = None
        if is_passed:
            # Recommend harder quiz
            harder_quizzes = db.query(Quiz).filter(
                and_(
                    Quiz.id != attempt.quiz_id,
                    Quiz.is_published == True,
                    Quiz.difficulty != quiz.difficulty
                )
            ).limit(1).all()
            if harder_quizzes:
                next_quiz = {
                    "quiz_id": harder_quizzes[0].id,
                    "title": harder_quizzes[0].title,
                    "difficulty": harder_quizzes[0].difficulty
                }
        
        db.commit()
        logger.info(f"Quiz attempt {attempt_id} submitted: score={final_score}, passed={is_passed}")
        
        return {
            "attempt_id": attempt.id,
            "quiz_id": attempt.quiz_id,
            "final_score": final_score,
            "passing_score": quiz.passing_score,
            "is_passed": is_passed,
            "accuracy": round(accuracy, 2),
            "points_awarded": points_awarded,
            "time_spent_seconds": total_time,
            "completed_at": attempt.completed_at,
            "leaderboard_rank": leaderboard_rank,
            "achievements_unlocked": achievements_unlocked,
            "next_quiz_recommended": next_quiz,
            "message": f"{'Great job!' if is_passed else 'Try again!'} Quiz submitted successfully!"
        }
```

### API Handler

```python
# app/api/v1/submissions.py
@router.post("/attempts/{attempt_id}/submit", status_code=200)
async def submit_quiz_attempt(
    attempt_id: int = Path(..., ge=1),
    submission: QuizSubmissionRequest,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Submit completed quiz attempt.
    
    Returns:
    - 200: Submission successful with results
    - 400: Incomplete quiz or validation error
    - 401: Unauthorized
    - 404: Attempt not found
    """
    try:
        logger.info(f"User {user.id} submitting attempt {attempt_id}")
        
        result = await SubmissionService.submit_quiz_attempt(
            attempt_id=attempt_id,
            user_id=user.id,
            db=db
        )
        
        return {
            "success": True,
            "data": result
        }
    
    except ValueError as e:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to submit: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to submit quiz"
        )
```

## 📊’ Security Considerations

1. **Completion Verification**
   - Ensure all questions answered before submission
   - Validate from database, not client
   - Prevent partial submissions

2. **Score Integrity**
   - Calculate final score server-side
   - Never accept client-provided score
   - Validate against stored responses

3. **Achievement Validation**
   - Verify achievement conditions met before awarding
   - Check user eligibility
   - Prevent double-awarding

4. **Session Termination**
   - Invalidate session immediately after submission
   - Prevent resuming submitted quiz
   - Clear sensitive data

5. **Audit Logging**
   - Log all submissions with timestamp
   - Track score calculation details
   - Enable fraud detection/review
