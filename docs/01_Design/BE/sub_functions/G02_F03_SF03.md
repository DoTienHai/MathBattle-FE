# G02_F03_SF03: Update Quiz Category

✅ MVP | **Admin Only** | **Difficulty**: Medium

## 📋 Description
Update quiz category metadata (name, description, icon_url, color_code). Admin-only endpoint.

## 🎯 API Specification

**Endpoint**: `PUT /api/v1/quiz-categories/{category_id}`

**Input (Request Body)**:
```json
{
  "name": "Geometry & Trigonometry",
  "description": "Updated description",
  "icon_url": "https://cdn.example.com/geom.png",
  "color_code": "#FF00FF"
}
```

**Output (200 OK)**:
```json
{
  "success": true,
  "data": {
    "category_id": 5,
    "name": "Geometry & Trigonometry",
    "description": "Updated description",
    "icon_url": "https://cdn.example.com/geom.png",
    "color_code": "#FF00FF",
    "updated_at": "2026-05-04T10:30:00Z"
  }
}
```

## 🗏️ Business Logic (9 Steps)

1. **Validate Authentication**: Verify JWT token
2. **Check Admin Role**: Verify user.role='admin', return 403 if not
3. **Fetch Category**: Query `SELECT * FROM quiz_categories WHERE id=:category_id AND is_deleted=false`
4. **Return 404 if Not Found**: Category doesn't exist or is deleted
5. **Validate Input**: Pydantic schema validation on all fields
6. **Check Name Uniqueness**: If name changed, ensure new name not in use (case-insensitive), return 409 if duplicate
7. **Update Fields**: Only update provided fields (partial update support)
8. **Persist Changes**: `UPDATE quiz_categories SET ... WHERE id=:category_id`
9. **Invalidate Cache**: Delete `categories:all` from Redis

## 💻 Implementation

### Service Layer

```python
async def update_category(
    db: AsyncSession,
    category_id: int,
    category_data: CategoryUpdate,
    user_id: int
) -> dict:
    """Update existing quiz category"""
    # Fetch existing
    category = await db.get(QuizCategory, category_id)
    if not category or category.is_deleted:
        raise HTTPException(status_code=404, detail="Not found")
    
    # Check uniqueness if name changed
    if category_data.name and category_data.name != category.name:
        existing = await db.execute(
            select(QuizCategory).where(
                QuizCategory.name.ilike(category_data.name)
            )
        )
        if existing.scalar():
            raise HTTPException(status_code=409, detail="Name exists")
    
    # Update fields
    update_dict = category_data.dict(exclude_unset=True)
    for key, value in update_dict.items():
        if value is not None:
            setattr(category, key, value)
    
    await db.commit()
    return {**category.__dict__, "updated_at": category.updated_at}
```

### API Handler

```python
@router.put("/{category_id}", status_code=status.HTTP_200_OK)
async def update_category(
    category_id: int = Path(..., gt=0),
    category_data: CategoryUpdate = Body(...),
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db)
):
    """Update quiz category (admin only)"""
    if current_user.role != "admin":
        raise HTTPException(status_code=403)
    return await QuizCategoryService.update_category(db, category_id, category_data, current_user.id)
```
