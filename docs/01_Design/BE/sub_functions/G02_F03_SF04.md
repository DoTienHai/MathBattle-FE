# G02_F03_SF04: Delete Quiz Category

✅ MVP | **Admin Only** | **Soft Delete** | **Difficulty**: Easy

## 📋 Description
Soft delete quiz category. Sets is_deleted=true. Prevents deletion if category has associated quizzes.

## 🎯 API Specification

**Endpoint**: `DELETE /api/v1/quiz-categories/{category_id}`

**Output (204 No Content or 200 OK)**:
```json
{
  "success": true,
  "message": "Category deleted successfully"
}
```

## 🗏️ Business Logic (8 Steps)

1. **Validate Authentication** & **Check Admin Role**
2. **Fetch Category**: WHERE id=category_id AND is_deleted=false
3. **Return 404 if Not Found**
4. **Check for Associated Quizzes**: `SELECT COUNT(*) FROM quizzes WHERE category_id=:id AND is_deleted=false`
5. **Return 409 if Quizzes Exist**: "Cannot delete category with associated quizzes"
6. **Soft Delete**: `UPDATE quiz_categories SET is_deleted=true, updated_at=NOW() WHERE id=:id`
7. **Invalidate Cache**: Delete Redis key `categories:all`
8. **Audit Log**: Log deletion with admin_user_id, category_id, timestamp

```sql
-- Soft delete
UPDATE quiz_categories SET is_deleted=true, updated_at=NOW() WHERE id=:id;

-- Cannot hard delete due to referential integrity
```

## 💻 Implementation

```python
# Service
async def delete_category(db: AsyncSession, category_id: int, user_id: int):
    category = await db.get(QuizCategory, category_id)
    if not category or category.is_deleted:
        raise HTTPException(status_code=404)
    
    # Check for quizzes
    quiz_count = await db.execute(
        select(func.count(Quiz.id)).where(
            and_(Quiz.category_id == category_id, Quiz.is_deleted == False)
        )
    )
    if quiz_count.scalar() > 0:
        raise HTTPException(status_code=409, detail="Category has associated quizzes")
    
    category.is_deleted = True
    await db.commit()
    logger.info(f"Category {category_id} deleted by user {user_id}")

# API Handler
@router.delete("/{category_id}", status_code=status.HTTP_204_NO_CONTENT)
async def delete_category(
    category_id: int = Path(..., gt=0),
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db)
):
    if current_user.role != "admin":
        raise HTTPException(status_code=403)
    await QuizCategoryService.delete_category(db, category_id, current_user.id)
```
