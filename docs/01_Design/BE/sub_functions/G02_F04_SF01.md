# G02_F04_SF01: Get Quiz Leaderboard

✅ MVP | **Top 100 Users** | **All-Time & Monthly** | **Difficulty**: Medium

## 📋 Description
Retrieve top 100 users by quiz scores. Supports all-time and monthly leaderboards. Includes current user's rank.

**Endpoint**: `GET /api/v1/quizzes/leaderboard`

**Query Parameters**:

## 🗏️ Business Logic (10 Steps)

1. Validate JWT token
2. Validate time_period (all_time | month), validate limit [1-100]
3. Build base query: `SELECT user_id, SUM(score) as total_score, COUNT(*) as quiz_count FROM quiz_attempts WHERE is_passed=true`
4. If time_period="month": Add WHERE `completed_at >= DATE_TRUNC('month', NOW())`
5. Group by user_id, ORDER BY total_score DESC
6. Apply offset/limit for pagination
7. Fetch user details (username) via JOIN to users table
8. Calculate current user's rank: count records with higher score
9. Cache result (5 minutes, key: `leaderboard:{time_period}`)
10. Return leaderboard + user's rank/percentile

## 🗄️ Database Schema

```sql
SELECT 
  ROW_NUMBER() OVER (ORDER BY SUM(qa.score) DESC) as rank,
  qa.user_id,
  u.username,
  SUM(qa.score) as total_score,
  COUNT(*) as quiz_count,
  AVG(qa.score) as average_score,
  MAX(qa.completed_at) as last_attempt
FROM quiz_attempts qa
JOIN users u ON qa.user_id = u.id
WHERE qa.is_passed = true
  AND qa.completed_at >= CASE 
    WHEN :time_period = 'month' THEN DATE_TRUNC('month', NOW())
    ELSE '1900-01-01'::timestamp
  END
GROUP BY qa.user_id, u.username
ORDER BY total_score DESC
LIMIT :limit OFFSET :offset;
```

## 💻 Implementation

```python
# Service
async def get_leaderboard(
    db: AsyncSession, user_id: int, time_period: str, limit: int, offset: int
):
    # Build query
    query = select(
        func.row_number().over(
            order_by(func.sum(QuizAttempt.score).desc())
        ).label('rank'),
        QuizAttempt.user_id,
        User.username,
        func.sum(QuizAttempt.score).label('total_score'),
        func.count(QuizAttempt.id).label('quiz_count'),
        func.avg(QuizAttempt.score).label('average_score'),
        func.max(QuizAttempt.completed_at).label('last_attempt')
    ).select_from(QuizAttempt).join(User).where(
        QuizAttempt.is_passed == True
    )
    
    if time_period == "month":
        query = query.where(
            QuizAttempt.completed_at >= func.date_trunc('month', func.now())
        )
    
    query = query.group_by(QuizAttempt.user_id, User.username)
    query = query.order_by(func.sum(QuizAttempt.score).desc())
    
    # Get leaderboard
    result = await db.execute(query.limit(limit).offset(offset))
    leaderboard = result.fetchall()
    
    # Get user's rank
    user_rank_result = await db.execute(
        select(func.count()).select_from(QuizAttempt).where(
            and_(
                QuizAttempt.is_passed == True,
                func.sum(QuizAttempt.score) > (
                    select(func.sum(QuizAttempt.score)).where(
                        QuizAttempt.user_id == user_id
                    ).scalar_subquery()
                )
            )
        )
    )
    user_rank = user_rank_result.scalar() + 1
    
    return {
        "leaderboard": [dict(row) for row in leaderboard],
        "user_rank": {"rank": user_rank},
        "pagination": {"total": len(leaderboard), "offset": offset, "limit": limit}
    }

# API Handler

    return {"success": True, "data": result}
```
