# G02_F04_SF02: Get User Quiz History

✅ MVP | **Pagination** | **Filters** | **Difficulty**: Medium

## 📋 Description
Retrieve all quizzes attempted by user with scores, dates, and pass/fail status. Paginated with optional filtering.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/quiz-history`

**Query Parameters**:
- `page`: 1-9999 (default: 1)
- `page_size`: 1-100 (default: 10)
- `status`: "passed" | "failed" | "all" (default: all)
- `sort_by`: "date" | "score" (default: date)

**Output (200 OK)**:
```json
{
  "success": true,
  "data": {
    "history": [
      {
        "attempt_id": 456,

      "total_pages": 5

  }
```

## 🗏️ Business Logic (9 Steps)

1. Validate JWT + extract user_id
2. Verify target user_id matches current user (or admin can view any user)
3. Validate pagination (page â‰¥ 1, page_size âˆˆ [1,100])
4. Build query: `SELECT ... FROM quiz_attempts WHERE user_id = :user_id`
5. Apply status filter: if status="passed" add `WHERE is_passed=true`
6. Apply sorting: order by `completed_at DESC` or `score DESC`
7. Paginate: offset = (page-1)*page_size, limit = page_size
8. JOIN quiz_attempts to quizzes for title + passing_score
9. Return history + pagination metadata

## 🗄️ Database

```sql
SELECT 
  qa.id as attempt_id,
  qa.quiz_id,
  q.title as quiz_title,
  qa.score,
  qa.is_passed,
  qa.time_spent_seconds,
  qa.completed_at,
  q.passing_score
FROM quiz_attempts qa
JOIN quizzes q ON qa.quiz_id = q.id
WHERE qa.user_id = :user_id
  AND (qa.is_passed = :status OR :status IS NULL)
ORDER BY qa.completed_at DESC
LIMIT :limit OFFSET :offset;
```

## 💻 Implementation

```python
async def get_user_history(
    db: AsyncSession, user_id: int, page: int, page_size: int, status: str
):
    query = select(
        QuizAttempt.id,
        QuizAttempt.quiz_id,
        Quiz.title,
        QuizAttempt.score,
        QuizAttempt.is_passed,
        QuizAttempt.time_spent_seconds,
        QuizAttempt.completed_at,
        Quiz.passing_score
    ).join(Quiz).where(QuizAttempt.user_id == user_id)
    
    if status != "all":
        query = query.where(QuizAttempt.is_passed == (status == "passed"))
    
    query = query.order_by(QuizAttempt.completed_at.desc())
    
    # Get total count
    count_query = select(func.count()).select_from(QuizAttempt).where(QuizAttempt.user_id == user_id)
    if status != "all":
        count_query = count_query.where(QuizAttempt.is_passed == (status == "passed"))
    total = (await db.execute(count_query)).scalar()
    
    # Get paginated results
    offset = (page - 1) * page_size
    result = await db.execute(query.limit(page_size).offset(offset))
    history = [dict(row) for row in result]
    
    return {
        "history": history,
        "pagination": {
            "page": page,
            "page_size": page_size,
            "total": total,
            "total_pages": (total + page_size - 1) // page_size
        }
    }
```
