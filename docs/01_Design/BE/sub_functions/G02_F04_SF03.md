# G02_F04_SF03: Get User Performance Stats

✅ MVP | **Analytics** | **Trends** | **Difficulty**: Medium

## 📋 Description
User performance statistics: average score, improvement trend, total quizzes, pass rate, category breakdown.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/performance-stats`

**Output (200 OK)**:
```json
{
  "success": true,
  "data": {
    "total_quizzes": 45,
    "total_passed": 38,

    "average_score": 76.5,

    "lowest_score": 45,
    "total_points": 3442,
    "improvement_trend": {
      "first_month_avg": 65.2,

    "last_activity": "2026-05-04T10:00:00Z"
  }
}
```

## 🗏️ Business Logic (9 Steps)

1. Validate JWT + user authorization
2. Query `quiz_attempts WHERE user_id = :user_id`
3. Calculate: total count, passed count, pass_rate = passed/total*100
4. Calculate: avg_score = AVG(score), max, min, sum of scores
5. Calculate improvement: compare avg score first month vs current month
6. GROUP BY category, calculate per-category stats
7. Get last attempt timestamp
8. Format response with all calculations
9. Cache for 1 hour (key: `user_stats:{user_id}`)

## 🗄️ Database

```sql
SELECT 
  COUNT(*) as total,
  SUM(CASE WHEN is_passed THEN 1 ELSE 0 END) as passed,
  AVG(score) as avg_score,
  MAX(score) as max_score,
  MIN(score) as min_score,
  SUM(score) as total_score,
  MAX(completed_at) as last_activity
FROM quiz_attempts
WHERE user_id = :user_id;
```

## 💻 Implementation

```python
async def get_performance_stats(db: AsyncSession, user_id: int):
    # Overall stats
    stats = await db.execute(
        select(
            func.count(QuizAttempt.id).label('total'),
            func.sum(case(
                (QuizAttempt.is_passed == True, 1),
                else_=0
            )).label('passed'),
            func.avg(QuizAttempt.score).label('avg_score'),
            func.max(QuizAttempt.score).label('max_score'),
            func.min(QuizAttempt.score).label('min_score'),
            func.sum(QuizAttempt.score).label('total_score'),
            func.max(QuizAttempt.completed_at).label('last_activity')
        ).where(QuizAttempt.user_id == user_id)
    )
    row = stats.first()
    
    total, passed, avg_score, max_score, min_score, total_score, last_activity = row
    pass_rate = (passed / total * 100) if total > 0 else 0
    
    # Improvement trend
    first_month_result = await db.execute(
        select(func.avg(QuizAttempt.score)).where(
            and_(
                QuizAttempt.user_id == user_id,
                QuizAttempt.completed_at >= func.date_trunc('month', func.now() - func.cast('1 month', Interval))
            )
        )
    )
    first_month_avg = first_month_result.scalar() or avg_score
    improvement = ((avg_score - first_month_avg) / first_month_avg * 100) if first_month_avg > 0 else 0
    
    # Category breakdown
    category_stats = await db.execute(
        select(
            QuizCategory.name,
            func.count(QuizAttempt.id).label('quizzes'),
            func.avg(QuizAttempt.score).label('avg'),
            func.sum(case((QuizAttempt.is_passed, 1), else_=0)) * 100 / func.count(QuizAttempt.id)
        ).join(Quiz).join(QuizCategory).where(QuizAttempt.user_id == user_id).group_by(QuizCategory.name)
    )
    
    return {
        "total_quizzes": total or 0,
        "total_passed": passed or 0,
        "pass_rate": float(pass_rate) if pass_rate else 0,
        "average_score": float(avg_score) if avg_score else 0,
        "highest_score": int(max_score) if max_score else 0,
        "lowest_score": int(min_score) if min_score else 0,
        "total_points": int(total_score) if total_score else 0,
        "improvement_trend": {
            "first_month_avg": float(first_month_avg) if first_month_avg else 0,
            "current_month_avg": float(avg_score) if avg_score else 0,
            "improvement_percent": float(improvement)
        },
        "category_breakdown": [dict(row) for row in category_stats],
        "last_activity": last_activity
    }
```
