# G02_F04_SF04: Get Quiz Performance Analytics

✅ MVP | **Admin** | **Difficulty**: Medium

## 📋 Description
Quiz-level analytics: average score, difficulty rating, attempt count, pass rate, common mistakes.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/quizzes/{quiz_id}/analytics`

**Output (200 OK)**:
```json
{
  "success": true,
  "data": {
    "quiz_id": 12,
    "quiz_title": "Basic Addition",
    "total_attempts": 245,
    "passed_attempts": 210,
    "pass_rate": 85.7,
    "average_score": 81.2,
    "median_score": 85,
    "difficulty_rating": 2.1,
    "avg_time_seconds": 1245,
    "completion_rate": 92.3,
    "common_questions_struggled": [
      {"question_id": 3, "text": "...", "fail_rate": 35.2},
      {"question_id": 7, "text": "...", "fail_rate": 28.5}
    ]
  }
}
```

## 🗏️ Business Logic (8 Steps)

1. Validate JWT + admin role
2. Fetch quiz by quiz_id, return 404 if not found
3. Query `quiz_attempts WHERE quiz_id = :quiz_id`
4. Calculate: total_attempts, passed, pass_rate, avg/median score
5. Calculate: avg_time_spent, completion_rate = completed/total
6. Query question attempt stats: fail_rate per question, order by highest failure
7. Assign difficulty_rating = pass_rate adjusted by attempt count (0-5 scale)
8. Cache for 1 hour

## 💻 Implementation

```python
async def get_quiz_analytics(db: AsyncSession, quiz_id: int):
    # Check quiz exists
    quiz = await db.get(Quiz, quiz_id)
    if not quiz:
        raise HTTPException(status_code=404)
    
    # Overall stats
    stats = await db.execute(
        select(
            func.count(QuizAttempt.id).label('total'),
            func.sum(case((QuizAttempt.is_passed, 1), else_=0)).label('passed'),
            func.avg(QuizAttempt.score).label('avg_score'),
            func.min(QuizAttempt.score).label('min_score'),
            func.avg(QuizAttempt.time_spent_seconds).label('avg_time')
        ).where(QuizAttempt.quiz_id == quiz_id)
    )
    row = stats.first()
    
    total, passed, avg_score, min_score, avg_time = row
    pass_rate = (passed / total * 100) if total > 0 else 0
    difficulty_rating = 5 - (pass_rate / 100 * 5)  # 5 = hardest, 0 = easiest
    
    # Question struggles
    questions = await db.execute(
        select(Question.id, Question.text, 
               func.sum(case((QuestionAttempt.is_correct == False, 1), else_=0)) * 100.0 / func.count(QuestionAttempt.id)
        ).join(QuestionAttempt).where(Question.quiz_id == quiz_id)
        .group_by(Question.id).order_by(desc('fail_rate')).limit(5)
    )
    
    return {
        "quiz_id": quiz_id,
        "quiz_title": quiz.title,
        "total_attempts": total or 0,
        "passed_attempts": passed or 0,
        "pass_rate": float(pass_rate),
        "average_score": float(avg_score) if avg_score else 0,
        "difficulty_rating": round(float(difficulty_rating), 1),
        "avg_time_seconds": int(avg_time) if avg_time else 0,
        "common_questions_struggled": [dict(row) for row in questions]
    }
```

## 📊’ Security

1. Admin-only access
2. Rate limiting: 50 requests/minute
3. No user PII in response
4. Parameterized queries
5. Cache: 1 hour
