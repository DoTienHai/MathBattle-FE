# G02_F04_SF05: Get Category Performance

✅ MVP | **Public** | **Popular/Least Popular** | **Difficulty**: Easy

## 📋 Description
Category performance metrics: most/least popular categories, average scores per category, attempt counts.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/categories/performance`

**Query**: `sort_by`: "popular" | "least_popular" | "avg_score"

**Output**:
```json
{
  "success": true,
  "data": {
    "categories": [
      {
        "category_id": 1,
        "name": "Arithmetic",
        "total_attempts": 5420,
        "unique_users": 1240,
        "average_score": 78.3,
        "pass_rate": 82.1
      }
    ]
  }
}
```

## 🗏️ Business Logic (6 Steps)

1. Validate JWT
2. Query categories with attempt stats via JOIN and GROUP BY
3. Calculate: total_attempts, unique_user_count, avg_score, pass_rate
4. Sort by: popularity (attempt count), least popular, or avg_score
5. Limit to top 50
6. Cache 1 hour

## 💻 Implementation

```python
async def get_category_performance(db: AsyncSession, sort_by: str = "popular"):
    query = select(
        QuizCategory.id, QuizCategory.name,
        func.count(QuizAttempt.id).label('total_attempts'),
        func.count(distinct(QuizAttempt.user_id)).label('unique_users'),
        func.avg(QuizAttempt.score).label('avg_score'),
        func.sum(case((QuizAttempt.is_passed, 1), else_=0)) * 100.0 / func.count(QuizAttempt.id)
    ).outerjoin(Quiz).outerjoin(QuizAttempt).group_by(QuizCategory.id, QuizCategory.name)
    
    if sort_by == "popular":
        query = query.order_by(func.count(QuizAttempt.id).desc())
    elif sort_by == "least_popular":
        query = query.order_by(func.count(QuizAttempt.id).asc())
    else:
        query = query.order_by(func.avg(QuizAttempt.score).desc())
    
    result = await db.execute(query.limit(50))
    return [dict(row) for row in result]
```
