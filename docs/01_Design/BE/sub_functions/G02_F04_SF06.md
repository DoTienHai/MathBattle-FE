# G02_F04_SF06: Export Quiz Results as CSV

✅ MVP | **Admin** | **CSV Export** | **Difficulty**: Medium

## 📋 Description
Export quiz attempt results as CSV file. Admin-only. Supports filtering by date range and quiz.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/quizzes/{quiz_id}/export-csv`

**Query Parameters**:
- `start_date`: "2026-05-01"
- `end_date`: "2026-05-04"

**Output**: CSV file (Content-Type: text/csv)

```csv
attempt_id,user_id,username,quiz_id,quiz_title,score,is_passed,time_spent_seconds,completed_at
1,123,john_doe,12,Basic Addition,85,true,1200,2026-05-04T09:30:00Z
2,124,jane_smith,12,Basic Addition,72,true,1500,2026-05-04T10:00:00Z
```

## 🗏️ Business Logic (7 Steps)

1. Validate JWT + admin role
2. Fetch quiz by quiz_id, return 404 if not found
3. Parse and validate date range (start_date, end_date)
4. Query `quiz_attempts WHERE quiz_id = :quiz_id AND completed_at BETWEEN :start AND :end`
5. JOIN with users and quizzes tables
6. Generate CSV with proper escaping (quote fields with commas)
7. Return as file download with appropriate headers

## 💻 Implementation

```python
import csv
from io import StringIO
from datetime import datetime

async def export_quiz_results(db: AsyncSession, quiz_id: int, start_date: str, end_date: str):
    quiz = await db.get(Quiz, quiz_id)
    if not quiz:
        raise HTTPException(status_code=404)
    
    start = datetime.fromisoformat(start_date)
    end = datetime.fromisoformat(end_date)
    
    results = await db.execute(
        select(
            QuizAttempt.id, QuizAttempt.user_id, User.username,
            QuizAttempt.quiz_id, Quiz.title,
            QuizAttempt.score, QuizAttempt.is_passed,
            QuizAttempt.time_spent_seconds, QuizAttempt.completed_at
        ).join(User).join(Quiz).where(
            and_(
                QuizAttempt.quiz_id == quiz_id,
                QuizAttempt.completed_at >= start,
                QuizAttempt.completed_at <= end
            )
        )
    )
    
    output = StringIO()
    writer = csv.writer(output)
    writer.writerow([
        "attempt_id", "user_id", "username", "quiz_id", "quiz_title",
        "score", "is_passed", "time_spent_seconds", "completed_at"
    ])
    
    for row in results:
        writer.writerow(row)
    
    return StreamingResponse(
        iter([output.getvalue()]),
        media_type="text/csv",
        headers={"Content-Disposition": f"attachment; filename=quiz_{quiz_id}_results.csv"}
    )

@router.get("/quizzes/{quiz_id}/export-csv")
async def export_results(
    quiz_id: int,
    start_date: str = Query(...),
    end_date: str = Query(...),
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db)
):
    if current_user.role != "admin":
        raise HTTPException(status_code=403)
    return await QuizService.export_quiz_results(db, quiz_id, start_date, end_date)
```
