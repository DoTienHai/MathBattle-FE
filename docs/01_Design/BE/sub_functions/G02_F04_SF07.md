# G02_F04_SF07: Get Achievement Statistics

✅ MVP | **User Stats** | **Achievements** | **Difficulty**: Medium

## 📋 Description
User achievement statistics: total achievements earned, progress toward next achievement, achievements by category.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/achievement-stats`

**Output**:
```json
{
  "success": true,
  "data": {
    "total_achievements": 24,
    "max_possible": 50,
    "completion_percentage": 48,
    "achievements_by_category": [
      {
        "category": "Quiz Master",
        "earned": 3,
        "total": 5,
        "progress_percent": 60
      }
    ],
    "recent_achievements": [
      {
        "id": 12,
        "name": "First 100 Points",
        "earned_at": "2026-05-01T10:00:00Z"
      }
    ]
  }
}
```

## 🗏️ Business Logic (6 Steps)

1. Validate JWT + authorization
2. Query `user_achievements WHERE user_id = :user_id`
3. Count earned achievements, compare to total available
4. Calculate completion %
5. GROUP BY category, show progress per category
6. Return recent 5 achievements
