# G04_F03_SF01: Get UI Theme Settings

✅ MVP | **User Settings** | **Difficulty**: Easy

## 📋 Description
Retrieve user's UI customization preferences: theme (light/dark), language, notification settings.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/ui-settings`

**Output (200)**:
```json
{
  "success": true,
  "data": {
    "theme": "dark",
    "language": "en",
    "notifications_enabled": true,
    "sound_enabled": true,
    "created_at": "2026-01-01T00:00:00Z",
    "updated_at": "2026-05-04T10:00:00Z"
  }
}
```

## 🗏️ Business Logic (5 Steps)

1. Validate JWT + authorization (own user or admin)
2. Query `user_settings WHERE user_id = :user_id`
3. Return defaults if no settings found
4. Cache for 24 hours
5. Return settings object
