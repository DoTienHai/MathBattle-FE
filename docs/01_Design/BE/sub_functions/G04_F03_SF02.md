# G04_F03_SF02: Update UI Settings

✅ MVP | **User Settings** | **Difficulty**: Easy

## 📋 Description
Update user's UI preferences (theme, language, notifications). Partial update supported.

## 🎯 API Specification

**Endpoint**: `PUT /api/v1/users/{user_id}/ui-settings`

**Input**:
```json
{
  "theme": "light",
  "language": "vi",
  "notifications_enabled": false,
  "sound_enabled": true
}
```

**Output (200)**:
```json
{
  "success": true,
  "data": {
    "theme": "light",
    "language": "vi",
    "updated_at": "2026-05-04T10:30:00Z"
  }
}
```
