# G04_F03_SF03: Reset Settings to Default

✅ MVP | **User Settings** | **Difficulty**: Easy

## 📋 Description
Reset all UI settings to application defaults.

## 🎯 API Specification

**Endpoint**: `POST /api/v1/users/{user_id}/ui-settings/reset`

**Input**: `{ "confirm": true }`

**Output (200)**:
```json
{
  "success": true,
  "message": "Settings reset to default",
  "data": {
    "theme": "auto",
    "language": "en",
    "notifications_enabled": true,
    "sound_enabled": true
  }
}
```

## 🗏️ Business Logic (4 Steps)

1. Validate JWT + authorization
2. Require `confirm=true`
3. Reset all settings to application defaults
4. Invalidate cache
