# G05_F01_SF01: Get User Points Balance

✅ MVP | **User Points** | **Difficulty**: Easy

## 📋 Description
Retrieve user's current points balance and lifetime points earned.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/points`

**Output (200)**:
```json
{
  "success": true,
  "data": {
    "current_balance": 2450,
    "lifetime_earned": 5680,
    "lifetime_spent": 3230,
    "last_updated": "2026-05-04T10:00:00Z"
  }
}
```

## 🗏️ Business Logic (5 Steps)

1. Validate JWT + authorization (own user or admin)
2. Query `user_points WHERE user_id = :user_id`
3. Calculate: current_balance, SUM(earned), SUM(spent) from transaction log
4. Cache for 1 hour
5. Return point balances
