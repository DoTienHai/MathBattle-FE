# G06_F01_SF01: Get User Profile

✅ MVP | **Social** | **Public Profile** | **Difficulty**: Easy

## 📋 Description
Retrieve public user profile: username, avatar, stats, bio. Can be own profile or other users' profiles.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/profile`

**Output (200)**:
```json
{
  "success": true,
  "data": {
    "user_id": 123,
    "username": "john_doe",
    "avatar_url": "https://cdn.example.com/avatars/123.jpg",
    "bio": "Math enthusiast and educator",
    "level": 15,
    "total_points": 4850,
    "quizzes_completed": 45,
    "average_score": 78.5,
    "achievement_count": 24,
    "friends_count": 32,
    "is_online": true,
    "last_active": "2026-05-04T10:00:00Z",
    "joined_date": "2025-01-15T00:00:00Z"
  }
}
```

## 🗏️ Business Logic (6 Steps)

1. Validate JWT
2. Fetch user by user_id, return 404 if not found
3. Check if user is blocked by current user or vice versa (hide profile if blocked)
4. Query user stats: quiz count, average score, achievements, friends
5. Get online status from last_activity timestamp
6. Cache profile for 1 hour per user
