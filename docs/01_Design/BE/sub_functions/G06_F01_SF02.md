# G06_F01_SF02: Get Friends List

✅ MVP | **Social** | **Pagination** | **Difficulty**: Medium

## 📋 Description
Retrieve paginated list of user's friends with online status and basic stats.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/friends`

**Query**: `page`, `page_size`, `sort_by` (online_first, name, recently_added)

**Output (200)**:
```json
{
  "success": true,
  "data": {
    "friends": [
      {
        "friend_id": 124,
        "username": "jane_smith",
        "avatar_url": "https://...",
        "is_online": true,
        "last_active": "2026-05-04T10:30:00Z",
        "mutual_friends": 5,
        "added_date": "2026-02-01T00:00:00Z"
      }
    ],
    "pagination": {"page": 1, "total": 32}
  }
}
```

## 🗏️ Business Logic (8 Steps)

1. Validate JWT + authorization (own or admin)
2. Query friendships: `WHERE (user_id1=:uid OR user_id2=:uid) AND is_accepted=true`
3. Count total friends
4. Apply sorting: online_first (is_online DESC), name (ASC), or added_date (DESC)
5. Paginate: offset, limit
6. Get online status for each friend
7. Count mutual friends
8. Cache for 1 hour per user
