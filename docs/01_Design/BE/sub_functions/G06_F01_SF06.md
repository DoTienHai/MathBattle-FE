# G06_F01_SF06: Block User

✅ MVP | **Social** | **Privacy** | **Difficulty**: Easy

## 📋 Description
Block another user: hide their profile, prevent messages, remove existing friendship.

## 🎯 API Specification

**Endpoint**: `POST /api/v1/users/{user_id}/block`

**Input**: (empty body)

**Output (200)**:
```json
{
  "success": true,
  "message": "User blocked successfully",
  "data": {
    "blocked_user_id": 124,
    "blocked_at": "2026-05-04T10:30:00Z"
  }
}
```

## 🗏️ Business Logic (6 Steps)

1. Validate JWT + extract user_id
2. Fetch target user, return 404 if not found
3. Insert block record: user_id, blocked_user_id
4. If friends, delete friendship
5. Cancel any pending messages
6. Prevent target from messaging or viewing profile
