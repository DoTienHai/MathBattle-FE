# G06_F01_SF05: Remove Friend

✅ MVP | **Social** | **Difficulty**: Easy

## 📋 Description
Remove existing friend. Deletes friendship bidirectionally.

## 🎯 API Specification

**Endpoint**: `DELETE /api/v1/users/{friend_id}/friends`

**Output (204 No Content)**: No response body

## 🗏️ Business Logic (4 Steps)

1. Validate JWT + authorization
2. Fetch friendship, return 404 if not found
3. Delete friendship record (both directions)
4. Return 204
