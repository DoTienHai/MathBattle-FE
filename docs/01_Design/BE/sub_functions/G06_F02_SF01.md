# G06_F01_SF02: Unblock User

✅ MVP | **Social** | **Privacy** | **Difficulty**: Easy

## 📋 Description
Unblock a previously blocked user. Restores ability to see their profile and receive messages.

## 🎯 API Specification

**Endpoint**: `DELETE /api/v1/users/{user_id}/block`

**Output (200)**:
```json
{
  "success": true,
  "message": "User unblocked successfully",
  "data": {
    "unblocked_user_id": 124,
    "unblocked_at": "2026-05-04T10:30:00Z"
  }
}
```

## 🗏️ Business Logic (4 Steps)

1. Validate JWT + extract user_id
2. Find block record, return 404 if not found
3. Delete block record
4. Restore ability to view profile and messaging
