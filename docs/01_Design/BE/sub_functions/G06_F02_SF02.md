# G06_F02_SF02: Send Message

✅ MVP | **Social** | **Messaging** | **Difficulty**: Medium

## 📋 Description
Send message to a friend. Must be connected as friends. Checks for blocks.

## 🎯 API Specification

**Endpoint**: `POST /api/v1/messages`

**Input**:
```json
{
  "recipient_id": 124,
  "content": "Hey, want to do a quiz together?",
  "message_type": "text"
}
```

**Output (201)**:
```json
{
  "success": true,
  "data": {
    "message_id": 890,
    "recipient_id": 124,
    "content": "Hey, want to do a quiz together?",
    "sent_at": "2026-05-04T10:30:00Z",
    "read": false
  }
}
```

## 🗏️ Business Logic (7 Steps)

1. Validate JWT + extract sender_id
2. Fetch recipient, return 404 if not found
3. Check friendship exists
4. Check sender not blocked by recipient
5. Validate content (max 1000 chars, not empty)
6. INSERT message record
7. Send notification to recipient (real-time via WebSocket if available)
