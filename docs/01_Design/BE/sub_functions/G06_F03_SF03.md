# G06_F03_SF03: Get Unread Notification Count

✅ MVP | **Social** | **Quick Count** | **Difficulty**: Easy

## 📋 Description
Get quick count of unread notifications. Useful for badge display in UI.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/notifications/unread/count`

**Output (200)**:
```json
{
  "success": true,
  "data": {
    "unread_count": 5,
    "by_type": {
      "friend_request": 2,
      "achievement": 1,
      "challenge": 2,
      "system": 0
    }
  }
}
```

## 🗏️ Business Logic (4 Steps)

1. Validate JWT + extract user_id
2. Query: COUNT(*) FROM notifications WHERE user_id=:user_id AND read=false
3. Group count by type
4. Cache for 1 minute
