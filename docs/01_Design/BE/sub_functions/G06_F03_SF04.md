# G06_F03_SF04: Delete Notification

✅ MVP | **Social** | **Cleanup** | **Difficulty**: Easy

## 📋 Description
Delete/archive notification. Useful for cleanup. Can be single or bulk delete.

## 🎯 API Specification

**Endpoint**: `DELETE /api/v1/notifications/{notification_id}`

Or bulk: `DELETE /api/v1/notifications`

**Input (bulk)**:
```json
{
  "notification_ids": [111, 112]
}
```

**Output (204)**: No content

## 🗏️ Business Logic (3 Steps)

1. Validate JWT + authorization
2. DELETE FROM notifications WHERE notification_id IN (...)
3. Return 204
