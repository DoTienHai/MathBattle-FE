# G06_F03_SF05: Clear All Notifications

✅ MVP | **Social** | **Batch Clear** | **Difficulty**: Easy

## 📋 Description
Clear all user notifications at once. Requires confirmation to prevent accidents.

## 🎯 API Specification

**Endpoint**: `POST /api/v1/notifications/clear-all`

**Input**:
```json
{
  "confirm": true
}
```

**Output (200)**:
```json
{
  "success": true,
  "message": "All notifications cleared",
  "data": {
    "deleted_count": 42
  }
}
```

## 🗏️ Business Logic (4 Steps)

1. Validate JWT + extract user_id
2. Require confirm=true parameter
3. DELETE FROM notifications WHERE user_id = :user_id
4. Return deleted count
