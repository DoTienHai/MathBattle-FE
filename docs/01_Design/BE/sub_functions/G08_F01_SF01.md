## 📋 Description

Retrieve comprehensive user statistics including total quizzes taken, average score, best score, current streak, and total points earned. This endpoint aggregates quiz attempt data and response metrics to provide a profile overview.

---

  - If provided: must match authenticated user_id or require admin role
  - If omitted: get stats for current authenticated user

## 🗏️ Business Logic (10 Steps)

1. **Validate User Authentication**
   - Verify JWT token is valid
   - Extract authenticated user_id from token
   - If user_id path parameter provided, verify it matches authenticated user or token has admin=true
   - Return 401 if token invalid/expired, 403 if unauthorized access to other user's stats

2. **Verify User Exists**
   - Query users table WHERE id = :user_id AND status != 'banned'
   - Return 404 if user not found or banned

3. **Calculate Total Quiz Statistics**
   - COUNT quiz_attempts WHERE user_id = :user_id â†’ total_attempts
   - SELECT COUNT(DISTINCT quiz_id) FROM quiz_attempts WHERE user_id = :user_id â†’ total_unique_quizzes
   - AVG(score) FROM quiz_attempts WHERE user_id = :user_id â†’ average_score
   - MAX(score) FROM quiz_attempts â†’ best_score
   - MIN(score) FROM quiz_attempts â†’ worst_score

4. **Calculate Passing Rate**
   - COUNT(*) WHERE user_id = :user_id AND is_passed = true â†’ passing_attempts
   - passing_rate = (passing_attempts / total_attempts) * 100
   - Precision: 1 decimal place

5. **Calculate Current & Longest Streaks**
   - Query quiz_attempts ordered by completed_at DESC
   - Iterate: count consecutive passed quizzes â†’ current_streak
   - Iterate all attempts: find max consecutive passed â†’ longest_streak
   - A streak breaks if quiz_attempt.is_passed = false

6. **Calculate Total Points Earned**
   - For each completed quiz_attempt:
     - SELECT points_reward FROM quizzes WHERE id = quiz_attempt.quiz_id
     - If is_passed = true: add full points_reward
     - If is_passed = false: add 0 points
   - SUM all points â†’ total_points_earned

7. **Fetch Category-Wise Breakdown**
   - GROUP BY quiz.category FROM quiz_attempts JOIN quizzes
   - For each category: calculate attempts, avg_score, best_score, passing_rate
   - Include only categories user has attempted

8. **Get Last Quiz Attempt Details**
   - SELECT quiz_attempts, quizzes WHERE user_id = :user_id
   - ORDER BY completed_at DESC LIMIT 1
   - Extract: completed_at, quiz_title, quiz_id

9. **Calculate Total Time Spent**
   - SUM(time_spent_seconds) FROM quiz_attempts WHERE user_id = :user_id
   - Convert to minutes: total_seconds / 60

10. **Return Aggregated Response**
    - Cache result for 30 minutes (Redis key: user_stats:{user_id})
    - Return success=true with all calculated metrics
    - Include account_created_at from users table

## âŒ Error Cases (7 Scenarios)

1. **Unauthorized Access (401)**
   - No JWT token provided
   - JWT token expired or malformed
   - Action: Return 401 with "Invalid or expired token"

2. **Forbidden Access (403)**
   - Authenticated user trying to access another user's stats
   - User has standard role (not admin)
   - Action: Return 403 with "Cannot access other user's statistics"

3. **User Not Found (404)**
   - user_id doesn't exist in database
   - User has status='banned'
   - Action: Return 404 with "User not found"

4. **No Quiz Attempts (200 with empty metrics)**
   - User has never attempted any quiz
   - all_attempts = 0, average_score = null, streaks = 0
   - Action: Return success with zero values

5. **Database Query Error (500)**
   - Database connection timeout
   - Query execution failure
   - Action: Return 500 with "Failed to retrieve user statistics"

6. **Cache Corruption (500)**
   - Invalid cached data format
   - Action: Invalidate cache and recompute from database

7. **Invalid user_id Parameter (400)**
   - user_id is negative or invalid type
   - Action: Return 400 with "Invalid user_id format"

### Service Layer

```python
# app/services/analytics_service.py
from sqlalchemy import and_, func, desc
from sqlalchemy.orm import Session
from typing import Dict, Optional, List, Tuple
from datetime import datetime
import logging
from app.models import QuizAttempt, Quiz, QuizCategory, User
from app.cache import redis_cache

logger = logging.getLogger(__name__)

class AnalyticsService:
    
    @staticmethod
    async def get_user_statistics(
        user_id: int,
        db: Session
    ) -> Dict:
        """
        Fetch comprehensive user statistics.
        
        Args:
            user_id: User ID to fetch stats for
            db: Database session
            
        Returns:
            Dictionary with all user statistics
        """
        # Check cache first
        cache_key = f"user_stats:{user_id}"
        cached_stats = await redis_cache.get(cache_key)
        if cached_stats:
            logger.debug(f"Returning cached stats for user {user_id}")
            return cached_stats
        
        # Verify user exists
        user = db.query(User).filter(
            and_(User.id == user_id, User.status != 'banned')
        ).first()
        if not user:
            raise ValueError(f"User {user_id} not found")
        
        # Get all quiz attempts for user
        attempts = db.query(QuizAttempt).filter(
            QuizAttempt.user_id == user_id
        ).order_by(desc(QuizAttempt.completed_at)).all()
        
        total_attempts = len(attempts)
        
        if total_attempts == 0:
            # No attempts yet
            stats = {
                "user_id": user_id,
                "total_quizzes_taken": 0,
                "total_quiz_attempts": 0,
                "average_score": None,
                "best_score": None,
                "worst_score": None,
                "passing_rate": 0.0,
                "current_streak": 0,
                "longest_streak": 0,
                "total_points_earned": 0,
                "quiz_categories": {},
                "last_quiz_attempt": None,
                "last_quiz_title": None,
                "total_time_spent_minutes": 0,
                "account_created_at": user.created_at
            }
            await redis_cache.set(cache_key, stats, ttl=1800)  # 30 min cache
            return stats
        
        # Calculate basic statistics
        scores = [a.score for a in attempts if a.score is not None]
        average_score = sum(scores) / len(scores) if scores else None
        best_score = max(scores) if scores else None
        worst_score = min(scores) if scores else None
        
        # Calculate passing rate
        passed_attempts = sum(1 for a in attempts if a.is_passed)
        passing_rate = (passed_attempts / total_attempts * 100) if total_attempts > 0 else 0
        
        # Calculate streaks
        current_streak = AnalyticsService._calculate_current_streak(attempts)
        longest_streak = AnalyticsService._calculate_longest_streak(attempts)
        
        # Calculate unique quizzes
        unique_quizzes = len(set(a.quiz_id for a in attempts))
        
        # Calculate total points
        total_points = AnalyticsService._calculate_total_points(attempts, db)
        
        # Get category-wise breakdown
        category_stats = AnalyticsService._get_category_stats(user_id, db)
        
        # Get last quiz attempt
        last_attempt = attempts[0] if attempts else None
        last_quiz_title = None
        if last_attempt:
            quiz = db.query(Quiz).filter(Quiz.id == last_attempt.quiz_id).first()
            last_quiz_title = quiz.title if quiz else None
        
        # Calculate total time spent
        total_seconds = db.query(func.sum(QuizAttempt.time_spent_seconds)).filter(
            QuizAttempt.user_id == user_id
        ).scalar() or 0
        total_minutes = total_seconds // 60
        
        stats = {
            "user_id": user_id,
            "total_quizzes_taken": unique_quizzes,
            "total_quiz_attempts": total_attempts,
            "average_score": round(average_score, 1) if average_score else None,
            "best_score": best_score,
            "worst_score": worst_score,
            "passing_rate": round(passing_rate, 1),
            "current_streak": current_streak,
            "longest_streak": longest_streak,
            "total_points_earned": total_points,
            "quiz_categories": category_stats,
            "last_quiz_attempt": last_attempt.completed_at if last_attempt else None,
            "last_quiz_title": last_quiz_title,
            "total_time_spent_minutes": total_minutes,
            "account_created_at": user.created_at
        }
        
        # Cache result
        await redis_cache.set(cache_key, stats, ttl=1800)
        
        return stats
    
    @staticmethod
    def _calculate_current_streak(attempts: List[QuizAttempt]) -> int:
        """Calculate current streak of passed quizzes"""
        streak = 0
        for attempt in attempts:
            if attempt.is_passed:
                streak += 1
            else:
                break
        return streak
    
    @staticmethod
    def _calculate_longest_streak(attempts: List[QuizAttempt]) -> int:
        """Calculate longest streak of passed quizzes"""
        longest = 0
        current = 0
        for attempt in attempts:
            if attempt.is_passed:
                current += 1
                longest = max(longest, current)
            else:
                current = 0
        return longest
    
    @staticmethod
    def _calculate_total_points(
        attempts: List[QuizAttempt],
        db: Session
    ) -> int:
        """Calculate total points earned"""
        total = 0
        for attempt in attempts:
            if attempt.is_passed:
                quiz = db.query(Quiz.points_reward).filter(
                    Quiz.id == attempt.quiz_id
                ).first()
                if quiz:
                    total += quiz.points_reward
        return total
    
    @staticmethod
    def _get_category_stats(user_id: int, db: Session) -> Dict[str, Dict]:
        """Get category-wise statistics"""
        # Query attempts grouped by category
        results = db.query(
            Quiz.category,
            func.count(QuizAttempt.id).label('attempts'),
            func.avg(QuizAttempt.score).label('avg_score'),
            func.max(QuizAttempt.score).label('best_score'),
            func.sum(
                func.cast(QuizAttempt.is_passed, Integer)
            ).label('passed')
        ).join(
            Quiz, QuizAttempt.quiz_id == Quiz.id
        ).filter(
            QuizAttempt.user_id == user_id
        ).group_by(
            Quiz.category
        ).all()
        
        category_stats = {}
        for row in results:
            passing_rate = (row.passed / row.attempts * 100) if row.attempts > 0 else 0
            category_stats[row.category] = {
                "attempts": row.attempts,
                "average_score": round(row.avg_score, 1) if row.avg_score else 0,
                "best_score": row.best_score or 0,
                "passing_rate": round(passing_rate, 1)
            }
        
        return category_stats
```

# app/api/v1/analytics.py
from fastapi import APIRouter, Depends, HTTPException, status, Path
from typing import Optional
import logging

router = APIRouter(prefix="/analytics", tags=["analytics"])
logger = logging.getLogger(__name__)

@router.get("/users/{user_id}/statistics", response_model=UserStatsResponse)
async def get_user_statistics(
    user_id: int = Path(..., gt=0),
    current_user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> UserStatsResponse:
    """
    Get comprehensive user statistics.
    
    Path Parameters:
    - user_id: User ID to fetch stats for
    
    Returns:
    - User statistics including quizzes taken, scores, streaks, points
    
    Notes:
    - Users can only access their own stats (or any stats if admin)
    - Results cached for 30 minutes
    """
    try:
        # Authorization check
        if user_id != current_user.id and not current_user.is_admin:
            logger.warning(
                f"User {current_user.id} attempted unauthorized access to user {user_id} stats"
            )
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Cannot access other user's statistics"
            )
        
        logger.info(f"Fetching statistics for user {user_id}")
        
        stats = await AnalyticsService.get_user_statistics(
            user_id=user_id,
            db=db
        )
        
        return UserStatsResponse(
            success=True,
            data=UserStatistics(**stats)
        )
    
    except ValueError as e:
        logger.warning(f"Validation error: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_404_NOT_FOUND,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Failed to fetch user statistics: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to retrieve user statistics"
        )

@router.get("/users/me/statistics", response_model=UserStatsResponse)
async def get_own_statistics(
    current_user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> UserStatsResponse:
    """
    Get statistics for authenticated user (convenience endpoint).
    
    Returns:
    - Current user's complete statistics
    """
    return await get_user_statistics(
        user_id=current_user.id,
        current_user=current_user,
        db=db
    )

## 📊’ Security Considerations

1. **Authentication & Authorization**
   - Verify JWT token for all requests
   - Users can only access own stats unless admin role
   - Return 403 for unauthorized access attempts
   - Log all unauthorized access attempts

2. **Data Privacy**
   - Never expose internal calculation details
   - Sanitize all returned data
   - Prevent timing attacks by caching all users equally
   - Don't expose banned user information

3. **Rate Limiting**
   - Implement rate limit: 200 requests per minute per user
   - Cache results aggressively to reduce database load
   - Return 429 Too Many Requests if exceeded

4. **Cache Security**
   - Cache user stats for max 30 minutes
   - Invalidate cache when user status changes (ban, password update)
   - Use user_id as cache key with namespace
   - Never cache incomplete or error responses

5. **SQL Injection Prevention**
   - Use parameterized queries (SQLAlchemy ORM)
   - Never concatenate user input into queries
   - Validate user_id format before use
