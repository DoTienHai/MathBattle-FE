# G08_F01_SF02: Get User Statistics

✅ MVP | **Analytics** | **User Progress**

## 📋 Description

Retrieve aggregated statistics for authenticated user: total points, quizzes attempted, average score, level progression, and streaks.

## 🎯 API Specification

**Endpoint**: `GET /api/v1/users/{user_id}/statistics`

**Output (200 OK)**
```json
{
  "success": true,
  "data": {
    "user_id": 1,
    "total_points": 2500,
    "level": 5,
    "points_to_next_level": 500,
    "total_quizzes_attempted": 45,
    "quizzes_passed": 38,
    "average_score": 82.5,
    "current_streak": 7,
    "longest_streak": 15,
    "total_questions_answered": 450,
    "accuracy_rate": 85.2,
    "favorite_category": "arithmetic",
    "first_activity": "2026-04-01T10:00:00Z",
    "last_activity": "2026-05-04T15:30:00Z"
  }
}
```

**Errors**: 401 (unauthorized), 404 (user not found), 500 (error)

## 🗏️ Business Logic (10 Steps)

1. **Authenticate User** - Verify JWT token
2. **Validate User ID** - Check user_id parameter
3. **Check User Exists** - Query user from database
4. **Calculate Total Points** - SUM from quiz_attempts WHERE user_id
5. **Get Current Level** - Query from users table
6. **Calculate Points to Next** - Hardcoded formula or level_progress table
7. **Count Quiz Attempts** - COUNT(*) from quiz_attempts
8. **Calculate Average Score** - AVG(score) from passed attempts
9. **Calculate Streaks** - Calculate current and longest consecutive pass days
10. **Return Statistics** - Compile and return all metrics

## 🗄️ Database Schema

Existing tables: `users`, `quiz_attempts`, `quiz_questions`

New optional: `user_streaks` for tracking

## 💻 Backend Implementation

```python
# app/api/v1/users.py
@router.get("/{user_id}/statistics")
async def get_user_statistics(
    user_id: int,
    current_user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
):
    """Get user statistics"""
    try:
        # Only users can view their own stats (admin can view any)
        if current_user.id != user_id and current_user.role != "admin":
            raise HTTPException(status_code=403, detail="Unauthorized")
        
        user = db.query(User).filter_by(id=user_id).first()
        if not user:
            raise HTTPException(status_code=404, detail="User not found")
        
        # Calculate statistics
        total_points = db.query(func.sum(QuizAttempt.points_earned)).filter(
            QuizAttempt.user_id == user_id
        ).scalar() or 0
        
        total_attempts = db.query(func.count(QuizAttempt.id)).filter(
            QuizAttempt.user_id == user_id
        ).scalar()
        
        passed_attempts = db.query(func.count(QuizAttempt.id)).filter(
            and_(
                QuizAttempt.user_id == user_id,
                QuizAttempt.is_passed == True
            )
        ).scalar()
        
        avg_score = db.query(func.avg(QuizAttempt.score)).filter(
            and_(
                QuizAttempt.user_id == user_id,
                QuizAttempt.score != None
            )
        ).scalar() or 0
        
        # Calculate streaks
        current_streak = calculate_current_streak(user_id, db)
        longest_streak = calculate_longest_streak(user_id, db)
        
        # Favorite category
        favorite_cat = db.query(QuizQuestion.category).filter(
            QuizAttempt.user_id == user_id
        ).group_by(QuizQuestion.category).order_by(
            func.count().desc()
        ).first()
        
        last_activity = db.query(func.max(QuizAttempt.completed_at)).filter(
            QuizAttempt.user_id == user_id
        ).scalar()
        
        return {
            "success": True,
            "data": {
                "user_id": user.id,
                "total_points": int(total_points),
                "level": user.level,
                "points_to_next_level": max(0, (user.level * 1000) - total_points),
                "total_quizzes_attempted": total_attempts or 0,
                "quizzes_passed": passed_attempts or 0,
                "average_score": round(float(avg_score), 1),
                "current_streak": current_streak,
                "longest_streak": longest_streak,
                "accuracy_rate": round((passed_attempts / max(1, total_attempts)) * 100, 1),
                "favorite_category": favorite_cat[0] if favorite_cat else None,
                "first_activity": user.created_at.isoformat(),
                "last_activity": last_activity.isoformat() if last_activity else None
            }
        }
    except HTTPException:
        raise
    except Exception as e:
        logger.error(f"Stats error: {str(e)}")
        raise HTTPException(status_code=500, detail="Failed to fetch statistics")
```
