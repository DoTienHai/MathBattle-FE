## 🗏️ Business Logic (10 Steps)

1. **Verify Admin Authentication**
   - Check JWT token is valid
   - Verify token has admin=true
   - Verify admin role not revoked
   - Return 401 if invalid, 403 if not admin

2. **Check Cache Status**
   - Query Redis for stats:{system}
   - If cache exists and not expired: return cached data with timestamp
   - If cache expired or missing: proceed to calculate

3. **Calculate User Metrics**
   - COUNT(*) FROM users WHERE status != 'banned' â†’ total_users
   - COUNT(*) FROM users WHERE status = 'banned' â†’ banned_users
   - COUNT(*) FROM quiz_attempts WHERE DATE(completed_at) = TODAY â†’ active_today (DISTINCT user_id)
   - COUNT(*) FROM users WHERE created_at >= DATE_SUB(NOW(), INTERVAL 1 DAY) â†’ new_users_today
   - COUNT(*) FROM users WHERE created_at >= DATE_SUB(NOW(), INTERVAL 7 DAY) â†’ new_users_this_week
   - COUNT(*) FROM users WHERE created_at >= DATE_SUB(NOW(), INTERVAL 30 DAY) â†’ new_users_this_month

4. **Calculate Quiz Metrics**
   - COUNT(*) FROM quizzes WHERE is_deleted=false AND is_published=true â†’ total_quizzes
   - COUNT(*) FROM quiz_questions WHERE is_deleted=false â†’ total_questions
   - COUNT(*) FROM quiz_attempts â†’ total_quiz_attempts

5. **Calculate Score Metrics**
   - AVG(score) FROM quiz_attempts WHERE score IS NOT NULL â†’ average_user_score
   - COUNT(*) WHERE is_passed=true / COUNT(*) * 100 â†’ passing_rate
   - AVG(time_spent_seconds) / 60 â†’ average_time_per_quiz

6. **Calculate Engagement**
   - total_quiz_attempts / total_users â†’ average_attempts_per_user
   - COUNT(DISTINCT quiz_id) / COUNT(*) FROM quiz_attempts * 100 â†’ quiz_completion_rate
   - SUM of DAU for last 30 days / 30 â†’ daily_active_users_average

7. **Category Breakdown**
   - For each quiz category:
     - COUNT(quizzes) in category â†’ quiz_count
     - COUNT(quiz_attempts) for quizzes in category â†’ attempt_count
     - AVG(score) for those attempts â†’ average_score

8. **Health Indicators**
   - Measure database response time (mock or real DB latency)
   - Query Redis for cache metrics
   - Check error logs for 24h error count
   - Determine status: healthy | degraded | critical

9. **Compile Response**
   - Add current timestamp
   - Include all calculated metrics
   - Calculate cache expiration time (current_time + 10 minutes)

10. **Cache and Return**
    - Store in Redis with TTL=600 seconds (10 minutes)
    - Log this admin action with admin_id, IP, timestamp
    - Return success=true with all metrics

## âŒ Error Cases (7 Scenarios)

1. **Unauthorized (401)**
   - No JWT token provided
   - Token expired or malformed
   - Action: Return 401 with "Invalid or expired token"

2. **Forbidden - Not Admin (403)**
   - User not admin
   - Admin status revoked
   - Action: Return 403 with "Admin access required"

3. **Database Error (500)**
   - Query execution failure
   - Connection timeout
   - Action: Return 500 with "Failed to retrieve system statistics"

4. **Cache Error (500)**
   - Redis connection failure
   - Cache corruption
   - Action: Log error, fall back to calculating fresh

5. **Invalid Token Claims (401)**
   - admin claim missing or false
   - Action: Return 401

6. **Rate Limit (429)**
   - Too many admin calls
   - Action: Return 429 (but cache usually prevents this)

7. **Timeout (504)**
   - Query takes too long (>30 seconds)
   - Action: Return 504 with "Gateway timeout"

```python
# app/services/admin_service.py
from sqlalchemy import and_, func, text
from sqlalchemy.orm import Session
from typing import Dict, Optional
from datetime import datetime, timedelta
import logging
from app.models import User, Quiz, QuizAttempt, QuizQuestion, AuditLog
from app.cache import redis_cache

logger = logging.getLogger(__name__)

class AdminService:
    
    @staticmethod
    async def get_system_statistics(db: Session, admin_id: int) -> Dict:
        """
        Fetch system statistics for admin dashboard.
        
        Args:
            db: Database session
            admin_id: Admin user ID (for logging)
            
        Returns:
            System statistics dictionary
        """
        # Check cache first
        cache_key = "stats:system"
        cached = await redis_cache.get(cache_key)
        if cached:
            logger.debug("Returning cached system statistics")
            return cached
        
        now = datetime.utcnow()
        
        # Calculate user metrics
        total_users = db.query(func.count(User.id)).filter(
            User.status != 'banned'
        ).scalar() or 0
        
        banned_users = db.query(func.count(User.id)).filter(
            User.status == 'banned'
        ).scalar() or 0
        
        today_start = now.replace(hour=0, minute=0, second=0, microsecond=0)
        active_today = db.query(func.count(func.distinct(QuizAttempt.user_id))).filter(
            QuizAttempt.completed_at >= today_start
        ).scalar() or 0
        
        new_today = db.query(func.count(User.id)).filter(
            User.created_at >= today_start
        ).scalar() or 0
        
        week_ago = now - timedelta(days=7)
        new_week = db.query(func.count(User.id)).filter(
            User.created_at >= week_ago
        ).scalar() or 0
        
        month_ago = now - timedelta(days=30)
        new_month = db.query(func.count(User.id)).filter(
            User.created_at >= month_ago
        ).scalar() or 0
        
        # Calculate quiz metrics
        total_quizzes = db.query(func.count(Quiz.id)).filter(
            and_(Quiz.is_deleted == False, Quiz.is_published == True)
        ).scalar() or 0
        
        total_attempts = db.query(func.count(QuizAttempt.id)).scalar() or 0
        
        total_questions = db.query(func.count(QuizQuestion.id)).filter(
            QuizQuestion.is_deleted == False
        ).scalar() or 0
        
        # Calculate score metrics
        avg_score = db.query(func.avg(QuizAttempt.score)).filter(
            QuizAttempt.score.isnot(None)
        ).scalar()
        
        passed_count = db.query(func.count(QuizAttempt.id)).filter(
            QuizAttempt.is_passed == True
        ).scalar() or 0
        
        passing_rate = (passed_count / total_attempts * 100) if total_attempts > 0 else 0
        
        avg_time = db.query(func.avg(QuizAttempt.time_spent_seconds)).filter(
            QuizAttempt.time_spent_seconds.isnot(None)
        ).scalar() or 0
        
        # Calculate engagement
        avg_attempts_per_user = (total_attempts / total_users) if total_users > 0 else 0
        
        distinct_quizzes = db.query(func.count(func.distinct(QuizAttempt.quiz_id))).scalar() or 0
        completion_rate = (distinct_quizzes / total_quizzes * 100) if total_quizzes > 0 else 0
        
        # DAU average for last 30 days
        dau_values = []
        for i in range(30):
            day_start = (now - timedelta(days=i)).replace(hour=0, minute=0, second=0)
            day_end = day_start + timedelta(days=1)
            dau = db.query(func.count(func.distinct(QuizAttempt.user_id))).filter(
                and_(
                    QuizAttempt.completed_at >= day_start,
                    QuizAttempt.completed_at < day_end
                )
            ).scalar() or 0
            dau_values.append(dau)
        
        avg_dau = sum(dau_values) // len(dau_values) if dau_values else 0
        
        # Category breakdown
        category_results = db.query(
            Quiz.category,
            func.count(Quiz.id).label('quiz_count'),
            func.count(QuizAttempt.id).label('attempt_count'),
            func.avg(QuizAttempt.score).label('avg_score')
        ).outerjoin(
            QuizAttempt, Quiz.id == QuizAttempt.quiz_id
        ).filter(
            Quiz.is_deleted == False
        ).group_by(
            Quiz.category
        ).all()
        
        category_breakdown = {}
        for row in category_results:
            category_breakdown[row.category] = {
                "quiz_count": row.quiz_count or 0,
                "attempt_count": row.attempt_count or 0,
                "average_score": round(row.avg_score, 1) if row.avg_score else None
            }
        
        # Health indicators (simplified)
        cache_hit_rate = 94.2  # Mock value - would integrate with Redis stats
        error_rate = 0.3  # Mock value - would check error logs
        db_response_time = 125  # Mock value - would measure actual DB latency
        
        health_status = "healthy"
        if error_rate > 1.0 or cache_hit_rate < 80:
            health_status = "degraded"
        if error_rate > 5.0 or cache_hit_rate < 50:
            health_status = "critical"
        
        stats = {
            "timestamp": now,
            "cache_expires_at": now + timedelta(seconds=600),
            "key_metrics": {
                "total_users": total_users,
                "active_users_today": active_today,
                "banned_users": banned_users,
                "total_quizzes": total_quizzes,
                "total_quiz_attempts": total_attempts,
                "average_user_score": round(avg_score, 1) if avg_score else None,
                "total_questions": total_questions
            },
            "user_metrics": {
                "new_users_today": new_today,
                "new_users_this_week": new_week,
                "new_users_this_month": new_month,
                "daily_active_users_average": avg_dau
            },
            "engagement_metrics": {
                "average_attempts_per_user": round(avg_attempts_per_user, 1),
                "passing_rate": round(passing_rate, 1),
                "average_time_per_quiz_minutes": round(avg_time / 60, 1),
                "quiz_completion_rate": round(completion_rate, 1)
            },
            "category_breakdown": category_breakdown,
            "health_indicators": {
                "system_status": health_status,
                "database_response_time_ms": db_response_time,
                "cache_hit_rate": cache_hit_rate,
                "error_rate_24h": error_rate
            }
        }
        
        # Log admin action
        await AdminService._log_audit(
            admin_id=admin_id,
            action="GET_SYSTEM_STATS",
            db=db
        )
        
        # Cache for 10 minutes
        await redis_cache.set(cache_key, stats, ttl=600)
        
        return stats
    
    @staticmethod
    async def _log_audit(
        admin_id: int,
        action: str,
        target_type: Optional[str] = None,
        target_id: Optional[int] = None,
        details: Optional[Dict] = None,
        db: Session = None
    ) -> None:
        """Log admin action to audit log"""
        try:
            audit_entry = AuditLog(
                admin_id=admin_id,
                action=action,
                target_type=target_type,
                target_id=target_id,
                ip_address="127.0.0.1",  # Would get from request context
                details=details,
                created_at=datetime.utcnow()
            )
            db.add(audit_entry)
            await db.commit()
            logger.info(f"Admin action logged: {action} by {admin_id}")
        except Exception as e:
            logger.error(f"Failed to log audit: {str(e)}")
            db.rollback()
```

```python
# app/api/v1/admin.py
from fastapi import APIRouter, Depends, HTTPException, status
from app.models import User
from app.auth import get_current_user
from app.database import get_db
from app.services.admin_service import AdminService
from app.schemas.admin import SystemStatsResponse, SystemStatistics
import logging

router = APIRouter(prefix="/admin", tags=["admin"])
logger = logging.getLogger(__name__)

@router.get("/statistics", response_model=SystemStatsResponse)
async def get_system_statistics(
    current_user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> SystemStatsResponse:
    """
    Get system statistics for admin dashboard (KPIs).
    
    Admin Only Endpoint:
    - Requires admin role
    - Requires Bearer token with admin=true
    - All requests logged
    
    Returns:
    - System KPIs: users, quizzes, attempts, scores
    - User metrics: growth, engagement
    - Health indicators: status, response times
    """
    try:
        # Verify admin role
        if not current_user.is_admin:
            logger.warning(f"Non-admin user {current_user.id} attempted admin access")
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Admin access required"
            )
        
        logger.info(f"Admin {current_user.id} fetching system statistics")
        
        stats = await AdminService.get_system_statistics(
            db=db,
            admin_id=current_user.id
        )
        
        return SystemStatsResponse(
            success=True,
            data=SystemStatistics(**stats)
        )
    
    except HTTPException:
        raise
    
    except Exception as e:
        logger.error(f"Failed to fetch system statistics: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to retrieve system statistics"
        )
```

## 📊’ Security Considerations

1. **Admin-Only Access**
   - Verify is_admin=True in JWT claims
   - Return 403 if not admin
   - Double-check admin role in database before granting access

2. **Audit Logging**
   - Log every admin statistics access with admin_id, timestamp, IP
   - Include in compliance audit trail

3. **Data Protection**
   - Cache prevents repeated database queries from revealing timing patterns
   - Never expose raw user personal data
   - Aggregate metrics only

4. **Rate Limiting**
   - Limit to 60 requests per minute per admin
   - Cache prevents abuse

5. **Input Validation**
   - No parameters to validate
   - Validate JWT token properly
