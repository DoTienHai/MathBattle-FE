# G11_F01_SF02: Edit Question

✅ MVP | **Admin** | **Question Management**

## 📋 Description

Admin edits existing question content, options, answer, difficulty, or category. Update question while maintaining referential integrity with quiz attempts.

## 🎯 API Specification

**Endpoint**: `PUT /api/v1/admin/questions/{question_id}`

**Input**
```json
{
  "text": "Updated question text",
  "options": ["Option 1", "Option 2", "Option 3", "Option 4"],
  "correct_option_index": 0,
  "difficulty": "medium",
  "category": "algebra",
  "explanation": "Why this is correct..."
}
```

**Output (200 OK)**
```json
{
  "success": true,
  "data": {
    "question_id": 123,
    "text": "Updated question text",
    "difficulty": "medium",
    "category": "algebra",
    "updated_at": "2026-05-04T10:00:00Z"
  }
}
```

**Errors**: 401 (unauthorized), 403 (forbidden), 404 (not found), 422 (validation), 500 (error)

## 🗏️ Business Logic (10 Steps)

1. **Authenticate Admin** - Verify admin role
2. **Fetch Question** - Query by question_id
3. **Check Not Locked** - Cannot edit if used in attempts
4. **Validate Question Text** - Non-empty, max 1000 chars
5. **Validate Options** - Exactly 4 options, each max 200 chars
6. **Validate Correct Index** - 0-3, must be valid
7. **Validate Difficulty** - enum: easy, medium, hard
8. **Validate Category** - Must exist in categories table
9. **Update Question** - UPDATE with new values
10. **Log Change** - Track admin edits for audit trail

## 🗄️ Database Schema

Existing: `quiz_questions`, `quiz_question_options`

Optional: `question_audit_log` for edit history

## 💻 Backend Implementation

```python
# app/api/v1/admin/questions.py
@router.put("/{question_id}")
async def edit_question(
    question_id: int,
    question_data: dict,
    admin: User = Depends(get_admin_user),
    db: Session = Depends(get_db)
):
    """Edit question (admin only)"""
    try:
        question = db.query(QuizQuestion).filter_by(id=question_id).first()
        if not question:
            raise HTTPException(status_code=404, detail="Question not found")
        
        # Check if question has attempts (soft lock)
        attempts = db.query(QuestionAttempt).filter_by(question_id=question_id).count()
        if attempts > 0:
            logger.warning(f"Attempt to edit question with {attempts} attempts")
            raise HTTPException(status_code=400, detail="Cannot edit question with existing attempts")
        
        # Validate input
        if not question_data.get("text") or len(question_data["text"]) > 1000:
            raise HTTPException(status_code=422, detail="Invalid question text")
        
        options = question_data.get("options", [])
        if len(options) != 4 or any(len(o) > 200 for o in options):
            raise HTTPException(status_code=422, detail="Must have exactly 4 options, max 200 chars each")
        
        correct_idx = question_data.get("correct_option_index")
        if correct_idx not in [0, 1, 2, 3]:
            raise HTTPException(status_code=422, detail="Correct option must be 0-3")
        
        difficulty = question_data.get("difficulty")
        if difficulty not in ["easy", "medium", "hard"]:
            raise HTTPException(status_code=422, detail="Invalid difficulty")
        
        # Update
        question.text = question_data["text"]
        question.difficulty = difficulty
        question.category = question_data.get("category")
        question.explanation = question_data.get("explanation")
        question.updated_at = datetime.utcnow()
        
        # Update options (delete old, create new)
        db.query(QuestionOption).filter_by(question_id=question_id).delete()
        for idx, option_text in enumerate(options):
            option = QuestionOption(
                question_id=question_id,
                text=option_text,
                is_correct=(idx == correct_idx)
            )
            db.add(option)
        
        db.commit()
        
        # Log edit
        audit_log = QuestionAuditLog(
            question_id=question_id,
            admin_id=admin.id,
            action="edit",
            changes=json.dumps(question_data)
        )
        db.add(audit_log)
        db.commit()
        
        logger.info(f"Question {question_id} edited by admin {admin.id}")
        
        return {
            "success": True,
            "data": {
                "question_id": question.id,
                "text": question.text,
                "difficulty": question.difficulty,
                "category": question.category,
                "updated_at": question.updated_at.isoformat()
            }
        }
    except HTTPException:
        raise
    except Exception as e:
        logger.error(f"Edit question error: {str(e)}")
        raise HTTPException(status_code=500, detail="Failed to edit question")
```

## 📊’ Security

1. **Admin Only** - Check admin role
2. **Audit Trail** - Log all edits with admin ID
3. **Soft Lock** - Cannot edit questions with existing attempts
4. **Input Validation** - Strict validation on all fields
