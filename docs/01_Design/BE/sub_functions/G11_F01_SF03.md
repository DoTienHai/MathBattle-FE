# G11_F01_SF03: Delete Question

✅ MVP | **Admin** | **Question Management**

## 📋 Description

Admin soft-deletes question to preserve quiz history and referential integrity. Deleted questions don't appear in new quizzes but remain in completed quiz history.

## 🎯 API Specification

**Endpoint**: `DELETE /api/v1/admin/questions/{question_id}`

**Output (200 OK)**
```json
{
  "success": true,
  "data": {
    "message": "Question deleted successfully",
    "question_id": 123
  }
}
```

**Errors**: 401 (unauthorized), 403 (forbidden), 404 (not found), 500 (error)

## 🗏️ Business Logic (7 Steps)

1. **Authenticate Admin** - Verify admin role
2. **Fetch Question** - Query by question_id
3. **Check Exists** - Return 404 if not found
4. **Soft Delete** - Set is_deleted = true
5. **Update Timestamp** - Set deleted_at = NOW()
6. **Log Deletion** - Record in audit log with admin ID
7. **Return Success** - HTTP 200 OK

## 🗄️ Database Schema

```sql
ALTER TABLE quiz_questions ADD COLUMN is_deleted BOOLEAN DEFAULT false;
ALTER TABLE quiz_questions ADD COLUMN deleted_at TIMESTAMP;

CREATE INDEX idx_questions_deleted ON quiz_questions(is_deleted);
```

## 💻 Backend Implementation

```python
# app/api/v1/admin/questions.py
@router.delete("/{question_id}")
async def delete_question(
    question_id: int,
    admin: User = Depends(get_admin_user),
    db: Session = Depends(get_db)
):
    """Delete question (admin only)"""
    try:
        question = db.query(QuizQuestion).filter_by(id=question_id).first()
        if not question:
            raise HTTPException(status_code=404, detail="Question not found")
        
        # Soft delete
        question.is_deleted = True
        question.deleted_at = datetime.utcnow()
        db.commit()
        
        # Log deletion
        audit_log = QuestionAuditLog(
            question_id=question_id,
            admin_id=admin.id,
            action="delete"
        )
        db.add(audit_log)
        db.commit()
        
        logger.info(f"Question {question_id} deleted by admin {admin.id}")
        
        return {
            "success": True,
            "data": {
                "message": "Question deleted successfully",
                "question_id": question_id
            }
        }
    except HTTPException:
        raise
    except Exception as e:
        logger.error(f"Delete question error: {str(e)}")
        raise HTTPException(status_code=500, detail="Failed to delete question")
```

## 📊’ Security

1. **Admin Only** - Verify admin role
2. **Soft Delete** - Preserve data for history
3. **Audit Trail** - Log who deleted and when
4. **Non-Destructive** - Can be undeleted if needed
