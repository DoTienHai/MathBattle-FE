## 📋 Description

Admin endpoint to categorize or recategorize questions into quiz categories. Tracks all categorization changes in audit log with justification text provided by admin.

---

## 🎯 Detailed Requirements

### Input Parameters

```python
# Request Body
{
  "question_id": integer (required, min: 1),
  "category_id": integer (required, min: 1),
  "admin_justification": string (required, min: 10, max: 500)
}
```

### Output Schemas

**Success Response (200 OK)**
```json
{
  "success": true,
  "data": {
```

## 🗏️ Business Logic (11 Steps)

1. **Authenticate Admin User**
   - Verify JWT token is valid
   - Extract admin_id from token claims
   - Verify user has "admin" or "super_admin" role
   - Return 401 if not authenticated or 403 if not admin

2. **Validate Request Payload**
   - Check question_id is positive integer
   - Check category_id is positive integer
   - Check admin_justification length âˆˆ [10, 500] chars
   - Verify justification contains meaningful text (not spam)
   - Return 422 if validation fails

3. **Verify Question Exists**
   - Query question by question_id
   - Confirm question is not deleted
   - Return 404 if question doesn't exist
   - Fetch old_category_id to track change

4. **Verify New Category Exists**
   - Query quiz_categories table for category_id
   - Confirm category is not deleted/archived
   - Return 404 if category not found

5. **Check if Change is Necessary**
   - Compare old_category_id with new_category_id
   - If same: Return 400 "Question already in this category"
   - If different: Proceed to update

6. **Create Audit Log Entry**
   - Insert record into question_categorization_audit table
   - Store: question_id, old_category_id, new_category_id, admin_id, justification, timestamp
   - Capture admin_id and admin email
   - Assign audit_log_id for reference

7. **Update Question Category**
   - UPDATE questions SET category_id = :new_category_id WHERE id = :question_id
   - Set updated_at = NOW()
   - Set categorized_by = admin_id
   - Run inside transaction for atomicity

8. **Invalidate Related Caches**
   - Clear Redis cache for: questions:all, questions:by_category:{old_cat}, questions:by_category:{new_cat}
   - Clear quiz_questions cache for affected quizzes

9. **Fetch Updated Question**
   - SELECT question with new category info
   - Get old_category name from audit log
   - Get new_category name from quiz_categories
   - Prepare response object

10. **Log Admin Activity**
    - Insert into admin_activity_log with action="CATEGORIZE_QUESTION"
    - Include admin_id, question_id, old/new category, timestamp

11. **Return Success Response**
    - Include question_id, category names, justification, timestamp
    - Include audit_log_id for reference
    - HTTP 200 OK

```python
# app/services/question_service.py
from sqlalchemy import and_, func
from sqlalchemy.orm import Session
from datetime import datetime
from typing import Dict
import logging

logger = logging.getLogger(__name__)

class QuestionService:
    
    @staticmethod
    async def categorize_question(
        question_id: int,
        category_id: int,
        admin_id: int,
        justification: str,
        db: Session,
        ip_address: str = None,
        user_agent: str = None
    ) -> Dict:
        """
        Recategorize a question with audit trail.
        
        Args:
            question_id: ID of question to recategorize
            category_id: ID of new category
            admin_id: ID of admin making change
            justification: Reason for categorization change (10-500 chars)
            db: Database session
            ip_address: IP address of request
            user_agent: User agent string
            
        Returns:
            Dict with categorization result and audit info
            
        Raises:
            ValueError: If question/category not found or already in category
            PermissionError: If user not admin
        """
        try:
            # Verify question exists and not deleted
            question = db.query(Question).filter(
                and_(
                    Question.id == question_id,
                    Question.is_deleted == False
                )
            ).first()
            
            if not question:
                raise ValueError(f"Question {question_id} not found or deleted")
            
            old_category_id = question.category_id
            old_category = db.query(QuizCategory).filter(
                QuizCategory.id == old_category_id
            ).first()
            
            # Verify new category exists and not deleted
            new_category = db.query(QuizCategory).filter(
                and_(
                    QuizCategory.id == category_id,
                    QuizCategory.is_deleted == False
                )
            ).first()
            
            if not new_category:
                raise ValueError(f"Category {category_id} not found or deleted")
            
            # Check if change is necessary
            if old_category_id == category_id:
                raise ValueError(f"Question already in category '{new_category.name}'")
            
            # Create audit log entry
            audit_log = QuestionCategorizationAudit(
                question_id=question_id,
                old_category_id=old_category_id,
                new_category_id=category_id,
                admin_id=admin_id,
                admin_justification=justification,
                ip_address=ip_address,
                user_agent=user_agent,
                created_at=datetime.utcnow()
            )
            db.add(audit_log)
            
            # Update question
            question.category_id = category_id
            question.categorized_by_id = admin_id
            question.updated_at = datetime.utcnow()
            
            # Log admin activity
            activity = AdminActivityLog(
                admin_id=admin_id,
                action="CATEGORIZE_QUESTION",
                resource_type="question",
                resource_id=question_id,
                details={
                    "old_category_id": old_category_id,
                    "new_category_id": category_id,
                    "justification": justification
                },
                created_at=datetime.utcnow()
            )
            db.add(activity)
            
            # Commit transaction
            db.commit()
            db.refresh(audit_log)
            
            logger.info(
                f"Question {question_id} recategorized by admin {admin_id}: "
                f"{old_category.name} â†’ {new_category.name}"
            )
            
            # Invalidate caches
            cache_keys = [
                f"questions:all",
                f"questions:by_category:{old_category_id}",
                f"questions:by_category:{category_id}",
                f"question:{question_id}"
            ]
            for key in cache_keys:
                redis_client.delete(key)
            
            return {
                "question_id": question_id,
                "old_category_id": old_category_id,
                "new_category_id": category_id,
                "question_title": question.title,
                "old_category": old_category.name,
                "new_category": new_category.name,
                "categorized_by": (await db.query(User).filter(
                    User.id == admin_id
                ).first()).email,
                "admin_justification": justification,
                "categorized_at": audit_log.created_at,
                "audit_log_id": audit_log.id
            }
        
        except Exception as e:
            db.rollback()
            logger.error(f"Categorization failed: {str(e)}", exc_info=True)
            raise
    
    @staticmethod
    async def get_categorization_history(
        question_id: int,
        db: Session
    ) -> list:
        """
        Get complete categorization history for a question.
        
        Args:
            question_id: ID of question
            db: Database session
            
        Returns:
            List of all categorization changes
        """
        audits = db.query(QuestionCategorizationAudit).filter(
            QuestionCategorizationAudit.question_id == question_id
        ).order_by(QuestionCategorizationAudit.created_at.desc()).all()
        
        result = []
        for audit in audits:
            old_cat = db.query(QuizCategory).filter(
                QuizCategory.id == audit.old_category_id
            ).first()
            new_cat = db.query(QuizCategory).filter(
                QuizCategory.id == audit.new_category_id
            ).first()
            admin = db.query(User).filter(User.id == audit.admin_id).first()
            
            result.append({
                "audit_log_id": audit.id,
                "old_category": old_cat.name if old_cat else None,
                "new_category": new_cat.name if new_cat else None,
                "admin_email": admin.email if admin else None,
                "justification": audit.admin_justification,
                "changed_at": audit.created_at
            })
        
        return result
```

```python
# app/api/v1/admin/questions.py
from fastapi import APIRouter, Depends, HTTPException, status, Request
from typing import Optional
import logging

logger = logging.getLogger(__name__)
router = APIRouter(prefix="/admin/questions", tags=["admin-questions"])

@router.post("/categorize", response_model=dict)
async def categorize_question(
    request_data: CategorizeQuestionRequest,
    request: Request,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Categorize or recategorize a question.
    
    Request:
    - question_id: ID of question to recategorize
    - category_id: ID of new category
    - admin_justification: Reason for change (10-500 chars)
    
    Returns:
    - Updated question with category info and audit trail
    
    Authorization: Admin only
    """
    try:
        # Verify admin role
        if user.role not in ["admin", "super_admin"]:
            logger.warning(f"Non-admin {user.id} attempted categorization")
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Only admins can categorize questions"
            )
        
        logger.info(
            f"Admin {user.id} requesting categorization: "
            f"Q{request_data.question_id} â†’ Cat{request_data.category_id}"
        )
        
        # Get IP and user agent
        ip_address = request.client.host if request.client else None
        user_agent = request.headers.get("user-agent")
        
        result = await QuestionService.categorize_question(
            question_id=request_data.question_id,
            category_id=request_data.category_id,
            admin_id=user.id,
            justification=request_data.admin_justification,
            db=db,
            ip_address=ip_address,
            user_agent=user_agent
        )
        
        logger.info(f"Question categorization successful: {result['audit_log_id']}")
        
        return {
            "success": True,
            "data": result
        }
    
    except ValueError as e:
        logger.warning(f"Validation error during categorization: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Categorization failed: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to categorize question"
        )


@router.get("/categorization-history/{question_id}")
async def get_categorization_history(
    question_id: int,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """Get categorization history for a question. Admin only."""
    if user.role not in ["admin", "super_admin"]:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="Only admins can view categorization history"
        )
    
    try:
        history = await QuestionService.get_categorization_history(question_id, db)
        return {
            "success": True,
            "data": {
                "question_id": question_id,
                "history_count": len(history),
                "history": history
            }
        }
    except Exception as e:
        logger.error(f"Failed to fetch history: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to fetch categorization history"
        )
```

## 📊’ Security Considerations

1. **Admin-Only Access**
   - Verify user has admin or super_admin role
   - Return 403 Forbidden if unauthorized
   - Log all admin actions with admin_id

2. **Audit Trail**
   - Record all categorization changes in audit table
   - Store admin_id, timestamp, justification, IP address, user agent
   - Prevent tampering with audit logs (separate permissions)

3. **Input Validation**
   - Validate justification length (10-500 chars)
   - Prevent SQL injection via ORM parameterization
   - Verify question and category exist before update

4. **Transactional Integrity**
   - Use database transactions for atomicity
   - Rollback on any failure
   - Prevent partial updates

5. **Rate Limiting**
   - Limit: 50 categorizations per admin per hour
   - Return 429 Too Many Requests if exceeded
   - Track per admin_id

6. **Logging & Monitoring**
   - Log all categorization events
   - Alert on unusual patterns (bulk recategorizations)
   - Track IP addresses for anomaly detection

## 📜Œ Implementation Notes

- **Caching**: Invalidate Redis cache keys after update (5-minute TTL)

- **Versioning**: Track question version after each categorization
- **Rollback**: Support reverting categorization via history
