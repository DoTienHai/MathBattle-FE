## 📋 Description

Admin endpoint to bulk import 100+ questions via CSV file. Validates data, creates questions in batch with rollback on errors, and provides detailed import report with success/error counts.

## 🗏️ Business Logic (12 Steps)

1. **Authenticate Admin User**
   - Verify JWT token is valid
   - Extract admin_id from token claims
   - Verify user has "admin" or "super_admin" role
   - Return 401/403 if unauthorized

2. **Validate File Upload**
   - Verify file is provided and not empty
   - Check file extension is .csv
   - Verify file size â‰¤ 50MB
   - Verify MIME type is text/csv
   - Return 400 if validation fails

3. **Parse CSV File**
   - Read CSV line by line
   - Skip header row
   - Parse each row into dict
   - Return 400 if CSV is malformed

4. **Verify Category Exists**
   - Query quiz_categories for category_id
   - Confirm category is not deleted
   - Return 404 if category not found

5. **Validate Each Row**
   - For each CSV row: Check required fields present (title, options_json, correct_answer, difficulty)
   - Validate title length (1-500 chars)
   - Validate options_json is valid JSON array with 2-10 elements
   - Validate correct_answer is 0-indexed valid option number
   - Validate difficulty âˆˆ [easy, medium, hard]
   - Validate points âˆˆ [1, 100]
   - Collect errors with row number and reason

6. **Check for Duplicates**
   - Query existing questions with same title in category
   - Flag duplicate questions as error (can be overridden via flag)
   - Store duplicates in error list

7. **Generate Import ID**
   - Create unique import_id (prefix: "imp_" + UUID)
   - Create entry in question_bulk_imports table
   - Store: import_id, admin_id, category_id, file_name, total_rows, started_at

8. **Begin Transaction**
   - Start database transaction (for atomicity)
   - If transaction fails: rollback all inserts

9. **Batch Insert Valid Questions**
   - For each valid row: INSERT into questions table
   - Set: title, options, correct_answer, difficulty, points, category_id, created_by_id, created_at
   - Collect inserted question_ids
   - Batch insert in chunks of 50 for performance

10. **Update Import Record**
    - UPDATE question_bulk_imports: successful_count, failed_count, completed_at
    - Set import status to "completed"
    - Calculate success_rate = (successful / total) * 100

11. **Commit Transaction**
    - Commit all inserts atomically
    - Return 500 if commit fails, rollback all changes

12. **Return Response**
    - Include import_id, success/error counts, imported question IDs
    - Include detailed error list with row numbers
    - Return HTTP 200 with detailed report

## 🗄️ Database Schema

### Tables Required

**questions**
```sql
CREATE TABLE questions (
  id SERIAL PRIMARY KEY,
  category_id INT NOT NULL REFERENCES quiz_categories(id),
  title VARCHAR(500) NOT NULL,
  description TEXT,
  options JSONB NOT NULL,
  correct_answer INT NOT NULL,
  difficulty VARCHAR(20) NOT NULL CHECK (difficulty IN ('easy', 'medium', 'hard')),
  points INT DEFAULT 10 CHECK (points BETWEEN 1 AND 100),
  created_by_id INT NOT NULL REFERENCES users(id),
  is_deleted BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(category_id, title)
);
```

**question_bulk_imports**
```sql
CREATE TABLE question_bulk_imports (
  id SERIAL PRIMARY KEY,
  import_id VARCHAR(50) UNIQUE NOT NULL,
  admin_id INT NOT NULL REFERENCES users(id),
  category_id INT NOT NULL REFERENCES quiz_categories(id),
  file_name VARCHAR(500),
  total_rows INT NOT NULL,
  successful_count INT DEFAULT 0,
  failed_count INT DEFAULT 0,
  status VARCHAR(20) DEFAULT 'in_progress' CHECK (status IN ('in_progress', 'completed', 'failed')),
  error_details JSONB,
  started_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  completed_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**question_bulk_import_rows**
```sql
CREATE TABLE question_bulk_import_rows (
  id SERIAL PRIMARY KEY,
  import_id VARCHAR(50) NOT NULL REFERENCES question_bulk_imports(import_id),
  row_number INT NOT NULL,
  status VARCHAR(20) CHECK (status IN ('success', 'error')),
  question_id INT REFERENCES questions(id),
  error_code VARCHAR(100),
  error_message TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

```python
# app/services/bulk_import_service.py
from sqlalchemy import and_, func
from sqlalchemy.orm import Session
from datetime import datetime
from typing import Tuple, List, Dict
import csv
import json
import logging
from io import StringIO
import uuid

logger = logging.getLogger(__name__)

class BulkImportService:
    
    @staticmethod
    async def validate_csv_row(
        row: dict,
        row_number: int
    ) -> Tuple[bool, Optional[str], Optional[dict]]:
        """
        Validate single CSV row.
        
        Returns:
            Tuple of (is_valid, error_code, error_message)
        """
        try:
            # Validate title
            title = row.get("title", "").strip()
            if not title or len(title) > 500:
                return (False, "INVALID_TITLE", 
                       "Title required and must be <= 500 chars")
            
            # Validate options JSON
            try:
                options = json.loads(row.get("options_json", "[]"))
                if not isinstance(options, list) or len(options) < 2 or len(options) > 10:
                    return (False, "INVALID_OPTIONS",
                           "Options must be JSON array with 2-10 elements")
            except json.JSONDecodeError:
                return (False, "INVALID_JSON",
                       "Options JSON is malformed")
            
            # Validate correct_answer
            try:
                correct_answer = int(row.get("correct_answer", -1))
                if correct_answer < 0 or correct_answer >= len(options):
                    return (False, "INVALID_ANSWER",
                           f"Correct answer must be 0-{len(options)-1}")
            except (ValueError, TypeError):
                return (False, "INVALID_ANSWER",
                       "Correct answer must be integer")
            
            # Validate difficulty
            difficulty = row.get("difficulty", "").lower()
            if difficulty not in ["easy", "medium", "hard"]:
                return (False, "INVALID_DIFFICULTY",
                       "Difficulty must be easy, medium, or hard")
            
            # Validate points
            try:
                points = int(row.get("points", 10))
                if points < 1 or points > 100:
                    return (False, "INVALID_POINTS",
                           "Points must be between 1 and 100")
            except (ValueError, TypeError):
                return (False, "INVALID_POINTS",
                       "Points must be integer")
            
            return (True, None, None)
        
        except Exception as e:
            logger.error(f"Validation error on row {row_number}: {str(e)}")
            return (False, "VALIDATION_ERROR", str(e))
    
    @staticmethod
    async def parse_csv_file(file_content: str) -> Tuple[List[dict], bool]:
        """
        Parse CSV file content.
        
        Returns:
            Tuple of (rows_list, is_valid)
        """
        try:
            csv_reader = csv.DictReader(StringIO(file_content))
            
            # Verify required columns
            required_cols = {"title", "options_json", "correct_answer", "difficulty"}
            if not required_cols.issubset(set(csv_reader.fieldnames or [])):
                return ([], False)
            
            rows = list(csv_reader)
            return (rows, True)
        
        except Exception as e:
            logger.error(f"CSV parsing failed: {str(e)}")
            return ([], False)
    
    @staticmethod
    async def bulk_import_questions(
        file_content: str,
        category_id: int,
        admin_id: int,
        file_name: str,
        db: Session,
        allow_duplicates: bool = False
    ) -> Dict:
        """
        Bulk import questions from CSV file.
        
        Args:
            file_content: CSV file content as string
            category_id: Category ID for all imported questions
            admin_id: Admin user ID performing import
            file_name: Original file name
            db: Database session
            allow_duplicates: Whether to allow duplicate titles
            
        Returns:
            Dict with import results and error details
        """
        import_id = f"imp_{uuid.uuid4().hex[:8]}"
        error_list = []
        imported_ids = []
        successful_count = 0
        
        try:
            # Create import record
            import_record = QuestionBulkImport(
                import_id=import_id,
                admin_id=admin_id,
                category_id=category_id,
                file_name=file_name,
                total_rows=0,
                started_at=datetime.utcnow()
            )
            db.add(import_record)
            db.flush()  # Flush to get ID
            
            # Parse CSV
            rows, parse_ok = await BulkImportService.parse_csv_file(file_content)
            if not parse_ok or not rows:
                raise ValueError("Invalid CSV format or empty file")
            
            import_record.total_rows = len(rows)
            
            # Validate category
            category = db.query(QuizCategory).filter(
                and_(
                    QuizCategory.id == category_id,
                    QuizCategory.is_deleted == False
                )
            ).first()
            if not category:
                raise ValueError(f"Category {category_id} not found")
            
            # Start transaction
            transaction = db.begin_nested()
            
            try:
                # Process each row
                for row_num, row in enumerate(rows, start=2):  # Start at 2 (header is row 1)
                    # Validate row
                    is_valid, error_code, error_msg = await BulkImportService.validate_csv_row(row, row_num)
                    
                    if not is_valid:
                        error_list.append({
                            "row": row_num,
                            "title": row.get("title", "N/A"),
                            "error_code": error_code,
                            "error_message": error_msg
                        })
                        continue
                    
                    # Check for duplicates
                    if not allow_duplicates:
                        existing = db.query(Question).filter(
                            and_(
                                Question.category_id == category_id,
                                Question.title == row["title"],
                                Question.is_deleted == False
                            )
                        ).first()
                        if existing:
                            error_list.append({
                                "row": row_num,
                                "title": row["title"],
                                "error_code": "DUPLICATE_TITLE",
                                "error_message": f"Question already exists in category"
                            })
                            continue
                    
                    try:
                        # Create question
                        question = Question(
                            category_id=category_id,
                            title=row["title"],
                            description=row.get("description", ""),
                            options=json.loads(row["options_json"]),
                            correct_answer=int(row["correct_answer"]),
                            difficulty=row["difficulty"].lower(),
                            points=int(row.get("points", 10)),
                            created_by_id=admin_id,
                            created_at=datetime.utcnow()
                        )
                        db.add(question)
                        db.flush()
                        
                        imported_ids.append(question.id)
                        successful_count += 1
                        
                        # Log import row
                        import_row = QuestionBulkImportRow(
                            import_id=import_id,
                            row_number=row_num,
                            status="success",
                            question_id=question.id
                        )
                        db.add(import_row)
                    
                    except Exception as e:
                        logger.error(f"Insert error row {row_num}: {str(e)}")
                        error_list.append({
                            "row": row_num,
                            "title": row.get("title", "N/A"),
                            "error_code": "INSERT_ERROR",
                            "error_message": str(e)
                        })
                
                # Commit nested transaction
                transaction.commit()
            
            except Exception as e:
                transaction.rollback()
                logger.error(f"Bulk import transaction failed: {str(e)}")
                raise
            
            # Update import record
            import_record.successful_count = successful_count
            import_record.failed_count = len(error_list)
            import_record.status = "completed"
            import_record.completed_at = datetime.utcnow()
            import_record.error_details = error_list
            
            db.commit()
            
            admin_email = db.query(User).filter(User.id == admin_id).first().email
            
            return {
                "import_id": import_id,
                "total_rows": len(rows),
                "successful_imports": successful_count,
                "failed_imports": len(error_list),
                "success_rate": round((successful_count / len(rows) * 100), 1) if rows else 0,
                "imported_question_ids": imported_ids,
                "errors": error_list,
                "import_timestamp": datetime.utcnow(),
                "import_completed_by": admin_email
            }
        
        except Exception as e:
            db.rollback()
            logger.error(f"Bulk import failed: {str(e)}", exc_info=True)
            raise
```

```python
# app/api/v1/admin/bulk_import.py
from fastapi import APIRouter, Depends, HTTPException, status, UploadFile, File
from typing import Optional
import logging

logger = logging.getLogger(__name__)
router = APIRouter(prefix="/admin/bulk-import", tags=["admin-bulk-import"])

@router.post("/questions", response_model=dict)
async def bulk_import_questions(
    file: UploadFile = File(...),
    category_id: int = Query(..., gt=0),
    allow_duplicates: bool = Query(False),
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Bulk import questions from CSV file.
    
    Parameters:
    - file: CSV file (max 50MB)
    - category_id: Category for imported questions
    - allow_duplicates: Whether to allow duplicate titles
    
    CSV Format (required columns):
    - title: Question title
    - options_json: JSON array of answer options
    - correct_answer: 0-indexed correct option
    - difficulty: easy|medium|hard
    - points: 1-100
    - description: Optional description
    
    Returns:
    - Import report with success/error counts and details
    
    Authorization: Admin only
    """
    try:
        # Verify admin role
        if user.role not in ["admin", "super_admin"]:
            logger.warning(f"Non-admin {user.id} attempted bulk import")
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Only admins can bulk import questions"
            )
        
        # Validate file
        if not file:
            raise HTTPException(
                status_code=status.HTTP_400_BAD_REQUEST,
                detail="File is required"
            )
        
        if not file.filename.lower().endswith(".csv"):
            raise HTTPException(
                status_code=status.HTTP_400_BAD_REQUEST,
                detail="File must be CSV format"
            )
        
        if file.size and file.size > 50 * 1024 * 1024:  # 50MB
            raise HTTPException(
                status_code=status.HTTP_400_BAD_REQUEST,
                detail="File size exceeds 50MB limit"
            )
        
        # Read file
        content = await file.read()
        file_text = content.decode("utf-8")
        
        logger.info(f"Admin {user.id} starting bulk import: {file.filename}")
        
        # Import questions
        result = await BulkImportService.bulk_import_questions(
            file_content=file_text,
            category_id=category_id,
            admin_id=user.id,
            file_name=file.filename,
            db=db,
            allow_duplicates=allow_duplicates
        )
        
        logger.info(
            f"Bulk import completed: {result['successful_imports']}/{result['total_rows']} success"
        )
        
        return {
            "success": True,
            "data": result
        }
    
    except ValueError as e:
        logger.warning(f"Validation error during import: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Bulk import failed: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Bulk import failed"
        )


@router.get("/import-status/{import_id}")

    import_id: str,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)

    if user.role not in ["admin", "super_admin"]:

            status_code=status.HTTP_403_FORBIDDEN,

        )
    
    try:

        if not import_record:
            raise HTTPException(

                detail=f"Import {import_id} not found"

        return {
            "success": True,
            "data": {
                "import_id": import_record.import_id,
                "status": import_record.status,
                "total_rows": import_record.total_rows,
                "successful_count": import_record.successful_count,
                "failed_count": import_record.failed_count,
                "started_at": import_record.started_at,
                "completed_at": import_record.completed_at
            }
        }
    except Exception as e:
        logger.error(f"Failed to get import status: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to get import status"
        )
```

## 📊’ Security Considerations

1. **Admin-Only Access**
   - Verify admin or super_admin role
   - Return 403 if unauthorized

2. **File Validation**
   - Verify file is CSV format (extension and MIME type)
   - Limit file size to 50MB
   - Scan for malicious content patterns

3. **Input Validation**
   - Validate all CSV fields per row
   - Check JSON validity before parsing
   - Prevent SQL injection via ORM

4. **Transaction Atomicity**
   - Use database transactions for all-or-nothing insert
   - Rollback on any row failure
   - Prevent partial imports

5. **Rate Limiting**
   - Limit: 5 bulk imports per admin per hour
   - Return 429 if exceeded

6. **Audit Logging**
   - Log all import attempts (success/failure)
   - Store file name, row count, admin_id
   - Track import_id for reference
