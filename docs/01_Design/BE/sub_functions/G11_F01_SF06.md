## 📋 Description

Admin endpoint to search and filter questions from question bank with full-text search, filtering by category/difficulty/author, and pagination with advanced sorting options.

---

```python
# Query Parameters
```

## 🗏️ Business Logic (11 Steps)

1. **Authenticate Admin User**
   - Verify JWT token is valid
   - Extract user_id from token claims
   - Verify user has "admin" or "super_admin" role (for include_deleted)
   - Return 401/403 if unauthorized

2. **Validate Query Parameters**
   - Check page â‰¥ 1
   - Check page_size âˆˆ [1, 100]
   - Check search_term â‰¤ 200 chars
   - Validate difficulty âˆˆ [easy, medium, hard] if provided
   - Validate sort_by âˆˆ [created_at, title, difficulty, points]
   - Return 422 if validation fails

3. **Build Base Query**
   - SELECT questions with all relevant fields
   - By default: WHERE is_deleted = false
   - If include_deleted=true AND admin: WHERE 1=1 (show all)

4. **Apply Full-Text Search**
   - If search_term provided: Use PostgreSQL full-text search on title + description
   - OR: Use ILIKE on title/description for case-insensitive substring match
   - Use query: (title ILIKE :search OR description ILIKE :search OR options @> :search_json)
   - Strip/normalize search term (remove extra spaces, special chars)

5. **Apply Category Filter**
   - If category provided: JOIN with quiz_categories, WHERE category.name = :category
   - Verify category exists (return 400 if not)
   - Default: no filter

6. **Apply Difficulty Filter**
   - If difficulty provided: WHERE difficulty = :difficulty
   - Default: no filter

7. **Apply Author Filter**
   - If author_id provided: WHERE created_by_id = :author_id
   - Verify author exists (return 404 if not)
   - Default: no filter

8. **Get Total Count**
   - Count all records matching filters (before pagination)
   - Store for pagination metadata

9. **Apply Sorting**
   - If sort_by="created_at": ORDER BY questions.created_at [ASC|DESC]
   - If sort_by="title": ORDER BY questions.title [ASC|DESC]
   - If sort_by="difficulty": ORDER BY FIELD(difficulty, 'easy', 'medium', 'hard')
   - If sort_by="points": ORDER BY questions.points [ASC|DESC]

10. **Apply Pagination**
    - Calculate offset = (page - 1) * page_size
    - LIMIT page_size OFFSET offset
    - Calculate total_pages = CEIL(total_count / page_size)

11. **Return Response**
    - Map questions to response schema
    - Include pagination metadata
    - Include applied filters summary
    - Track query execution time
    - Cache for 2 minutes (Redis)

## 🗄️ Database Schema

### Tables Required

**questions**
```sql
CREATE TABLE questions (
  id SERIAL PRIMARY KEY,
  category_id INT NOT NULL REFERENCES quiz_categories(id),
  title VARCHAR(500) NOT NULL,
  description TEXT,
  options JSONB NOT NULL,
  correct_answer INT NOT NULL,
  difficulty VARCHAR(20) NOT NULL CHECK (difficulty IN ('easy', 'medium', 'hard')),
  points INT DEFAULT 10 CHECK (points BETWEEN 1 AND 100),
  created_by_id INT NOT NULL REFERENCES users(id),
  is_deleted BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**quiz_categories**
```sql
CREATE TABLE quiz_categories (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100) UNIQUE NOT NULL,
  description TEXT,
  is_deleted BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Indexes (CRITICAL for performance)

```python
# app/services/question_search_service.py
from sqlalchemy import and_, or_, func, text
from sqlalchemy.orm import Session
from datetime import datetime, timedelta
from typing import Tuple, List, Dict, Optional
import logging
import time

logger = logging.getLogger(__name__)

class QuestionSearchService:
    
    @staticmethod
    async def search_questions(
        search_term: Optional[str] = None,
        category: Optional[str] = None,
        difficulty: Optional[str] = None,
        author_id: Optional[int] = None,
        page: int = 1,
        page_size: int = 20,
        sort_by: str = "created_at",
        sort_order: str = "desc",
        include_deleted: bool = False,
        user_id: int = None,
        user_role: str = None,
        db: Session = None
    ) -> Tuple[List[Dict], int, Dict]:
        """
        Search and filter questions with full-text search.
        
        Args:
            search_term: Full-text search on title/description
            category: Filter by category name
            difficulty: Filter by difficulty level
            author_id: Filter by creator user_id
            page: Page number (1-indexed)
            page_size: Items per page
            sort_by: Sort column (created_at, title, difficulty, points)
            sort_order: ASC or DESC
            include_deleted: Include deleted questions (admin only)
            user_id: User performing search
            user_role: User role for permission check
            db: Database session
            
        Returns:
            Tuple of (questions_list, total_count, filters_applied_dict)
        """
        start_time = time.time()
        
        try:
            # Build base query
            query = db.query(Question).join(
                QuizCategory, Question.category_id == QuizCategory.id
            ).join(
                User, Question.created_by_id == User.id
            )
            
            # Apply deletion filter
            if not include_deleted or user_role not in ["admin", "super_admin"]:
                query = query.filter(Question.is_deleted == False)
            
            filters_applied = {
                "search_term": search_term,
                "category": category,
                "difficulty": difficulty,
                "author_id": author_id
            }
            
            # Apply search filter (full-text)
            if search_term and search_term.strip():
                search_pattern = f"%{search_term.strip()}%"
                query = query.filter(
                    or_(
                        Question.title.ilike(search_pattern),
                        Question.description.ilike(search_pattern)
                    )
                )
            
            # Apply category filter
            if category:
                cat = db.query(QuizCategory).filter(
                    QuizCategory.name == category
                ).first()
                if not cat:
                    raise ValueError(f"Category '{category}' not found")
                query = query.filter(Question.category_id == cat.id)
            
            # Apply difficulty filter
            if difficulty:
                query = query.filter(Question.difficulty == difficulty)
            
            # Apply author filter
            if author_id:
                author = db.query(User).filter(User.id == author_id).first()
                if not author:
                    raise ValueError(f"Author {author_id} not found")
                query = query.filter(Question.created_by_id == author_id)
            
            # Get total count
            total_count = query.count()
            
            # Apply sorting
            if sort_by == "title":
                order_col = Question.title
            elif sort_by == "difficulty":
                order_col = func.field(
                    Question.difficulty, "easy", "medium", "hard"
                )
            elif sort_by == "points":
                order_col = Question.points
            else:  # created_at
                order_col = Question.created_at
            
            if sort_order.lower() == "asc":
                query = query.order_by(order_col.asc())
            else:
                query = query.order_by(order_col.desc())
            
            # Apply pagination
            offset = (page - 1) * page_size
            questions = query.offset(offset).limit(page_size).all()
            
            # Map to response
            result = []
            for q in questions:
                creator = db.query(User).filter(User.id == q.created_by_id).first()
                result.append({
                    "question_id": q.id,
                    "title": q.title,
                    "category": (db.query(QuizCategory).filter(
                        QuizCategory.id == q.category_id
                    ).first().name),
                    "difficulty": q.difficulty,
                    "points": q.points,
                    "created_by": creator.email if creator else "Unknown",
                    "created_at": q.created_at,
                    "updated_at": q.updated_at,
                    "is_deleted": q.is_deleted,
                    "option_count": len(q.options) if q.options else 0,
                    "description": q.description
                })
            
            elapsed_ms = round((time.time() - start_time) * 1000)
            logger.info(
                f"Search completed: {len(result)} results in {elapsed_ms}ms "
                f"(total: {total_count})"
            )
            
            return result, total_count, filters_applied
        
        except Exception as e:
            logger.error(f"Search failed: {str(e)}", exc_info=True)
            raise
    
    @staticmethod
    async def get_search_suggestions(
        search_prefix: str,
        db: Session,
        limit: int = 10
    ) -> List[str]:
        """
        Get autocomplete suggestions for search.
        
        Args:
            search_prefix: Partial search term
            db: Database session
            limit: Max suggestions
            
        Returns:
            List of matching titles
        """
        try:
            search_pattern = f"{search_prefix.lower()}%"
            suggestions = db.query(Question.title).filter(
                and_(
                    Question.title.ilike(search_pattern),
                    Question.is_deleted == False
                )
            ).distinct().limit(limit).all()
            
            return [s[0] for s in suggestions]
        
        except Exception as e:
            logger.error(f"Failed to get suggestions: {str(e)}")
            return []
    
    @staticmethod
    async def get_filter_options(db: Session) -> Dict:
        """
        Get available filter options (categories, authors).
        
        Returns:
            Dict with filter options
        """
        try:
            categories = db.query(QuizCategory).filter(
                QuizCategory.is_deleted == False
            ).all()
            
            authors = db.query(User).join(
                Question, User.id == Question.created_by_id
            ).filter(
                Question.is_deleted == False
            ).distinct().all()
            
            return {
                "categories": [
                    {"id": c.id, "name": c.name} for c in categories
                ],
                "difficulties": ["easy", "medium", "hard"],
                "authors": [
                    {"id": a.id, "email": a.email} for a in authors
                ],
                "sort_options": [
                    "created_at", "title", "difficulty", "points"
                ]
            }
        
        except Exception as e:
            logger.error(f"Failed to get filter options: {str(e)}")
            return {}
```

```python
# app/api/v1/admin/questions_search.py
from fastapi import APIRouter, Depends, Query, HTTPException, status
from typing import Optional
import logging
import math

logger = logging.getLogger(__name__)
router = APIRouter(prefix="/admin/questions", tags=["admin-questions"])

@router.get("/search", response_model=dict)
async def search_questions(
    search_term: Optional[str] = Query(None, max_length=200),
    category: Optional[str] = Query(None),
    difficulty: Optional[str] = Query(None, regex="^(easy|medium|hard)$"),
    author_id: Optional[int] = Query(None, gt=0),
    page: int = Query(1, ge=1),
    page_size: int = Query(20, ge=1, le=100),
    sort_by: str = Query("created_at", regex="^(created_at|title|difficulty|points)$"),
    sort_order: str = Query("desc", regex="^(asc|desc)$"),
    include_deleted: bool = Query(False),
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Search and filter questions in question bank.
    
    Query Parameters:
    - search_term: Full-text search on title/description (max 200 chars)
    - category: Filter by category name
    - difficulty: Filter by difficulty (easy, medium, hard)
    - author_id: Filter by creator user_id
    - page: Page number (default: 1)
    - page_size: Items per page (default: 20, max: 100)
    - sort_by: Sort column (created_at, title, difficulty, points)
    - sort_order: Sort direction (asc, desc)
    - include_deleted: Include deleted questions (admin only)
    
    Returns:
    - Paginated question list with filters applied
    
    Authorization: Admin only
    """
    try:
        # Verify admin role
        if user.role not in ["admin", "super_admin"]:
            logger.warning(f"Non-admin {user.id} attempted question search")
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Only admins can search questions"
            )
        
        logger.info(
            f"Admin {user.id} searching questions: "
            f"term='{search_term}', cat='{category}', diff='{difficulty}'"
        )
        
        questions, total_count, filters = await QuestionSearchService.search_questions(
            search_term=search_term,
            category=category,
            difficulty=difficulty,
            author_id=author_id,
            page=page,
            page_size=page_size,
            sort_by=sort_by,
            sort_order=sort_order,
            include_deleted=include_deleted,
            user_id=user.id,
            user_role=user.role,
            db=db
        )
        
        total_pages = math.ceil(total_count / page_size) if total_count > 0 else 1
        
        return {
            "success": True,
            "data": {
                "questions": questions,
                "pagination": {
                    "page": page,
                    "page_size": page_size,
                    "total_items": total_count,
                    "total_pages": total_pages,
                    "has_next": page < total_pages,
                    "has_prev": page > 1
                },
                "filters_applied": filters
            }
        }
    
    except ValueError as e:
        logger.warning(f"Validation error during search: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Search failed: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Search failed"
        )
```

## 📊’ Security Considerations

1. **Admin-Only Access**
   - Verify admin or super_admin role
   - Return 403 if unauthorized

2. **Deleted Questions Access**
   - Regular admins: Cannot see deleted questions
   - Super admins: Can see if include_deleted=true
   - Prevent data leakage

3. **Input Validation**
   - Limit search_term to 200 chars
   - Validate all filter parameters
   - Prevent SQL injection via ORM

4. **Rate Limiting**
   - Limit: 100 searches per admin per minute
   - Return 429 if exceeded
   - Track per user_id

5. **Performance Protection**
   - Set query timeout (2 seconds max)
   - Limit page_size to 100
   - Use database indexes for fast filtering

6. **Audit Logging**
   - Log all searches with search term and filters
   - Track admin_id and timestamp
   - Monitor unusual search patterns
