## 📋 Description

Admin endpoint to ban user from platform (soft delete, can unban). Bans user account, revokes all active sessions, resets user streak, sends email notification, and logs to audit trail.

---

## 🗏️ Business Logic (12 Steps)

1. **Authenticate Admin User**
   - Verify JWT token is valid
   - Extract admin_id from token claims
   - Verify user has "admin" or "super_admin" role
   - Return 401/403 if unauthorized

2. **Validate Request Payload**
   - Check user_id is positive integer
   - Check reason length âˆˆ [10, 500] chars
   - Check duration âˆˆ ["permanent", "temporary"]
   - If duration="temporary": Verify until_date is ISO 8601 format and > now
   - Return 422 if validation fails

3. **Verify Target User Exists**
   - Query user by user_id
   - Confirm user is not already deleted/banned
   - Return 404 if user doesn't exist
   - Prevent self-ban (admin_id != user_id)

4. **Validate Ban Parameters**
   - If temporary ban: until_date must be in future (> now)
   - If temporary ban: Duration max 365 days
   - Calculate ban_end_date
   - Return 400 if invalid

5. **Begin Transaction**
   - Start database transaction for atomicity
   - All operations must commit together or rollback

6. **Update User Account**
   - SET users.is_banned = true
   - SET users.ban_reason = :reason
   - SET users.ban_start_date = NOW()
   - SET users.ban_until_date = :until_date (if temporary)
   - SET users.banned_by_id = :admin_id
   - SET users.updated_at = NOW()

7. **Revoke All Active Sessions**
   - Query user_sessions table for user_id
   - For each active session: SET revoked_at = NOW()
   - Count revoked sessions
   - Invalidate Redis session tokens (key: sessions:{user_id}:*)

8. **Reset User Streak**
   - Query user_streaks table for user_id
   - SET current_streak = 0
   - SET streak_reset_at = NOW()
   - Set streak_lost_reason = "user_banned"
   - Backup old streak value for audit

9. **Create Audit Log Entry**
   - INSERT into user_ban_audit table
   - Store: user_id, admin_id, reason, duration, ban_dates, sessions_count, streak_info
   - Assign audit_log_id

10. **Send Email Notification**
    - Get user email from users table
    - Build ban notification email
    - Include reason, duration, appeal instructions
    - Queue email in async task/background job

11. **Commit Transaction**
    - Commit all changes atomically
    - Return 500 if commit fails

12. **Return Success Response**
    - Include user info, ban details, sessions revoked, streak reset status
    - Include audit_log_id for reference
    - HTTP 200 OK

```python
from pydantic import BaseModel, Field, field_validator
from typing import Optional
from datetime import datetime, timedelta

class BanUserRequest(BaseModel):
    """Request to ban a user"""
    user_id: int = Field(..., gt=0)
    reason: str = Field(..., min_length=10, max_length=500)
    duration: str = Field(..., regex="^(permanent|temporary)$")
    until_date: Optional[str] = None  # ISO 8601 format
    
    @field_validator("until_date")
    @classmethod
    def validate_until_date(cls, v, info):
        if info.data.get("duration") == "temporary":
            if not v:
                raise ValueError("until_date required for temporary bans")
            try:
                date = datetime.fromisoformat(v.replace("Z", "+00:00"))
                if date <= datetime.now():
                    raise ValueError("until_date must be in the future")
                # Max 365 days
                if (date - datetime.now()).days > 365:
                    raise ValueError("Ban duration cannot exceed 365 days")
            except ValueError as e:
                raise ValueError(f"Invalid until_date format: {str(e)}")
        return v
    
    class Config:
        json_schema_extra = {
            "example": {
                "user_id": 123,
                "reason": "Multiple account violations and cheating detected",
                "duration": "permanent",
                "until_date": None
            }
        }

class BanUserResponse(BaseModel):
    """Response after user ban"""
    user_id: int
    username: str
    email: str
    ban_status: str
    ban_reason: str
    ban_duration: str
    ban_start_date: datetime
    ban_until_date: Optional[datetime]
    sessions_revoked: int
    streak_reset: bool
    notifications_sent: int
    audit_log_id: int
    banned_by: str
    banned_at: datetime
    
    class Config:
        from_attributes = True
```

### Service Layer

```python
# app/services/user_ban_service.py
from sqlalchemy import and_, func
from sqlalchemy.orm import Session
from datetime import datetime, timedelta
from typing import Dict
import logging

logger = logging.getLogger(__name__)

class UserBanService:
    
    @staticmethod
    async def ban_user(
        user_id: int,
        reason: str,
        duration: str,
        until_date: Optional[datetime],
        admin_id: int,
        db: Session,
        send_email: bool = True
    ) -> Dict:
        """
        Ban a user from platform.
        
        Args:
            user_id: ID of user to ban
            reason: Ban reason/explanation
            duration: "permanent" or "temporary"
            until_date: Date to lift ban (if temporary)
            admin_id: ID of admin performing ban
            db: Database session
            send_email: Whether to send notification email
            
        Returns:
            Dict with ban details and audit info
            
        Raises:
            ValueError: If user/admin not found or invalid parameters
        """
        try:
            # Verify user exists
            user = db.query(User).filter(User.id == user_id).first()
            if not user:
                raise ValueError(f"User {user_id} not found")
            
            # Prevent self-ban
            if user_id == admin_id:
                raise ValueError("Cannot ban yourself")
            
            # Check if already banned
            if user.is_banned:
                raise ValueError(f"User already banned")
            
            # Verify admin exists
            admin = db.query(User).filter(User.id == admin_id).first()
            if not admin:
                raise ValueError(f"Admin {admin_id} not found")
            
            # Start transaction
            transaction = db.begin_nested()
            
            try:
                # Update user ban status
                user.is_banned = True
                user.ban_reason = reason
                user.ban_start_date = datetime.utcnow()
                user.ban_until_date = until_date
                user.banned_by_id = admin_id
                user.updated_at = datetime.utcnow()
                
                # Revoke all active sessions
                sessions = db.query(UserSession).filter(
                    and_(
                        UserSession.user_id == user_id,
                        UserSession.is_active == True,
                        UserSession.revoked_at == None
                    )
                ).all()
                
                sessions_revoked = len(sessions)
                for session in sessions:
                    session.revoked_at = datetime.utcnow()
                    session.is_active = False
                    # Invalidate in Redis
                    redis_key = f"sessions:{user_id}:{session.id}"
                    redis_client.delete(redis_key)
                
                # Reset user streak
                streak = db.query(UserStreak).filter(
                    UserStreak.user_id == user_id
                ).first()
                
                old_streak = 0
                if streak:
                    old_streak = streak.current_streak
                    streak.current_streak = 0
                    streak.streak_reset_at = datetime.utcnow()
                    streak.streak_lost_reason = "user_banned"
                else:
                    # Create new streak record if doesn't exist
                    streak = UserStreak(
                        user_id=user_id,
                        current_streak=0,
                        longest_streak=0,
                        streak_reset_at=datetime.utcnow(),
                        streak_lost_reason="user_banned"
                    )
                    db.add(streak)
                
                # Create audit log
                ban_audit = UserBanAudit(
                    user_id=user_id,
                    admin_id=admin_id,
                    ban_reason=reason,
                    ban_duration=duration,
                    ban_start_date=datetime.utcnow(),
                    ban_until_date=until_date,
                    sessions_revoked=sessions_revoked,
                    old_streak=old_streak
                )
                db.add(ban_audit)
                db.flush()
                
                # Commit nested transaction
                transaction.commit()
                db.flush()
                
                logger.info(
                    f"User {user_id} banned by admin {admin_id}: {reason} "
                    f"({sessions_revoked} sessions revoked)"
                )
                
                # Send notification email (async)
                if send_email:
                    await UserBanService._send_ban_notification(user, reason, duration, until_date)
                
                return {
                    "user_id": user_id,
                    "username": user.username,
                    "email": user.email,
                    "ban_status": "banned",
                    "ban_reason": reason,
                    "ban_duration": duration,
                    "ban_start_date": datetime.utcnow(),
                    "ban_until_date": until_date,
                    "sessions_revoked": sessions_revoked,
                    "streak_reset": True,
                    "notifications_sent": 1,
                    "audit_log_id": ban_audit.id,
                    "banned_by": admin.email,
                    "banned_at": datetime.utcnow()
                }
            
            except Exception as e:
                transaction.rollback()
                logger.error(f"Ban transaction failed: {str(e)}")
                raise
        
        except Exception as e:
            db.rollback()
            logger.error(f"Ban user failed: {str(e)}", exc_info=True)
            raise
    
    @staticmethod
    async def _send_ban_notification(
        user: User,
        reason: str,
        duration: str,
        until_date: Optional[datetime]
    ) -> None:
        """
        Send ban notification email.
        
        Args:
            user: User being banned
            reason: Ban reason
            duration: Ban duration
            until_date: When ban expires (if temporary)
        """
        try:
            subject = "Your account has been suspended"
            
            body = f"""
Dear {user.username},

Your account has been suspended due to the following reason:
{reason}

Ban Type: {duration.capitalize()}
"""
            
            if duration == "temporary" and until_date:
                body += f"Ban Until: {until_date.strftime('%Y-%m-%d %H:%M:%S UTC')}\n"
            
            body += """
If you believe this is a mistake, you can file an appeal through our support system.

Best regards,
MathBattle Platform
"""
            
            # Queue in background job
            email_queue.enqueue(
                send_email_task,
                to_email=user.email,
                subject=subject,
                body=body
            )
            
            logger.info(f"Ban notification queued for {user.email}")
        
        except Exception as e:
            logger.error(f"Failed to send ban notification: {str(e)}")
    
    @staticmethod
    async def unban_user(
        user_id: int,
        admin_id: int,
        db: Session
    ) -> Dict:
        """
        Unban a user (only if ban was temporary and expired).
        
        Args:
            user_id: ID of user to unban
            admin_id: ID of admin performing unban
            db: Database session
            
        Returns:
            Dict with unban details
        """
        try:
            # Verify user exists and is banned
            user = db.query(User).filter(User.id == user_id).first()
            if not user:
                raise ValueError(f"User {user_id} not found")
            
            if not user.is_banned:
                raise ValueError("User is not banned")
            
            # Check ban expiration
            if user.ban_until_date and user.ban_until_date > datetime.utcnow():
                raise ValueError("Ban is still active, cannot unban yet")
            
            # Unban user
            user.is_banned = False
            user.ban_reason = None
            user.ban_until_date = None
            user.updated_at = datetime.utcnow()
            
            db.commit()
            
            logger.info(f"User {user_id} unbanned by admin {admin_id}")
            
            return {
                "user_id": user_id,
                "username": user.username,
                "ban_status": "active",
                "unbanned_at": datetime.utcnow()
            }
        
        except Exception as e:
            db.rollback()
            logger.error(f"Unban user failed: {str(e)}")
            raise
```

```python
# app/api/v1/admin/users_ban.py
from fastapi import APIRouter, Depends, HTTPException, status
from typing import Optional
import logging

logger = logging.getLogger(__name__)
router = APIRouter(prefix="/admin/users", tags=["admin-users"])

@router.post("/ban", response_model=dict)
async def ban_user(
    request_data: BanUserRequest,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """
    Ban a user from platform.
    
    Request:
    - user_id: ID of user to ban
    - reason: Ban reason (10-500 chars)
    - duration: "permanent" or "temporary"
    - until_date: ISO 8601 date (required if duration="temporary")
    
    Returns:
    - User ban details with sessions revoked and streak reset info
    
    Authorization: Admin only
    """
    try:
        # Verify admin role
        if user.role not in ["admin", "super_admin"]:
            logger.warning(f"Non-admin {user.id} attempted user ban")
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Only admins can ban users"
            )
        
        logger.info(
            f"Admin {user.id} banning user {request_data.user_id}: "
            f"{request_data.reason[:50]}..."
        )
        
        # Parse until_date if provided
        until_date = None
        if request_data.until_date:
            until_date = datetime.fromisoformat(
                request_data.until_date.replace("Z", "+00:00")
            )
        
        result = await UserBanService.ban_user(
            user_id=request_data.user_id,
            reason=request_data.reason,
            duration=request_data.duration,
            until_date=until_date,
            admin_id=user.id,
            db=db
        )
        
        logger.info(f"User {request_data.user_id} banned successfully")
        
        return {
            "success": True,
            "data": result
        }
    
    except ValueError as e:
        logger.warning(f"Validation error during ban: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    
    except Exception as e:
        logger.error(f"Ban user failed: {str(e)}", exc_info=True)
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to ban user"
        )


@router.post("/unban/{user_id}")
async def unban_user(
    user_id: int,
    user: User = Depends(get_current_user),
    db: Session = Depends(get_db)
) -> dict:
    """Unban a user (only if ban expired)"""
    if user.role not in ["admin", "super_admin"]:
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="Only admins can unban users"
        )
    
    try:
        result = await UserBanService.unban_user(user_id, user.id, db)
        return {"success": True, "data": result}
    except ValueError as e:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
    except Exception as e:
        logger.error(f"Unban failed: {str(e)}")
        raise HTTPException(
            status_code=status.HTTP_500_INTERNAL_SERVER_ERROR,
            detail="Failed to unban user"
        )
```

## 📊’ Security Considerations

1. **Admin-Only Access**
   - Verify admin or super_admin role
   - Return 403 if unauthorized
   - Log all ban operations

2. **Self-Ban Prevention**
   - Prevent admin from banning themselves
   - Return 400 if user_id == admin_id

3. **Session Revocation**
   - Revoke all active sessions
   - Invalidate Redis session tokens
   - Prevent concurrent login attempts

4. **Ban Audit Trail**
   - Log all bans with admin_id, reason, duration
   - Store old streak value for reference
   - Track sessions revoked count

5. **Email Security**
   - Send encrypted notification email
   - Queue in background job (async)
   - Include appeal instructions

6. **Rate Limiting**
   - Limit: 10 bans per admin per hour
   - Return 429 if exceeded
